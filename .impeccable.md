## Design Context

### Users
EastSea tools are used by search-driven visitors who want fast, practical answers in the browser without signup, installs, or API calls. The current tool request also makes bilingual support important for both Korean and English readers landing on individual utility pages.

### Brand Personality
Practical, trustworthy, concise. Interfaces should feel utility-first, fast to scan, and confident rather than decorative.

### Aesthetic Direction
Keep the visual language close to the existing calculator tools: strong contrast, clear sections, compact inputs, prominent result blocks, and mobile-friendly spacing. Favor clean gradients, restrained accents, and obvious interaction states over novelty-heavy layouts.

### Design Principles
1. Make the primary task obvious within the first screen.
2. Keep each tool self-contained, fast, and dependency-free.
3. Present results as both quick KPIs and detailed breakdowns.
4. Support Korean and English without bloating the interface.
5. Prefer readable, durable utility UI over experimental styling.
