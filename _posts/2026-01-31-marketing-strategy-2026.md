# 🎮 East Sea Games — 마케팅 전략서 2026

> **작성일:** 2026-01-30
> **대상:** 42개 HTML5 캐주얼 게임 포트폴리오 (eastsea.monster/games/)
> **플랫폼:** 텔레그램 Mini App + 웹 + itch.io + 기타

---

## 📊 Executive Summary

East Sea Games는 42개 HTML5 캐주얼 게임을 보유한 인디 게임 포트폴리오로, 텔레그램 Mini App을 핵심 유통 채널로 활용한다. 텔레그램 Stars 결제(7종 상품), SEO/PWA 최적화, 크로스프로모션이 이미 구현된 상태이며, 이 전략서는 첫 유저 획득부터 수익화까지의 로드맵을 제시한다.

### 핵심 차별점
- **42개 다양한 장르** — 퍼즐, 액션, 아케이드, 시뮬레이션, 전략, 리듬
- **100% 브라우저 플레이** — 설치 불필요, 즉시 접근
- **텔레그램 네이티브** — Stars 결제, 딥링크, 봇 통합
- **비(非)크립토** — 순수 게임 가치로 승부 (TG 게임 시장에서 희소)

### 핵심 인사이트
1. **모든 TG 메가히트는 크립토 연동** — 우리는 다른 길 (순수 게임)
2. **레퍼럴 = 핵심 성장 엔진** — 친구 초대 → 보상 루프 필수
3. **단순함이 승리** — 첫 10초 안에 재미를 느껴야 함
4. **80% 유저는 봇 농장 가능성** — 크립토 없는 우리에게는 진성 유저가 모임
5. **채널 외부 프로모션 필수** — TG 앱스토어 미존재, 외부 트래픽 끌어와야

### 인디 웹게임 바이럴 사례
- **Die in the Dungeon**: itch.io → Steam, 5개월 10만 카피 판매
- **Reddit r/WebGames**: 웹게임 직접 플레이 가능 → 높은 전환율
- **itch.io 프로토타입**: 2주 6,000 플레이 달성 사례 (Reddit + Twitter)
- **핵심**: "show, don't sell" — 게임 자체를 보여주고 팔지 마라

## 📣 채널별 전략

### 1. 텔레그램 (핵심 채널)
- **봇**: `/start` → TOP 5 게임, `/shop` → Stars 상점, `/games` → 전체 목록
- **채널**: @EastSeaGames (생성 필요) — 신작/업데이트/이벤트
- **그룹**: 플레이어 커뮤니티 (피드백, 리더보드 인증)
- **레퍼럴**: 친구 초대 시 보너스 코인/스핀 지급
- **Stars 전략**: 
  - 저가 상품(10-15 Stars)으로 첫 결제 유도
  - VIP 1일(50 Stars)로 프리미엄 경험 맛보기
  - 수익: 100 Stars = $2.4 유저 지불 → $1.3 개발자 수취

### 2. Reddit (런칭 부스터)
- **r/WebGames** (640K): 브라우저 직접 플레이 가능 → 높은 전환율
- **r/IndieGaming** (451K): 인디게임 셀프 프로모션 허용 (마일스톤당 1회)
- **r/playmygame** (80K): 피드백 요청 형태로 포스팅
- **r/html5games**: 기술적 관심 커뮤니티
- **규칙**: "show, don't sell" — GIF/영상 중심, CTA 최소화

### 3. itch.io (인디 허브)
- **TOP 5 게임 우선 등록** (무료 플레이)
- **태그 최적화**: html5, browser, casual, puzzle, arcade, mobile-friendly
- **번들 세일**: 향후 유료 전환 시 활용
- **커뮤니티 페이지**: devlog 정기 업데이트
- **참고**: itch.io 자체 트래픽은 적음 → 외부 마케팅 필수

### 4. X (Twitter) — 인디 게임 커뮤니티
- **해시태그**: #indiedev, #gamedev, #html5games, #webgames, #indiegame
- **콘텐츠**: GIF 게임플레이, 개발 비하인드, 주간 업데이트
- **인디게임 개발자 네트워킹** (상호 RT, 게임잼 참여)
- **빈도**: 주 3-5회

### 5. TikTok — 바이럴 잠재력
- **숏폼 영상**: 30초 게임플레이 + "이 게임 중독성 미쳤다" 스타일
- **트렌드 활용**: 인기 사운드 + 게임 화면
- **타겟 게임**: 시각적으로 화려한 것 (슬라임 서바이버, 네온 스네이크, 크리스탈 매치)
- **해시태그**: #mobilegames, #freegames, #addictinggames, #browsergames

### 6. 웹게임 포털 (CrazyGames, Poki, Newgrounds)
- **CrazyGames/Poki**: HTML5 게임 API 연동 → 광고 CPM 수익
- **Newgrounds**: 레트로 게이머 커뮤니티, 광고 수익 분배
- **Game Jolt**: 인디 커뮤니티, 무료 등록

## 🎯 핵심 전술

### 1. 레퍼럴 루프 (K-factor 극대화)
```
유저 A 플레이 → 점수 공유 (TG 메시지) → 
친구 B 클릭 → 게임 플레이 → 보너스 코인 양쪽 지급 →
B가 또 공유 → C, D, E...
```

### 2. 크로스프로모션 (이미 구현됨)
- 게임 A 클리어 → "비슷한 게임 추천" → 게임 B
- 42개 게임 간 트래픽 순환
- 게임 오버 화면에 "다른 게임 도전하기"

### 3. 데일리 인게이지먼트
- 매일 다른 "오늘의 게임" 추천
- 데일리 챌린지 + 보상
- 스트릭 보너스 (연속 접속 보상)

### 4. 콘텐츠 마케팅
- 주간 게임 리뷰/공략 블로그 (eastsea.monster)
- 게임 개발 비하인드 스토리
- Jekyll 블로그 + SEO → 오가닉 검색 트래픽
