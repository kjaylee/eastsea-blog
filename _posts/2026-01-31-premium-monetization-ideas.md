### 🥇 1위: 이미지 자동 생성 API + 앱스토어 스크린샷 통합 플랫폼

**왜 1위인가:**
- Bannerbear $630K ARR이 검증한 시장 (그리고 아직 $100M+ 시장 여유)
- 주인님의 이미지 처리 + API 개발 스킬과 완벽 매칭
- 한국/일본/아시아 시장 현지화 = 즉각적 차별화
- API SaaS = 높은 이탈 비용, 안정적 MRR

**실행 로드맵:**
| 주차 | 태스크 | 목표 |
|------|--------|------|
| 1주 | 핵심 API 개발 — Puppeteer 기반 HTML→이미지 렌더링, REST API | 동작하는 MVP |
| 2주 | 템플릿 시스템 + 대시보드 — 기본 웹 UI, API 키 관리 | 베타 테스트 가능 |

## 🎯 결론: 첫 번째 실행 추천

**"이미지 API 플랫폼"을 핵심 제품으로 시작하되, 동시에 크롬 확장 1-2개를 사이드로 출시하라.**

이유:
1. 이미지 API = 장기적 핵심 자산 (높은 이탈 비용, 복리 성장)
2. 크롬 확장 = 빠른 수익 확인 (1-2주 개발, 즉시 크롬 웹스토어 배포)
3. 두 제품이 시너지 — 확장이 API의 사용 사례를 증명, API가 확장의 백엔드
