**우선순위 기준**: 수익 영향도 > 시간 대비 효율 > 블로킹 임팩트

## 🔥 HIGH PRIORITY

### 1. Gumroad 디지털 제품 등록

**예상 소요 시간**: 10분  
**예상 수익**: 월 $50-200

디지털 제품을 판매할 수 있는 가장 빠른 방법입니다. 다음 3가지 제품을 준비했습니다:

1. **HTML5 게임 번들** ($29)
   - 10개 완성 게임 소스 코드
   - 단일 HTML 파일, MIT 라이선스
   - 모바일 반응형

2. **Claude 프롬프트 가이드** ($19)
   - 생산성 향상을 위한 100+ 프롬프트
   - 카테고리별 분류
   - 실전 예제 포함

3. **Notion 생산성 템플릿** ($15)
   - 프로젝트 관리, 일정 관리 등
   - 인디 개발자 최적화

**절차**:
- Gumroad 계정 생성 (Google 로그인)
- 결제 수신 설정 (PayPal/Stripe)
- 3개 제품 등록 (각 2-3분)
- 썸네일, 설명, 태그 설정

**장점**:
- 수수료 10% (낮은 편)
- 등록 즉시 판매 시작
- 자동 파일 전송
- Pay What You Want 가격 정책 가능

---

### 2. 모바일 게임 앱스토어 출시

**예상 소요 시간**: 15분  
**예상 수익**: 월 수백만원 이상

지금까지 만든 게임 중 하나를 네이티브 앱으로 패키징하여 앱스토어에 출시합니다.

**iOS App Store** (8분):
1. App Store Connect 접속
2. 앱 기본 정보 입력
3. 메타데이터 작성 (설명, 스크린샷, 키워드)
4. 빌드 업로드 (Xcode Archive)
5. 심사 제출 (1-3일 대기)

**Android Google Play** (7분):
1. Play Console 접속
2. 앱 생성 및 정보 입력
3. 메타데이터 작성
4. AAB 파일 업로드
5. 심사 제출 (몇 시간~1일)

**수익 모델**:
- 인앱 광고
- 인앱 구매 (아이템, 스킨 등)
- 프리미엄 버전

**필요 비용**:
- iOS: $99/년
- Android: $25 (일회성)

---

### 3. SaaS 환경변수 설정

**예상 소요 시간**: 3분  
**목표**: ContentForge 서비스 활성화

현재 개발 중인 SaaS (블로그 포스트 → 5개 플랫폼 자동 변환)의 환경변수를 설정해야 합니다.

**절차**:
1. Vercel 프로젝트 접속
2. Settings → Environment Variables
3. API 키 입력 (Production/Preview/Development)
4. 재배포
5. 서비스 작동 확인

완료되면 2주 후 런칭 예정입니다.

---

### 4. 클라우드 VM 인증

**예상 소요 시간**: 5분  
**목표**: 추가 서비스 배포 인프라 구축

GCP VM에 SSH 접속 및 Traefik 설정을 위한 인증 작업입니다.

**절차**:
1. gcloud CLI 인증 (`gcloud auth login`)
2. VM 인스턴스 확인
3. SSH 접속 테스트
4. 서비스 라우팅 설정 준비

완료되면 다양한 서브도메인으로 서비스를 외부에 공개할 수 있습니다.

## ⚡ MEDIUM PRIORITY

### 5. GameDistribution 게임 등록

**예상 소요 시간**: 15분  
**예상 수익**: 월 $30-300

GameDistribution은 HTML5 게임 포털 네트워크로, 광고 수익을 50-70% 분배합니다.

**등록할 게임** (5개):
1. Crystal Match Quest (매치-3 퍼즐)
2. Zombie Survivor Town (서바이벌 액션)
3. Screw Sort Factory (소팅 퍼즐)
4. Polygon Dungeon Survivor (로그라이크)
5. Rhythm Pulse (리듬 게임)

**절차**:
- 계정 생성
- 각 게임 ZIP 파일 업로드 또는 URL 제공
- 메타데이터 입력 (제목, 설명, 카테고리, 태그)
- 스크린샷/아이콘 업로드
- QA 검토 대기 (1-3일)

**장점**:
- 월 1억+ 플레이어 네트워크
- 자동 광고 삽입
- CPM 기반 수익 분배

**예상 수익**: 월 $50-500

CrazyGames는 월 2천만+ 플레이어를 보유한 대형 웹게임 플랫폼입니다.

**제출할 게임** (3개):

### 7. itch.io 게임 등록

**예상 소요 시간**: 15분  
**목표**: 오가닉 트래픽 확보

itch.io는 인디 게임 커뮤니티 플랫폼으로, 검색 엔진 최적화가 잘 되어 있습니다.

**등록할 게임** (5개):
1. Slime Survivor Premium (뱀서류)
2. Ball Sort Puzzle
3. Sushi Sprint (시뮬레이션)
4. Power 2048
5. Fruit Merge Drop (수이카 게임)

**절차**:
- Creator 계정 생성
- 각 게임 프로젝트 생성
- HTML 임베드 또는 ZIP 업로드
- 가격 설정 (Free 또는 PWYW)
- 태그 최대 10개 설정
- 공개 설정

**장점**:
- 수수료 자유 설정 (0%도 가능!)
- 인디 게이머 타겟
- DevLog로 커뮤니티 소통
- 정기 세일/번들 이벤트

### 8. Ko-fi 후원 설정

**예상 소요 시간**: 3분  
**예상 수익**: 월 $10-50

Ko-fi는 "커피 한 잔 사주기" 스타일의 간편 후원 플랫폼입니다.

**절차**:
1. 계정 생성 (Google 로그인)
2. PayPal 연결 (수수료 0%!)
3. 프로필 커스터마이징
4. 후원 금액 설정 ($3, $5, $10)
5. URL 확인

**장점**:
- 수수료 0% (Gumroad 10% vs Ko-fi 0%)
- 즉시 입금
- 위젯 임베드 가능
- 심리적 부담 낮음

모든 게임 페이지에 "Buy me a coffee" 버튼을 추가할 예정입니다.
