## Executive Summary

딥시크(DeepSeek) V4가 2026년 3월 4일 중국 최대 정치행사인 양회(兩會) 개막에 맞춰 공개된다. 단순한 신모델 출시가 아니다. V4는 1조 파라미터 MoE 아키텍처, 엔그램(Engram) 조건부 메모리, 100만 토큰 컨텍스트 윈도우를 탑재하고 코딩 벤치마크에서 Claude와 GPT-4를 넘어서는 것으로 알려졌다. 더 결정적인 것은 전략적 포지셔닝이다. 딥시크는 엔비디아·AMD에 사전 접근권을 주지 않고 화웨이·캠브리콘에 수주 단위 선행 시간을 제공했다. 이는 AI 모델 전쟁이 칩 생태계 독립 전쟁으로 전환됐음을 의미한다. 한편 중국 AI 모델은 이미 글로벌 최대 LLM API 플랫폼 OpenRouter에서 토큰 소비량 61%를 점령했다. MiniMax M2.5는 Claude Opus 4.6 대비 1/20 가격에 동급 코딩 성능을 제공한다. 한국에서는 역설적으로 Claude 결제액이 1년 새 1131% 폭증(197억 원/월)했고, 법인 비중이 61%에 달해 기업 AI 도입이 본격화되고 있음을 보여준다. Master Jay Lee의 인디 게임·앱 개발 사업에서 AI API 비용 구조가 근본적으로 재편되는 변곡점이 왔다.

## 1. 배경 분석: AI 모델 전쟁의 판이 바뀌고 있다

### 1.1 딥시크가 만들어낸 새로운 서사

2025년은 딥시크가 AI 업계의 전제를 뒤집은 해였다. V3와 R1이 등장하면서 두 가지 고정관념이 깨졌다.

**고정관념 1: 수조 원을 써야 프론티어 LLM을 만들 수 있다**
딥시크 V3는 훈련 비용 600만 달러(약 80억 원)로 GPT-4o급 성능을 달성했다. 이는 OpenAI가 GPT-4에 쏟아부은 것으로 추정되는 수백억 달러와 비교해 수십 배 저렴한 수준이었다.

**고정관념 2: 엔비디아 GPU 없이는 프론티어 AI를 훈련할 수 없다**
미국의 대중 반도체 수출통제(H100, A100 금지)에도 불구하고 딥시크는 제한된 칩 환경에서 혁신적인 알고리즘(MLA, MoE 최적화)으로 성능 격차를 좁혔다.

2026년, V4는 세 번째 전제를 무너뜨리려 한다: **추론(inference) 단계에서도 중국 칩이 미국 칩을 대체할 수 있다.**

### 1.2 양회 타이밍의 정치경제학

딥시크 V4의 3월 4일 공개는 우연이 아니다. 중국의 최대 정치 행사인 양회(전국인민대표대회 + 전국인민정치협상회의)는 중국 정부가 연간 경제·기술 방향을 선언하는 자리다. 2026년 양회에서는 AI 자립과 반도체 내재화가 핵심 의제로 예정돼 있다.

딥시크가 양회 개막 직전 V4를 발표하는 것은 '기술 성과로 정치적 서사를 뒷받침하는' 전형적인 중국식 테크-내셔널리즘 전략이다. 시진핑 정부가 AI를 국가 경쟁력의 핵심으로 설정하고 2030년까지 AI 분야 세계 1위를 목표로 한 상황에서, 딥시크 V4는 단순한 상업적 제품이 아닌 국가 전략 무기에 가깝다.

## 2. 심층 분석: 딥시크 V4의 기술적 혁신과 전략적 의미

### 2.1 아키텍처 분석 — 1조 파라미터의 역설

딥시크 V4에 대해 현재까지 확인된 사실과 누출된 정보를 구분해 살펴보자.

**확인된 사실:**
- **엔그램(Engram) 조건부 메모리** — 2026년 1월 동료 심사(peer-reviewed) 논문으로 발표, 오픈소스 코드 공개 (arXiv:2601.07372)
- **100만 토큰 컨텍스트 윈도우** — 2026년 2월 11일 프로덕션 환경에서 실측 확인

**누출된 정보 (공식 미확인):**
- **총 파라미터 수**: ~1조 (V3: 671B 대비 50% 증가)
- **활성 파라미터 수**: ~32B (V3: 37B보다 오히려 감소 — 효율성 역설)
- **코딩 벤치마크**: HumanEval 90% (Claude 88%, GPT-4 82% 대비 우위)
- **SWE-Bench**: >80%
- **추론 속도**: V3 대비 1.8배 향상

**엔그램 아키텍처의 핵심 혁신:**

기존 LLM은 '파리의 수도는 어디인가'라는 단순한 사실 조회에도 수억 개의 뉴런을 동원한다. 엔그램은 이 비효율을 해결한다. 정적 지식은 DRAM 기반 O(1) 해시 테이블로 처리하고, 동적 추론만 GPU 컴퓨팅을 사용한다. 결과적으로:
- Needle-in-a-Haystack 테스트: 84.2% → 97% (12.8%p 향상)
- 지식 처리 효율: 25% 정적 조회 + 75% 동적 추론의 최적 비율 확인

**총 파라미터 증가 vs 활성 파라미터 감소의 역설:**
V4의 총 파라미터가 1조로 늘었지만 토큰당 활성 파라미터는 37B→32B로 줄었다. 이는 **실제 추론 비용이 V3보다 낮아질 수 있음**을 의미한다. 모델이 더 크고 더 똑똑해지면서 동시에 더 저렴해지는 구조다.

### 2.2 화웨이 칩 최적화 — 칩 전쟁의 새 전선

딥시크가 V4 개발에서 엔비디아·AMD에 사전 접근권을 주지 않고 화웨이·캠브리콘에 수주 단위 선행 시간을 부여한 것은 AI 산업 관행의 역전이다.

**왜 이것이 중요한가?**

전통적으로 AI 기업들은 대형 모델을 출시하기 전 엔비디아에 사전 버전을 제공해 CUDA 생태계에서 최적화된 추론 성능을 확보한다. 딥시크가 이 관행을 뒤집은 것은 다음을 의미한다:

1. **화웨이 Ascend 910C/920 칩의 실제 성능 검증**: 딥시크 V4가 화웨이 칩에서 경쟁력 있는 성능을 낼 경우, 중국 데이터센터들이 엔비디아 GPU를 대체할 근거가 생긴다.

2. **추론 단계의 탈엔비디아**: 훈련에서는 여전히 엔비디아 칩이 우위지만(일부 불법 루트를 통한 H100/블랙웰 접근 의혹 포함), **추론 단계에서 화웨이 칩을 표준화**하면 엔비디아 의존도를 실질적으로 낮출 수 있다.

3. **중국 AI 서비스 독립 생태계 완성**: 모델(딥시크) + 칩(화웨이) + 클라우드(알리바바, 텐센트)의 삼각 구도가 완성되면, 미국의 추가 수출통제도 제한적인 효과를 낼 수 있다.

미국 정부 관계자는 딥시크 V4 개발 과정에서 수출통제 대상인 엔비디아 블랙웰 칩이 사용됐을 가능성을 제기했으나, 딥시크는 이에 대해 공식 입장을 밝히지 않았다.

### 2.3 멀티모달의 도약 — 텍스트를 넘어서

파이낸셜 타임스는 V4가 텍스트뿐 아니라 이미지·영상 생성 기능까지 지원하는 **네이티브 멀티모달 모델**로 공개될 것이라고 보도했다. 이는 단순한 기능 추가가 아니다.

중국 AI 경쟁사들(Moonshot, 알리바바 Qwen, 바이트댄스 Seed)은 이미 멀티모달 플래그십 모델을 보유하고 있다. V4의 멀티모달 지원은 딥시크가 텍스트 중심의 코딩/추론 특화 모델에서 **범용 프론티어 AI**로 진화함을 의미한다.

인디 게임 개발자 관점에서 보면, 딥시크 V4가 오픈소스로 공개될 경우 게임 에셋 생성, 스토리 생성, 코드 자동화를 하나의 모델로 처리할 수 있는 환경이 만들어진다.

## 3. 중국 AI 모델의 OpenRouter 61% 점령 — 가격 전쟁의 실상

### 3.1 숫자가 말하는 충격적 현실

2026년 2월 24일 공개된 OpenRouter 데이터는 AI 모델 시장에서 구조적 전환이 완료됐음을 보여준다:

| 순위 | 모델 | 개발사(국적) | 주간 토큰 소비 | 전주 대비 |
|------|------|------------|--------------|---------|
| 1 | MiniMax M2.5 | MiniMax (중국) | 2.45조 | +197% |
| 2 | Kimi K2.5 | Moonshot AI (중국) | 1.21조 | -20% |
| 3 | GLM-5 | 智谱AI (중국) | 780억 | +158% |
| 4 | (비공개 미국) | — | — | — |
| 5 | DeepSeek V3.2 | DeepSeek (중국) | — | — |

**중국 모델 합계: 5.3조 토큰 (전체 상위 10개 8.7조의 61%)**

### 3.2 가격 전쟁의 실체

중국 모델의 압도적 점유율 이면에는 극단적인 가격 우위가 있다:

| 모델 | 입력 ($/1M 토큰) | 출력 ($/1M 토큰) | SWE-Bench |
|------|----------------|----------------|----------|
| MiniMax M2.5 | $0.30 | $1.10 | 80.2% |
| GLM-5 | $0.30 | $2.55 | — |
| Claude Opus 4.6 | $5.00 | $25.00 | 80.8% |
| 가격 차이 | **17배** | **23배** | **0.6%p** |

MiniMax M2.5는 Claude Opus 4.6 대비 17~23배 저렴하면서 SWE-Bench(소프트웨어 엔지니어링) 성능은 80.2% vs 80.8%로 사실상 동급이다. 개발자 관점에서 합리적 선택은 자명하다.

### 3.3 구조적 변화 — 코딩이 AI의 주력 용도가 됐다

OpenRouter COO 크리스 클라크의 발언이 핵심을 찌른다: "중국 오픈웨이트 모델들은 미국 기업들이 운영하는 에이전틱 플로우에서 불균형적으로 많이 사용되고 있다."

OpenRouter 내 코딩 관련 토큰 사용 비율: **11%(2025년 초) → 50%+(현재)**
에이전트 자동화 흐름이 전체 출력 토큰의 절반 이상을 차지

이것이 의미하는 바: AI는 더 이상 인간이 직접 대화하는 챗봇이 아니다. 코딩 에이전트가 24시간 API를 호출하는 자동화 파이프라인이 토큰 소비의 주력으로 부상했다. 이 분야에서 가격이 1/20인 중국 모델의 우위는 지속 가능하다.

## 4. 한국의 역설: Claude 1131% 폭증이 말하는 것

### 4.1 수치의 맥락

글로벌에서 중국 AI가 OpenRouter를 점령하는 동안, 한국에서는 정반대의 현상이 나타났다:

- Claude 국내 결제액 (2026.2.1~22): **197억 원** (전년 동기 16억 원 → **+1131%**)
- 법인카드 비중: **61%** (ChatGPT 45% 대비 16%p 높음)
- 건당 결제액 추이: 4만 2,600원(2025.1) → 9만 6,400원(2025.12) → **10만 6,000원(2026.2)**
- 시장 점유율: ChatGPT(51.2%) → Claude(29.0%) → Gemini(14.3%)

### 4.2 왜 한국 기업들은 Claude를 선택하나?

**Claude Code + Claude Cowork의 기업 침투:**
2025년 5월 공개된 Claude Code와 2026년 1월 출시된 Claude Cowork가 기업 코딩·업무 자동화 수요를 흡수했다. 한국 IT 기업들이 SW 개발 생산성 향상을 위해 Claude를 팀 단위로 도입하면서 법인 결제가 급증했다.

**ChatGPT 대비 차별점:**
- 긴 컨텍스트 처리 능력 (코드 리뷰, 대형 프로젝트 분석)
- 코드 생성 품질 (특히 복잡한 로직 구현)
- 기업용 API 안정성

### 4.3 한국 AI 시장의 구조적 의미

2026년 국내 7개 AI 서비스 합산 결제액 연환산: **2조 268억 원** (첫 2조 돌파)

이 시장은 빠르게 성숙하고 있다. 재구독률 80%(쿠팡 수준)는 AI 도구가 이미 기업 인프라에 필수재로 자리 잡았음을 의미한다. 기업들이 AI에 지출하는 비용은 앞으로 줄어들기 어렵다.

## 5. 시나리오 분석: V4 이후 AI 생태계의 세 가지 미래

### 시나리오 A — Best Case: 오픈소스 혁명 가속

**조건**: 딥시크 V4가 예상 성능 그대로 오픈소스로 공개되고, 화웨이 칩에서도 경쟁력 있는 추론 성능을 입증

**결과**:
- Hugging Face, Ollama 등 오픈소스 생태계에서 V4 기반 파생 모델 폭발적 증가
- 로컬 실행 가능한 1T 파라미터급 모델의 시대 개막 (RTX 5090 / Apple Silicon 클러스터)
- API 비용이 현재의 1/5~1/10 수준으로 하락
- **인디 개발자에게 최상의 환경**: 게임 NPC 대화, 동적 스토리 생성, 코드 자동화 비용이 사실상 0에 수렴

**확률**: 40%

### 시나리오 B — Base Case: 이중 구도 고착

**조건**: V4가 예상대로 출시되지만 서방 시장에서의 규제·안보 우려로 채택이 분화

**결과**:
- 중국/동남아 시장: DeepSeek V4 + 화웨이 칩 생태계 표준화
- 서방 시장: Claude/GPT-4 계열 + OpenRouter에서 저비용 중국 모델의 코딩 에이전트 병용
- API 비용: 중간 수준 유지 (클로드 고급 추론 + MiniMax 저비용 자동화의 이중 구도)
- **인디 개발자**: 용도에 따른 모델 분리 전략이 최적

**확률**: 45%

### 시나리오 C — Worst Case: AI 철의 장막

**조건**: V4 출시 이후 미국 정부가 딥시크 API 사용 금지 또는 오픈소스 모델 규제 강화

**결과**:
- OpenRouter에서의 중국 모델 접근이 규제 대상이 됨
- API 비용 다시 상승 (공급 경쟁 감소)
- 한국 기업들은 Claude/Gemini/GPT의 과점 체제에 다시 종속
- **인디 개발자**: 비용 상승 압박 + 대안 부재의 이중고

**확률**: 15%

## 6. Master Jay Lee에게 미칠 영향 분석

### 6.1 인디 게임 개발 비용 구조

현재 게임/앱 개발에서 AI API 비용은 다음 용도로 쓰인다:
- **코드 생성**: Godot GDScript, Rust/WASM 자동화
- **게임 내 NPC 대화**: 동적 텍스트 생성
- **마케팅 카피**: 앱스토어 설명, 스크린샷 텍스트
- **에셋 설명**: Blender 스크립트 생성

딥시크 V4 오픈소스 공개 + MiniMax급 저가 모델 확산 시, 이 비용들이 현재의 1/10~1/20 수준으로 낮아질 수 있다.

**구체적 수치**: Claude Opus 4.6으로 월 10만 토큰 사용 시 약 $27.5. MiniMax M2.5로 대체 시 $1.4. 에이전트 자동화 수준에서 100배 이상 차이가 난다.

### 6.2 Switch 2 인디 게임 기회 연계

Nintendo Indie World Showcase가 오늘(3월 3일 23:00 KST) 생방송된다. 이 쇼케이스에서 Switch 2 대응 인디 게임 라인업이 공개되면, 포팅 기회와 마켓 방향성이 명확해진다.

Ampere Analysis에 따르면 Switch 2는 2026년 말까지 3,100만 대 보급, 콘텐츠 기회 총액 **70~80억 달러(약 10조 원)**로 예측된다. 인디 게임 개발자에게 Switch 2는 역대급 시장 기회다.

여기서 AI 비용 절감과 Switch 2 기회가 교차한다: **저비용 AI 코딩 에이전트로 개발 속도를 높이면서 Switch 2 플랫폼의 기회를 빠르게 포착하는 전략**이 가능하다.

### 6.3 클로드 한국 결제 급증의 신호

Claude의 국내 1131% 폭증은 한국 기업들이 AI 코딩 도구에 월 10만 원 이상을 지출할 의지가 있음을 보여준다. 이는 역으로:
- AI 기반 개발 도구/서비스를 한국 기업에 B2B로 판매하는 시장이 있음을 의미
- Claude Code 유사 개발 자동화 서비스의 국내 수요가 증명됨
