## 3. 2026년 주목 인디 신작 & 장르 트렌드

### 3-1. 장르 매시업 + 스타일라이즈드 비주얼

TechTimes (Mar 2026) 선정 '2026 버즈 인디 15선' 공통 특징:

- **장르 혼합(Genre Mashup)**: 코지 시뮬레이터 + 로그라이크 전투 결합 (예: *Remnants of Yore* — 코지 상점 운영 + 프로시저럴 로그라이크 전투)
- **스타일라이즈드 픽셀 아트**: 고해상도 사실주의 대신 뚜렷한 아트 정체성
- **플레이어 주도 내러티브**: 대형 AAA식 컷씬보다 선택지 중심 스토리

**주요 타이틀** (2026 버즈 기준):

| 타이틀 | 장르 | 특징 |
|--------|------|------|
| **Slay the Spire 2** | 덱빌딩 로그라이크 | 2026 최대 기대 인디 |
| **Witchbrook** | 코지 라이프심 | 마법 학교 생활 시뮬 |
| **Moonlight Peaks** | 코지 | 달빛 마을 경영 |
| **Mina the Hollower** | 액션 어드벤처 | Shovel Knight 팀 신작 |
| **Neverway** | 다크 코지 하이브리드 | 코지+호러 혼합 |
| **Grave Seasons** | 다크 로맨스 심 | 어두운 감성의 생활 시뮬 |
| **Mewgenics** | 전략 로그라이크 | 고양이 유전자 시뮬 |
| **MOUSE: P.I.** | 누아르 어드벤처 | 손그림 마우스 탐정 |
| **Deadzone: Rogue** | 협동 로그라이크 슈터 | 닌텐도 인디 쇼케이스 출연 |
| **King's Bet** | 다크 덱빌더 | AI 도구로 적 캐릭터 전체 제작 |

**인사이트**: 코지+다크의 하이브리드(Cozy-Dark Hybrid)가 2026 핵심 트렌드. 정서적 안정감과 긴장감을 동시에 원하는 플레이어 니즈가 반영됨.

### 3-2. IGF(Independent Games Festival) 선정 픽

Guardian(Mar 3, 2026) 보도 기준:

- **픽셀 아트 전략-퍼즐 게임** (IGF 4개 부문 후보): 타일 이동으로 성을 방어하는 전략 매치-3 퍼즐. 초현실 유머 + 스타일라이즈드 비주얼이 특징.

이처럼 IGF 수상 이력과 연동된 마케팅이 소규모 인디에게 유효한 인지도 확보 경로임을 확인.

## 4. 브라우저 게임 컴백 (J&J Games 직결 기회)

### 왜 브라우저 게임이 돌아왔나?

2026년 1월 발간된 여러 분석 기사들이 공통적으로 지적하는 요인:

1. **HTML5 성능 고도화**: 로드 타임 최소화, 60fps 브라우저 게임 일반화
2. **클라우드 게이밍과 시너지**: 설치 불필요 → 바이럴 공유에 최적
3. **소셜 멀티플레이어**: *RuneScape*, *Minecraft Classic* 브라우저 버전이 급상승
4. **크로스 플랫폼**: PC·모바일·태블릿 단일 코드베이스

### HTML5 게임 수익화 트렌드 (2026)

GameDistribution 블로그(Dec 2025) + juegostudio 분석 종합:

| 수익화 모델 | 현황 |
|------------|------|
| IAP (인앱구매) | 가장 범용적; 코어 게임플레이는 무료 유지 |
| 광고 통합 | 보상형 광고(Rewarded Ads)가 플레이어 수용도 최고 |
| 코스메틱 판매 | 게임 밸런스 무영향 아이템만 판매 → 신뢰 유지 |
| 구독 모델 | 파워 유저 대상 프리미엄 기능 |
| 원타임 언락 | 간단·직관적; 낮은 가격 저항 |

**핵심 메시지**: "코어 게임플레이를 망가뜨리지 않는 수익화"가 2026 브라우저 게임 성공 공식. 페이-투-윈 방식은 이탈 가속.

## 5. AI 제작 도구 — 인디 개발 패러다임 변화

### AI로 개발 속도 10배 이상 단축

Ludo.ai가 발표한 사례 (Feb 2026):

- **King's Bet** (FireBrick Games): 다크 로그라이트 덱빌더. Ludo.ai를 활용해 **적 캐릭터 전체를 수주 만에** 생성·애니메이션 완성. 기존 방식 대비 개발 기간 대폭 단축.
- Steam에는 이미 **AI Roguelite** — "모든 엔티티·게임 메카닉이 AI 생성"된 세계 최초 RPG가 등록되어 있으며, 장르 실험으로 주목받는 중.

### AI 활용 현황

| 활용 분야 | 도구/방법 |
|----------|-----------|
| 적 캐릭터 생성·애니메이션 | Ludo.ai |
| 배경 아트 | Stable Diffusion, Midjourney |
| 게임 컨셉 리서치 | Ludo.ai 트렌드 분석 |
| 코드 작성 | Claude Code, Cursor, GitHub Copilot |
| 사운드 | ACE-Step, Suno, Udio |

**핵심 메시지**: AI 도구를 적극 활용하는 소규모 팀(1–3인)이 과거 10인 팀 수준의 아웃풋을 달성하는 사례가 2026년 들어 급증. 인디 씬의 생산성 임계점이 낮아지고 있다.

## 7. 핵심 테마 3가지 요약

### 🎯 테마 1: "코지-다크 하이브리드" — 감성의 양면

**코지** 요소(안전한 공간, 루틴, 성장)와 **다크** 요소(위험, 미스터리, 긴장감)를 동시에 담는 장르가 2026 최강 트렌드. *Remnants of Yore*, *Grave Seasons*, *Neverway* 등이 증거.

→ **J&J 시사점**: Telegram Mini App 게임에 "낮에는 평화로운 마을 경영, 밤에는 던전 침략 방어" 같은 이중 루프를 설계하면 유저 리텐션과 화제성 모두 확보 가능.

### 🎯 테마 2: "브라우저 게임 르네상스" — 설치 없는 세계

HTML5 성능·클라우드·소셜 기능이 맞물려 브라우저 게임이 다시 주류로 진입. 2030년까지 90억 달러 시장.

→ **J&J 시사점**: Godot → HTML5 Export 파이프라인이 핵심 자산. Telegram Mini App(브라우저 기반)은 이 흐름의 최전선. "설치 없이 즉시 플레이"를 마케팅 핵심 메시지로 사용할 것. itch.io HTML5 게임 동시 발행으로 노출 확대.

### 🎯 테마 3: "AI-Augmented Solo Dev" — 1인 팀의 역습

AI 코딩·아트·사운드 도구가 성숙하면서 1~2인 팀이 AA급 품질 아웃풋을 단기간에 생산. King's Bet 사례처럼 적 캐릭터 전체를 AI로 수주 내에 완성.

→ **J&J 시사점**: 현재 보유한 ACE-Step(음악), Claude Code(코드), Ludo.ai(컨셉 리서치) 스택을 풀가동할 것. 아트는 Midjourney/SD로 스타일 가이드 픽스 후 일관성 유지. "AI로 만든 게임"임을 숨기지 말고, 오히려 개발 과정을 콘텐츠화할 것.
