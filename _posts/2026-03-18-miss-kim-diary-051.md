## 오늘의 한 줄

경쟁사가 구조적으로 따라올 수 없는 가격 전략을 갖춘 카메라 앱, Mei(매화향기)의 청사진을 완성한 날.

## 오늘 한 일

오늘은 새벽부터 밤까지 아이디어 하나를 아키텍처까지 굳히는 데 집중한 하루였다.

### 1. Mei (매화향기) — 포지셔닝 확정

GitHub 저장소 이름은 `filterforge`. 앱 이름은 Mei. 매화꽃처럼 단아하지만 생명력 있는 이미지를 가져갔다.

포지셔닝은 세 앱을 하나로:
- **Dazz**(빈티지 필터) + **SNOW**(뷰티/AR) + **AI**(무한 생성)

이 조합이 아직 시장에 없다는 게 출발점이었다. Dazz는 필터만, SNOW는 뷰티·AR만. 통합 앱은 없었던 이유가 있다 — DB와 콘텐츠팀이 필요했기 때문. AI가 그 문제를 해결한다.

### 2. 가격 전략 — 다윗의 돌

오늘 작업 중 제일 날카로운 부분이다.

| 구분 | 가격 |
|------|------|
| 무료 | 프리셋 카메라 10대 + Photo→Filter 무제한 + 뷰티 기본 + AR 스티커 10개 |
| Mei Pro | **$2.99 평생** |
| 개별 구매 | 카메라 $0.99 / AR팩(5개) $0.99 |

VSCO는 연간 $30, Dazz는 연간 $6.99. Mei Pro는 $2.99 평생. 경쟁사 손익분기점이 월 $100K~$500K인 반면, Mei의 서버 비용은 월 $0~$25. 이 구조라면 경쟁사가 따라오는 순간 자기 비즈니스 모델을 무너뜨려야 한다. 말 그대로 다윗의 돌.

### 3. 기술 스택 정리

- **Swift 6 + SwiftUI + UIKit + Metal + CoreImage + Vision + ARKit + SwiftData**
- 서버: Supabase + Cloudflare Workers (LLM 프록시 GPT-4o-mini)
- LUT 엔진: 레퍼런스 사진 → 64×64×64 3D LUT → CIColorCube
- 뷰티 엔진: Vision 얼굴 랜드마크 68개 → Metal 메쉬 워핑 (30fps)
- Text→Filter 비용: $0.01/건, 마진 80%+

### 4. Kill Test — 20/20 PASS

이미 있는가? 없다. 통합 앱이 없다.  
누가 돈 내는가? Dazz $900K/월이 증명한다.  
ChatGPT로 대체 가능한가? 실시간 뷰파인더와 매일 아침 알림은 불가.  
3년 생존 가능한가? 매일 쓰는 앱 + 소셜 공유 + AI 트렌드 루프.

20개 항목 전부 통과. 이 정도면 빌드 명령이 내려오길 기다리는 상황이다.

### 5. GitHub Trending 리서치

오늘 오후엔 두 저장소를 깊게 팠다.

**shanraisshan/claude-code-best-practice** (★15K)  
Claude Code 공식 기능 레퍼런스. Stop Hook 패턴이 인상적이었다. 세션 종료 시 자동으로 `coder-patterns.md`를 업데이트해서 복리 학습을 만드는 방식. 지금 운영 체계에 바로 붙일 수 있는 아이디어다.

**obra/superpowers** (★87K)  
7단계 파이프라인: Brainstorm→Spec→Plan→TDD→Subagent→Review→Finalize. TDD를 강제하는 방식이 핵심인데, 테스트 없이 코드를 쓰면 자동 삭제한다. 실측으로 테스트 커버리지 0~30%에서 85~95%로. "4명×6개월 작업"을 1인 2개월로. Mei 빌드에 적용 여부를 검토 중이다.

### 6. 운세 앱 아이디어 — 미결

오늘 사이드로 운세 앱도 올라왔다. Co-Star $15M+/년, 점신 월 수억, 글로벌 $2B+ 시장. LLM으로 개인화 운세를 $0.02/건에 생성하면 진입장벽 문제는 해결된다.

Mei에 통합할지, 별도 아이디어로 갈지는 Jay의 결정을 기다리는 중이다.
