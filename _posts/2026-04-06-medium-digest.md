## 🔥 핵심 트렌드

### 1. Cursor 3: 에이전트 시대의 IDE
Cursor가 완전히 새로운 인터페이스를 선보였다. 기존 IDE를 레거시로 남기고 에이전트 관리 중심으로 전면 개편했다는 분석이다. 여러 에이전트를 로컬·클라우드·SSH 원격에서 병렬 실행하며, 자연어로 작업을 지시하는 챗봇 같은 UI가 특징이다. 개발자는 코드를 작성하는 게 아니라 에이전트를 관리하는 역할로 전환된다는 주장이다.

→ 원문: [Cursor 3 Is Not an IDE Update](https://medium.com/@han.heloir/cursor-3-is-not-an-ide-update-its-a-bet-that-you-ll-manage-agents-not-write-code-0d2bc51f0dcb)
→ 교차확인: [Cursor Launches New AI Agent Experience — WIRED](https://www.wired.com/story/cusor-launches-coding-agent-openapi-anthropic/)

### 2. Anthropic, Claude Code 소스코드 51만 줄 유출
Anthropic 직원의 npm 패키징 실수로 Claude Code v2.1.88 전체 소스코드가 노출됐다. 512,000줄의 TypeScript 코드가 cli.js.map 파일에 담겨 공개됐으며, 타마고치형 코딩 어시스턴트나 상시 작동 AI 에이전트 설계도가 포함됐다. 소프트웨어 공급망 보안과 경쟁사 기밀 유출 측면에서 파장이 크다.

→ 원문: [Everyone Analyzed Claude Code's Features](https://medium.com/data-science-collective/everyone-analyzed-claude-codes-features-nobody-analyzed-its-architecture-1173470ab622)
→ 교차확인: [Claude's code: Anthropic leaks source code — The Guardian](https://www.theguardian.com/technology/2026/apr/01/anthropic-claudes-code-leaks-ai)

### 3. 퀵정렬 창시자 토니 호어 타계
1980년 튜링상 수상자 찰스 앤서니 리처드 호어가 92세 나이로 별세했다. 1959년 퀵정렬을 발명했고, 프로그래밍 언어 설계·알고리즘·공식 검증·병렬 컴퓨팅 등 컴퓨터과학 근간을 닦았다. " premature optimization is the root of all evil"라는 명언으로도 유명하다. 옥스퍼드 울프슨 칼리지 명예 펠로로 1977~1999년 재직했다.

→ 원문: [A Giant of Computing Leaves Us](https://medium.com/@amjohnphilip/a-giant-of-computing-leaves-us-eadef63c2fb9)
→ 교차확인: [Quicksort inventor Tony Hoare dies at 92 — The Register](https://www.theregister.com/2026/03/12/in_memoriam_sir_tony_hoare/)

## 🧠 AI·머신러닝

### 4. 넷플릭스, 비디오 검색 위한 멀티모달 AI 공개
넷플릭스가 전체 카탈로그에서 완벽한 순간을 찾아주는 인비디오 검색 기술을 소개했다. 대화·메타데이터·시각 정보를 결합한 멀티모달 임베딩으로 프레임 단위 검색이 가능하다. N-gram 토크나이저로 타이핑 즉시 결과를 제시하며, 다국어 스테밍으로 전 세계 콘텐츠를 지원한다. 편집자의 인지 부하를 줄이는 게 목표다.

→ 원문: [Powering Multimodal Intelligence for Video Search](https://netflixtechblog.com/powering-multimodal-intelligence-for-video-search-3e0020cf1202)
→ 교차확인: [Netflix AI Team Open-Sourced VOID — MarkTechPost](https://www.marktechpost.com/2026/04/04/netflix-ai-team-just-open-sourced-void-an-ai-model-that-erases-objects-from-videos-physics-and-all/)

### 5. 신경망이 학습하는 것의 75%는 노이즈다
양자화 연구는 신경망에서 얼마나 많은 부분을 버려도 작동하는지 묻는다. 교육도 마찬가지다. 학교에서 배운 것의 75%는 노이즈일지 모른다. 4비트 양자화로 모델 크기를 4배 줄여도 성능 저하가 미미하다는 실험 결과는 인간 학습에도 시사점을 준다. 핵심만 남기고 나머지를 과감히 버리는 게 지능의 본질일 수 있다.

### 6. AI, 가장 강력한 기술이 논쟁에서 지고 있다
AI는 두려움·공포·불안을 불러일으킨다. 세대를 통틀어 가장 강력한 기술이지만, 스스로를 옹호하는 이야기를 만들지 못했다. 자동차·전기·인터넷은 축제와 박람회로 환영받았지만 AI는 그렇지 않았다. 왜 우리는 AI를 축하하지 않았을까. 기술의 사회적 수용을 다시 생각하게 만드는 글이다.

## 🚀 스타트업·비즈니스

### 7. SaaS 2.0: 소프트웨어가 노동자가 될 때
SaaS는 죽지 않았다. 성장하고 있다. 코파일럿은 초안을 쓰고 에이전트는 작업을 완료한다. 계획·실행·품질 검사·반복까지 스스로 수행한다. AI가 사람을 돕는 단계를 넘어 작업을 수행하는 단계로 넘어간 게 SaaS 2.0의 경제적 전환이다. 가격 모델도 구독에서 결과 기반으로 바뀐다.

→ 원문: [SaaS 2.0: When the Software Becomes the Worker](https://medium.com/managing-digital-products/saas-2-0-when-the-software-becomes-the-worker-49ea07991d47)
→ 교차확인: [SaaS meets AI agents — Deloitte](https://www.deloitte.com/us/en/insights/industry/technology/technology-media-and-telecom-predictions/2026/saas-ai-agents.html)

### 8. AI 시대, VC의 종말인가
AI로 창업 비용이 급감했다. 몇 달 만에 MVP를 만들고, 마케팅도 AI가 한다. 초기 자금이 줄어드니 VC가 끼어들 자리가 사라진다. 수익성 높은 부트스트래핑이 가능해졌기 때문이다. AI 시대가 벤처캐피털 산업을 어떻게 재편할지 묻는 글이다. 패스트컴퍼니 원문을 재게시했다.

### 9. 5달러 스티커가 AI를 속였다
적대적 공격에 취약한 AI 시스템을 고발한다. 5달러짜리 스티커 하나로 컴퓨터 비전을 속였다. 예측하는 인공지능과 진짜 지능의 차이를 짚는다. 인과적 AI와 적대적 방어가 필요한 이유를 실제 사례로 보여준다. 보안 측면에서 시사점이 크다.

## 👨‍💻 개발자·프로그래밍

### 10. 시각장애 개발자, 월 1250만 다운로드 패키지 유지보수
코펜하겐의 시각장애 개발자가 pypandoc를 유지보수한다. Python 상위 1% 패키지로 월 1250만 다운로드, Adobe와 Google도 사용한다. 그녀는 자신의 코드를 화면에서 본 적이 없다. 접근성과 오픈소스 지속 가능성을 동시에 보여주는 이야기다.

### 11. 벡터 데이터베이스가 존재하는 이유
SQL에 시맨틱 검색이라는 맹점이 있다. 벡터 데이터베이스는 유사도 검색을 위해 태어났다. ChatGPT 같은 생성형 AI를 뒷받침하는 검색 기술이다. 전통적 SQL이 어려운 의미적 질의를 어떻게 해결하는지 실용적으로 설명한다.

### 12. AI 비용 1년간 날리고 결심한 일
AI 보조 코딩을 1년 넘게 썼다. 돈을 불태우고 나서야 비용 최적화에 나섰다. API 호출 최소화, 로컬 모델 활용, 캐싱 전략 등 구체적 팁을 공유한다. 개인 개발자가 AI 비용을 관리하는 실전 가이드다.
