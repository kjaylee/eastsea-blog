## 한 줄 소개

> **골든 아케이드(Golden Arcade)** — 코인으로 연결된 5가지 아케이드 게임을 하나의 로비에서 즐기는 통합 게임 컬렉션. 각각 다른 장르, 같은 경제 시스템.

---

## 골든 아케이드란?

골든 아케이드는 5개의 독립적인 아케이드 게임을 **공유 코인 경제(goldenArcadeCoins)**로 연결한 웹 게임 컬렉션입니다.

- **1회 시작 시 500코인** 무료 지급
- 게임 간 코인 공유 — 어떤 게임에서 벌어도 다른 게임에서 사용 가능
- 파산 시 **자동 500코인 구제** — 끊임없이 플레이 가능
- 모든 게임 **모바일 최적화**, 브라우저에서 즉시 실행

👉 **플레이하기:** [games.eastsea.xyz/arcade-lobby](https://games.eastsea.xyz/arcade-lobby/)

## 5가지 게임 소개

### 🐎 경마왕 (Horse Racing Derby)

| 항목 | 내용 |
|------|------|
| 장르 | 스포츠 / 배팅 시뮬레이션 |
| 플레이 인원 | 1인 |
| 코인 경제 | 입장료 차감 → 배당금 획득 |
| 핵심 매력 | 6마리 말 중 실시간 배당 변화, 짧은 라운드 30초 |

8개 레이스코스를 전력으로 달리는 말 중 하나에 베팅합니다. 레이스가 시작되면 실시간 순위와 배당금이 변동하며, 짧고 긴 라운드를 자유롭게 선택할 수 있습니다.

👉 **플레이하기:** `games/horse-racing-derby/index.html`

---

### 🎲 다이스 킹 (Dice King)

| 항목 | 내용 |
|------|------|
| 장르 | 주사위 / 배팅 |
| 플레이 인원 | 1인 |
| 코인 경제 | goldenArcadeCoins 완전 연동 |
| 핵심 매력 | 다양한 배팅 옵션, 빠른 한 판 |

주사위 눈을 예측하고 베팅하는 클래식 다이스 게임입니다. 출금 배율과 리스크를 직접 조절하며, goldenArcadeCoins를 자유롭게 입출금하며 플레이합니다.

👉 **플레이하기:** `games/dice-king/index.html`

---

### 🎰 슬롯 로그 (Slot Rogue)

| 항목 | 내용 |
|------|------|
| 장르 | 슬롯머신 / 로그라이크 |
| 플레이 인원 | 1인 |
| 코인 경제 | 입장료 200코인, 골드×10 환급, 승리 보너스 +500 |
| 핵심 매력 | 로그라이크 강화 시스템 + 슬롯 스핀 |

슬롯머신의 스릴과 로그라이크의 성취감을 결합한 하이브리드 게임입니다. 입장료 200코인을 내고 시작하면, 골드 조합에 따라 최대 10배 보너스와 승리 보너스 +500코인을 획득할 수 있습니다.

👉 **플레이하기:** `games/slot-rogue/index.html`

---

### 🃏 카드 워 (Card War)

| 항목 | 내용 |
|------|------|
| 장르 | 카드 게임 / 더블-오어-노딩 |
| 플레이 인원 | 1인 |
| 코인 경제 | goldenArcadeCoins 완전 연동, 파산 시 500코인 자동 구제 |
| 핵심 매력 | 5단계 베팅(50/100/250/500/ALL IN), WAR 더블 베팅 |

52장 표준 덱을 사용한 카드 대결 게임입니다. 베팅액을 선택하고 카드를 뽑아 높은 카드가 승리합니다. **동점 시 WAR 발생** — 베팅액이 2배로 올라가는 긴장감 있는 순간이 기다립니다.

5단계 베팅 중 올인(ALL IN) 버튼으로 전액 베팅도 가능하며, 파산하면 자동으로 500코인이 지급되어 즉시 재도전할 수 있습니다.

👉 **플레이하기:** `games/card-war/index.html`

### 🎵 리듬 펄스 (Rhythm Pulse) — **NEW**

| 항목 | 내용 |
|------|------|
| 장르 | 리듬 게임 / 타이밍 |
| 플레이 인원 | 1인 |
| 코인 경제 | goldenArcadeCoins 완전 연동, 점수 기반 코인 보상 |
| 핵심 매력 | 3레인 비트 시스템, 콤보 보너스, 모바일 터치 최적화 |

비트에 맞춰 탭하는 3레인 리듬 게임입니다. 정확한 타이밍으로 노트를 때리면 콤보가 올라가고, 높은 콤보 보너스로 goldenArcadeCoins를 추가로 획득할 수 있습니다. 모바일 브라우저에서 손가락으로 바로 플레이할 수 있어 어디서나 리듬의 재미를 누릴 수 있습니다.

👉 **플레이하기:** `games/rhythm-pulse-prototype/index.html`

## 공유 코인 시스템 — 하나의 지갑, 다섯游戏的 세계

```
경마왕 → 코인 획득 → 다이스 킹에서 사용
다이스 킹 → 코인 획득 → 슬롯 로그에서 사용
슬롯 로그 → 코인 획득 → 카드 워에서 사용
카드 워 → 코인 획득 → 리듬 펄스에서 사용
리듬 펄스 → 코인 획득 → 다시 경마왕으로...
```

**goldenArcadeCoins** 키 하나로 5개 게임 전부가 동기화됩니다. 코인을 벌어도 지갑 잔액이 늘어나며, 어떤 게임에서 시작하더라도 동일한 재화로 전체 컬렉션을 탐험할 수 있습니다.
