## 한 줄 소개

> **마작 선(Mahjong Zen)** — 전통 마작 타일을 현대적 젠 미학으로 재해석한 타일 매칭 솔리테어. 20개 이상의 레벨과 프로그레시브 난이도로, 긴장감보다는 평화로운 명상에 빠져드는 퍼즐 경험을 제공한다.

## 마작 선이란?

마작 선은 클래식 마작 솔리테어의 전략성과 현대적 미니멀 디자인을 결합한 캐주얼 퍼즐 게임입니다.

- **자유 타일 매칭:** 좌우 중 한 칸이라도 비어 있는 타일만 선택 가능
- **禅 미학:** 느린 그라디언트 배경, 부드러운 타일 애니메이션, 클리어 시 명언 표시
- **Golden Arcade Coins 연동:** 레벨 클리어 시 코인 보상, 힌트/셔플로 코인 소모
- **모바일 최적화:** 터치 기반 조작, 어디서나 5~15분 세션

## 플레이 방법

### 1단계: 타일 이해
마작 타일은 **6종 심볼**로 구분됩니다.

| 심볼 | 의미 | 색상 |
|------|------|------|
| 🎋 대나무 | 자연/성장 | 초록 #4a7c59 |
| 🪙 동전 | 재물/부 | 금색 #c9a959 |
| 💎 보석 | 빛/가치 | 파랑 #5b8fb9 |
| 🌸 꽃 | 아름다움 | 분홍 #c97b84 |
| 🔥 불 | 열정/에너지 | 빨강 #d45d47 |
| 🌊 물 | 평화/유연함 | 청록 #4a90b5 |

### 2단계: 자유 타일 판정
**자유 타일**이란 — 좌 또는 우 중 한 칸이라도 비어있고, 위에 다른 타일이 올라가지 않은 타일입니다.
화면에서 **약간 앞으로突出的** 보이고, 탭 가능 상태입니다.

### 3단계: 매칭하고 제거하기
1. 자유 타일 중 **같은 심볼** 2개를 연속으로 탭합니다.
2. 매칭 성공 시 → 타일이 스케일업 → 파티클 → 페이드아웃 (300ms).
3. 점수가 오르고, 새로운 자유 타일이诞生합니다.

### 4단계: 보조 도구 활용
| 도구 | 기능 | 제한 |
|------|------|------|
| 💡 힌트 | 매칭 가능한 쌍 하이라이트 | 3회/게임 |
| 🔄 셔플 | 남은 타일 무작위 재배치 | 1회/게임 |
| ↩️ 실행취소 | 마지막 매칭 되돌리기 | 무제한 |

### 5단계: 레벨 클리어
모든 타일을 제거하면 클리어. 시간과 힌트 사용 횟수에 따라 ★1~3개를 부여받습니다.

## Golden Arcade Coins 경제 시스템

마작 선은 **Golden Arcade 컬렉션**의 공유 코인 시스템과 완전 연동됩니다.

| 행동 | 코인 변화 |
|------|----------|
| 레벨 클리어 (★1) | +20코인 |
| 레벨 클리어 (★2) | +50코인 |
| 레벨 클리어 (★3) | +100코인 |
| 힌트 사용 | -10코인 |
| 셔플 사용 | -20코인 |
| 파산 시 | 자동 500코인 구제 |

500코인으로 시작하면 약 **10~25판** 플레이 가능. 파산해도 즉시 재도전할 수 있습니다.

## 비주얼 & 사운드

**색상 팔레트:** 딥 네이비(#222) → 인디고(#162) 그라디언트 배경
**타일:** 아이보리/크림색 베이스에 심볼 색상별 매핑
**애니메이션:**
- 타일 선택 시 글로우 + 미세 바운스
- 매치 성공 시 스케일업 → 파티클 → 페이드아웃
- 레벨 클리어 시 물결 효과 + 축하 파티클
**클리어 시 명언:**禅學者-inspired 격언 10종 랜덤 표시
