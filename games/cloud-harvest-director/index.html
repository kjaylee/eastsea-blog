Cloud Harvest Director — 무료 브라우저 게임

Cloud Harvest Director은(는) 브라우저에서 바로 플레이할 수 있는 무료 HTML5 시뮬레이션 게임입니다. 다운로드 없이 PC와 모바일 모두에서 즐길 수 있습니다.

🎮 조작 방법

  • 방향키 또는 WASD로 이동 및 조작
  • 마우스 클릭 또는 화면 탭으로 조작

✨ 게임 특징

  • 100% 무료 — 설치 없이 브라우저에서 바로 플레이
  • 점수 시스템으로 실력을 기록하고 최고 기록에 도전하세요
  • 자동 저장 — 언제든 이어서 플레이
  • PC와 모바일 모두 지원

📖 게임 소개

자원을 수집하고 시설을 건설하는 경영 시뮬레이션 게임입니다. 전략적 판단으로 번영하는 제국을 만들어 나가세요.
