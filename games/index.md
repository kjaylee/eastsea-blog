# Eastsea Games Index (Markdown)

- [Tile Flip — 양면 타일 뒤집기 전략](./tile-flip/) — 타일 클릭 시 상하좌우가 함께 반전되는 규칙에 🔒고정·💫대각·🔄전파 특수 타일을 더해, 3x3~6x6의 12개 레벨을 최소 클릭으로 통일하는 네온 로직 퍼즐.
- [Frequency — 주파수 매칭 퍼즐](./frequency/) — 타겟 파형(사인/사각/삼각/톱니)과 플레이어 파형을 주파수·진폭·위상 슬라이더로 맞추고, 2~3파 additive synthesis까지 공명시키는 네온 사운드 퍼즐.
- [Decay — 시간 붕괴 퍼즐](./decay/) — 밟은 발판이 2초 뒤 붕괴되는 6x6~8x8 네온 퍼즐에서 얼음·전기·리셋 타일을 활용해 모든 발판을 한 번에 방문하는 해밀턴 경로 챌린지.
- [Mirror Run — 거울 대칭 동시 조작 러너](./mirror-run/) — 중앙 거울선 기준으로 분신이 반대로 움직이며, 수직·수평·대각·회전 거울 8레벨에서 양쪽 충돌을 동시에 피해야 하는 네온 러너.
- [Gravity Garden — 중력 정원](./gravity-garden/) — 중력 방향을 90도씩 전환해 씨앗과 흙 블록을 함께 낙하시켜 물에 닿게 만드는 네온 중력 퍼즐.
- [Chain Reaction — 연쇄 폭발 전략](./chain-reaction/) — 한 지점의 첫 폭발로 일반·대형·전기·방패 구체를 연쇄 점화해 목표 제거 수를 넘기는 1탭 전략 퍼즐.
- [Rhythm Forge — 리듬 대장간](./rhythm-forge/) — 좌우에서 접근하는 비트 노트를 D/F/J/K로 판정해 무기를 단조하고, 특수 노트(강타·연타·담금질)로 품질 ★을 끌어올리는 네온 리듬 단조 게임.
- [Shrink Escape — 크기 변환 퍼즐 플랫포머](./shrink-escape/) — 소/중/대 3단계 크기 전환으로 틈새·바람·약한 바닥·무게 스위치를 돌파하는 네온 퍼즐 플랫포머.
- [Signal Chain — 신호 전파 타이밍 퍼즐](./signal-chain/) — 노드를 드래그로 연결하고 게이트·스플리터·딜레이어 타이밍을 맞춰 모든 타겟에 신호를 전파하는 네온 로직 퍼즐.
- [Orbital Sling — 중력 슬링샷 퍼즐](./orbital-sling/) — 여러 중력 우물을 연속 스윙바이해 목표 별에 도달하는 네온 우주 궤도 퍼즐.
- [Dual Mind — 양손 분할 동시 조작 액션](./dual-mind/) — 화면 좌/우를 동시에 조작해 두 캐릭터를 동기화 지점에 맞춰 보내 문을 여는 협동 퍼즐 오토러너.
- [Ink War — 실시간 영역 칠하기 전략](./ink-war/) — 터치/드래그로 인접 칸을 칠하며 AI와 60초 실시간 영역 전쟁을 벌이는 네온 전략 게임.
- [Polarity Dash — 자석 극성 플랫포머](./polarity-dash/) — 극성을 +/−로 전환하며 같은 극은 반발, 다른 극은 흡착을 이용해 8개 레벨을 탈출하는 네온 액션 플랫포머.
- [Echo Runner — 시간 분신 협동 러너](./echo-runner/) — 런을 완료해 남긴 에코(과거의 나)와 협동해 문과 스위치 퍼즐을 푸는 시간 루프 플랫폼 게임.
