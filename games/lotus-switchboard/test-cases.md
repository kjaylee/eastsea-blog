# Test Cases — lotus-switchboard

## Functional
- TC-F001: 타이틀 화면과 시작 버튼이 390x844에서 한 화면 내 렌더링된다.
- TC-F002: 시작 버튼 탭 후 플레이 상태로 진입한다.
- TC-F003: 보드 타일 탭 시 90도 회전하고 시각 반영이 즉시 보인다.
- TC-F004: `점등` 버튼 탭 시 현재 신호가 경로를 따라 이동하고 성공/실패가 판정된다.
- TC-F005: 성공 시 점수와 콤보가 증가한다.
- TC-F006: 실패 시 스파크가 감소하고 0이 되면 게임오버 패널이 열린다.
- TC-F007: 재시작 버튼 탭 시 새 게임으로 초기화된다.

## Wow factor
- TC-W001: 예보 큐 3개와 추천 오멘 문구가 표시된다.
- TC-W002: 3연속 성공 시 Lotus Bloom 배너와 배수 보너스가 발동한다.
- TC-W003: 4턴마다 Festival Decree 3지선다 패널이 열리고 선택이 반영된다.
- TC-W004: 서울 시간 기준 일일 과업 진행도/완료 상태가 표시된다.
- TC-W005: 종료 후 Chronicle Share Card 문구가 생성되고 복사 버튼이 동작한다.

## Persistence / quality
- TC-P001: 최고 점수, 최고 콤보, 일일 기록이 `localStorage`에 저장되고 새로고침 후 복원된다.
- TC-P002: 모바일 viewport 390x844에서 가로 스크롤이 발생하지 않는다.
- TC-P003: `?autotest=1` 실행 시 JS `pageerror`가 0건이다.
- TC-P004: 외부 네트워크 자산 없이도 게임이 로드된다.
- TC-P005: 게임 파일과 카탈로그 등록 후 라우트 접근이 가능하다.
