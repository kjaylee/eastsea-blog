Sigil Forge Rogue — 시길 포지 로그

Sigil Forge Rogue는 룬 체인을 그려 주문을 주조하는 모바일 로그라이크 액션입니다. 한 번의 런마다 봉인 초안을 골라 빌드를 바꾸고, 오늘의 오멘과 데일리 계약으로 다시 도전할 이유를 남깁니다.

🎮 조작 방법

  • 룬 노드를 드래그로 연결해 주문 체인을 만듭니다.
  • 서로 다른 룬 3개 이상 = 노바, 시작/끝 룬 동일 = 배리어, 공허 룬 포함 = 연쇄 번개

✨ 게임 특징

  • 첫 10초부터 손맛이 느껴지는 드래그 룬 체인 입력
  • 웨이브 돌파 시 봉인 초안을 고르는 런 중 선택 훅
  • 포지 서지와 데일리 오멘으로 30~90초 구간이 급격히 살아나는 에스컬레이션
  • 엔드카드 칭호와 계약 성공 보너스로 스코어 자랑이 쉬운 쇼오프 순간
  • 데일리 계약·영구 유물·최고 기록 저장으로 재도전 동기 유지
