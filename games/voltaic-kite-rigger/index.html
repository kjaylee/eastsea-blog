Voltaic Kite Rigger — 볼테익 카이트 리거

    
Voltaic Kite Rigger는 세 개의 전력 연을 동시에 조율해 발전량을 끌어올리는 모바일 스톰 리깅 아케이드입니다. 직접 드래그로 각도를 맞추고, Storm Front와 Rig Draft를 넘기며 Trident Sync 고점을 노리게 설계했습니다.


    
🎮 조작 방법

    
    
      
  • 카이트 몸체를 직접 드래그해 풍향 화살표 쪽으로 각도를 맞춥니다.
    • 
      
  • Kite 1~3 버튼으로 선택 후 Angle 버튼으로 미세조정할 수 있습니다.
    • 
      
  • Trim Pulse는 전체 카이트를 현재 풍향 쪽으로 순간 보정합니다.
    • 
    


    
✨ 게임 특징

    
    
      
  • Drag-to-Rig 입력으로 첫 10초 안에 차별화된 조작 손맛 제공
    • 
      
  • Storm Front로 30~90초 구간 난류와 진폭이 살아나는 에스컬레이션
    • 
      
  • Rig Draft로 허용 오차, 보너스, 안정도 회복을 직접 선택
    • 
      
  • Trident Sync 순간 연출로 스크린샷/GIF 자랑 포인트 강화
    • 
      
  • Daily Contract와 최고 기록 저장으로 재도전 동기 유지
    •
