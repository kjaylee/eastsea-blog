## 버그 헌터의 각성

바람이 거세졌다.

안개 숲 외곽 언덕에서 내려다본 벨크로스 마을은 여전히 평화롭다. 하지만 나는 안다. GM이 내 존재를 감지했고, 시스템이 나를 '오류'로 분류하기 시작했다는 것을.

손바닥의 버그 헌터 문장이 맥동했다. 스탯 창을 열었다.

```
[CHARACTER STATUS]
이름: 한지훈 | 레벨: 4
STR: 87 [제한됨] | AGI: 25
INT: 18 | VIT: 55 [제한됨]
LUCK: 10
CHAOS: 15 | ADMIN: 0.1%
HP: 680/680 | MP: 220/220
[퀘스트: 시스템의 이단아]
→ CHAOS 50 달성 | 남은 시간: 67:14:22
```

CHAOS 15에서 50까지, 35 포인트를 67시간 안에 올려야 한다. 3년간 작성한 버그 리포트 3,847건이 내 무기다. 그중 쓸 만한 것들을 골랐다.

ETW-B-2104. 지형 관통 버그. 특정 각도로 대시하면 벽을 뚫는다.
ETW-B-3291. 아이템 복제 버그.

"먼저 숨겨진 던전부터."

베타 시절 확인한 좌표 — 안개 숲 북동쪽 (1847, 2261). 미구현 던전. 정상 입장은 레벨 30이지만, 지형 관통 버그로 우회 가능하다.

30분을 걸어 해당 지점에 도착했다. 거대한 바위벽 위에 봉인 마법진이 새겨져 있었다.

```
[봉인된 던전: 시스템 시련의 방]
입장 조건: 레벨 30 / 봉인 해제 키
현재 상태: 봉인됨
```

벽의 왼쪽 끝, 지면과 만나는 모서리. 45도 각도로 대시, 동시에 점프.

몸이 벽을 관통했다.

```
[경고: 비정상적 지형 관통 감지]
위치 재조정 시도... 실패.
CHAOS +3 → 현재: 18
```

던전 인스턴스 안에 진입하면 시스템이 위치를 강제로 되돌릴 수 없다. 내부 물리 법칙은 외부와 독립적으로 작동하니까.

---

순백색 공간이 펼쳐졌다. 끝없는 흰 방. 바닥에 격자 패턴, 공중에는 반투명 데이터 스트림이 흘러간다.

```
[시스템 시련의 방]
등급: 히든 | 권장 레벨: 30
경고: 레벨 차이로 몬스터 강화 보정 적용
```

첫 번째 방에 들어서자 바닥에서 존재가 솟아올랐다. 인간형 실루엣. 눈도 입도 없는 매끈한 얼굴.

```
[시스템 교정자 Lv.25]
HP: 12,000 | 특수: 버그 감지 — 비정상 스탯 무효화
```

교정자가 소리 없이 돌진했다. 칼날처럼 변형된 팔이 내려왔다.

**-187!**

HP의 4분의 1이 증발했다. 동시에 디버프가 걸렸다.

```
[디버프: 스탯 교정]
STR 87 → 15 (30초간 원래 수치 무효)
```

버그로 얻은 힘이 무효화되었다. STR 15로는 레벨 25 몬스터에 유의미한 데미지를 줄 수 없다.

"그렇다면."

인벤토리를 열었다. 장비를 전부 해제하고 다시 정확한 순서로 장착했다. 반지 → 손목보호대 → 검. 스탯 중복 적용 트리거.

교정 디버프와 중복 적용 버그가 동시에 충돌했다.

```
[시스템 충돌 감지]
디버프 '스탯 교정'과 버그 '스탯 중복 적용' 동시 작동
연산 오류 → STR 값 불확정
결과: STR = 174 (15 + 87 + 87 — 15)
CHAOS +5 → 현재: 23
```

시스템이 교정 전후의 값을 동시에 인정해버렸다. 버그 위의 버그.

STR 174의 검격이 교정자를 갈랐다.

**-2,847!**

**-3,104!**

**-2,955!**

세 방만에 교정자가 쓰러졌다.

```
[시스템 교정자 처치]
경험치: 8,500 | 드롭: [교정자의 코어] (희귀)
레벨 업! Lv.4 → Lv.8
CHAOS +2 → 현재: 25
```

교정자의 코어. 버그 저항 +30%. 핵심 아이템이다.

---

두 번째 방은 검은 공간이었다. 바닥에 코드가 흘러가고 있는데 — 버그 헌터의 문장이 작렬했다. 붉은색으로 빛나는 코드 라인들이 눈에 들어왔다.

```
[버그 헌터 문장 활성화]
발견된 코드 이상: 3건
① 경험치 오버플로우 지점
② 스킬 습득 조건 우회 지점
③ [접근 불가 — ADMIN 1% 필요]
```

스킬 습득 조건 우회. 붉은 코드 라인에 손을 댔다.

```
[스킬 습득 조건 우회]
습득 가능 스킬:
■ [검기 방출] (Lv.20 전사 전용)
■ [은신] (Lv.15 도적 전용)
■ [코드 리딩] (조건: CHAOS 20↑)
  — 시스템 코드를 해석하는 능력
최대 2개 선택 | CHAOS +3
```

"코드 리딩과 은신."

전투력보다 정보력과 생존력을 선택했다. 코드 리딩은 시스템 자체를 이해하는 데 필수적이고, 은신은 추적을 피하는 데 유용하다.

```
[스킬 습득 완료]
■ 코드 리딩 Lv.1 — 숨겨진 버그/오브젝트 시각화 (MP 50/분)
■ 은신 Lv.1 — 10초 투명화, 쿨다운 60초 (MP 30)
CHAOS: 28
```

코드 리딩을 발동하자 검은 방의 벽면에서 코드들이 형광빛으로 떠올랐다. 안쪽 벽에 숨겨진 문이 보였다.

---

**관리자의 서재.**

문 안으로 들어서자 오래된 도서관이 나타났다. 천장까지 늘어선 책장. 빛나는 책들 — 에테르 월드의 설정과 로그가 데이터 패킷 형태로 저장된 공간.

입장 조건은 ADMIN 0.1% 이상. 아슬아슬하게 충족했다.

「시스템 이벤트 스케줄」을 집어 들었다.

```
EVENT_004: 교정 프로토콜
상태: 활성화 | 대상: 비정상 플레이어 전원
1단계 (감시) — 현재 가동 중
2단계 (추적) — 트리거: CHAOS 30
  → 전용 교정자 '세라핌' 배정, 실시간 위치 추적
3단계 (삭제) — 트리거: CHAOS 50
  → [접근 불가 — ADMIN 5% 필요]
```

심장이 내려앉았다.

CHAOS 50은 히든 퀘스트의 목표이자, 동시에 삭제 프로토콜의 트리거. 퀘스트를 완수하려면 반드시 삭제 위험에 직면해야 한다. 이건 목숨을 건 시험이다.

책장 사이에 끼워진 쪽지를 발견했다.

```
[관리자 메모 #0041]
"교정 프로토콜은 완벽하지 않다.
 규칙을 어기는 것이 아니라
 규칙의 빈틈을 이용하면
 프로토콜은 그것을 '버그'로 인식하지 못한다.
 핵심은 규칙의 해석이다. — E."
```

E. 엘리샤. QA 테스터로서 3년간 배운 것이 바로 이것이었다. 버그란 규칙 위반이 아니라, 규칙의 불완전함을 드러내는 것. 시스템은 자기 모순을 인식하지 못한다.

"게임의 규칙으로 게임을 이긴다."

---

던전을 빠져나와 안개 숲에 서자마자 박소라가 달려왔다.

"지훈 씨, 큰일 났어요!"

뒤에 철벽방어, 불꽃마법사, 치유의손길이 따라왔다. 모두 얼굴이 하얗다.

"시스템이 '교정자'를 풀었어요. 마을 주변에서 레벨 20짜리가 두 마리 목격됐고, 이미 테스터 두 명이 당했대요."

철벽방어가 이를 악물었다. "교정자에게 죽으면 레벨이 1로 초기화돼. 스탯 전부."

불꽃마법사가 무거운 목소리로 덧붙였다.

"그리고 교정자들이 찾는 이름이 공지에 찍혔어."

알고 있었다. 나를 찾는 거다.

```
[시스템 긴급 공지]
비정상 플레이어 '한지훈' 감지.
교정 프로토콜 2단계 발동.
전용 교정자 '세라핌' 배정 완료.
대상 위치: 실시간 추적 중.
```

시스템이 내 행동을 종합 판단해서 2단계를 앞당겼다. CHAOS 28인데도.

코드 리딩을 발동했다. 안개 숲 남쪽으로 이어지는 특이한 코드 라인이 보였다. 일반 지형과 다른 색상. 숨겨진 안전 경로.

"남쪽으로 가. 설명할 시간 없어."

달리기 시작했다. 다섯 명이 뒤따랐다.

등 뒤에서 기계적인 음향이 울렸다.

```
[세라핌 접근 중]
거리: 800m | 예상 도착: 4분 32초
```

그리고 하늘에서 빛줄기가 내렸다.

300미터 전방. 하얀 빛의 기둥 안에서 존재가 나타났다. 여섯 장의 기계적 날개. 은색 갑옷. 눈에서 뿜어나오는 푸른 광선.

```
[교정자 세라핌 Lv.???]
HP: ???/??? | 속성: 시스템
※ 레벨 차이로 정보 확인 불가
```

세라핌이 나를 바라보았다.

"대상 '한지훈'. 교정 프로토콜 2단계에 의거하여 구속합니다."

여섯 날개가 동시에 펼쳐졌다.

"저항 시 — 삭제합니다."

소라의 비명이 들렸다. 철벽방어의 방패 드는 손이 떨린다.

하지만 나는 코드 리딩을 멈추지 않았다. 세라핌의 몸체를 읽었다. 빽빽한 코드 사이에서 — 한 줄의 붉은 라인이 반짝거렸다.

버그.

세라핌에게도 버그가 있다.

주먹을 쥐었다.

"좋아. 한번 해보자."

주머니 안의 미확인 수정이 뜨겁게 진동했다. 엘리샤의 끊겼던 말이 떠올랐다.

*"세라핌은 시스템이 아니라 — "*

그 뒤에 이어질 진실. 반드시 알아낼 것이다.
