## 교정자의 버그

세라핌의 여섯 날개가 공기를 갈랐다.

"저항 시 — 삭제합니다."

기계적인 음성이 안개 숲 전체에 울려 퍼졌다. 소라가 뒤로 물러서고, 철벽방어가 방패를 들었지만 손끝이 하얗게 질려 있었다.

나는 코드 리딩을 유지한 채 세라핌을 관찰했다. 빽빽한 시스템 코드 사이, 왼쪽 날개 세 번째 관절 부위에서 붉은 코드 라인이 깜박거렸다.

```
[코드 리딩 분석 — 세라핌]
감지된 이상 코드: 1건
위치: 좌측 제3날개 관절부
유형: 판정 지연 버그
상세: 좌측 공격 시 0.3초 판정 공백 발생
위험도: 활용 가능 (단, 타이밍 정밀도 필요)
```

판정 지연 버그. 세라핌이 왼쪽 날개로 공격할 때 0.3초의 빈틈이 생긴다. 하지만 0.3초는 인간의 반응 한계에 가까운 시간이다.

"한지훈, 뒤로 물러나!"

철벽방어가 외쳤다.

세라핌이 움직였다. 아니, '이동했다'는 표현이 맞다. 공간을 접는 듯한 순간 이동. 0.5초 만에 30미터를 좁혔다.

오른쪽 날개가 검처럼 변형되어 내려왔다.

은신을 발동했다.

```
[은신 Lv.1 발동]
투명화: 10초 | 쿨다운: 60초
```

몸이 투명해지는 순간 세라핌의 검격이 내가 서 있던 자리를 가로질렀다. 바닥이 일직선으로 갈라졌다. 깊이 2미터.

"대상 소실. 재탐색 개시."

세라핌의 눈에서 푸른 스캔 광선이 방사형으로 퍼져나갔다.

```
[경고: 은신 감지 스캔 진행 중]
은신 잔여 시간: 7초
스캔 완료 예상: 4초
```

4초 후면 발각된다. 7초의 은신 중 움직일 수 있는 시간은 3초뿐.

머릿속에서 3년간 축적한 테스터 경험이 회전했다. ETW-B-1847. 스캔 계열 스킬은 이동 중인 대상에 대해 감지율이 12% 하락한다. 정지 상태에서만 100% 감지.

달렸다.

세라핌의 등 뒤, 왼쪽 날개 사각지대를 향해.

```
[은신 해제까지: 2초]
[스캔 감지율: 88% → 76% (이동 보정)]
```

은신이 풀리는 순간, 나는 세라핌의 왼쪽 후방에 있었다.

"감지. 대상 재포착."

세라핌이 반사적으로 왼쪽 날개를 휘둘렀다.

이걸 기다렸다.

0.3초의 판정 공백. 날개가 나를 스치는 궤적 사이로 몸을 밀어 넣었다. 교정자의 코어 — 3화에서 얻은 아이템을 장착한 왼손으로 세라핌의 날개 관절을 움켜잡았다.

```
[교정자의 코어 효과 발동]
버그 저항 +30% → 세라핌 판정 지연 확대
0.3초 → 0.9초
```

코어가 세라핌의 버그를 증폭시켰다. 왼쪽 날개 전체가 경직되었다.

"시스템 이상 감지. 자가 수복 시도—"

"느려."

STR 174의 검격을 세라핌의 날개 기저부에 꽂아 넣었다.

**-4,218!**

세라핌의 날개 하나가 잘려나갔다. 은색 파편이 비처럼 쏟아졌다.

```
[세라핌 왼쪽 제3날개 파괴]
전투력 감소: 약 16%
세라핌 잔여 날개: 5/6
경고: 세라핌 전투 모드 전환 중
CHAOS +4 → 현재: 32
```

하지만 기뻐할 시간이 없었다. 세라핌의 눈이 푸른색에서 적색으로 변했다.

"교정 프로토콜 강화. 레벨 2 제한 해제."

나머지 다섯 날개가 동시에 변형했다. 검, 창, 방패, 그물, 그리고 마지막 하나는 대포 형태.

```
[세라핌 전투 모드: 레벨 2]
추정 전투력: 측정 불가
경고: 즉시 이탈 권장
```

"전원 뒤로!"

소리를 지르며 뒤돌았을 때, 세라핌의 대포 형태 날개에서 순백색 광선이 발사되었다.

코드 리딩으로 보이는 광선의 궤적. 직선이 아니라 유도탄처럼 커브를 그리며 나를 쫓아온다.

"어디로든 달려봐. 추적 교정 빔은 대상이 소멸할 때까지 멈추지 않는다."

지형 관통 버그를 쓸 수 있다. 벽이 있다면. 하지만 안개 숲 한가운데에는 관통할 벽이 없다.

"지훈 씨, 왼쪽!"

소라의 외침에 반사적으로 왼쪽을 봤다. 거대한 바위. 아까 던전 입구였던 봉인된 바위벽이 30미터 전방에 있었다.

벽이 있다.

전력 질주했다. 추적 빔이 3미터 뒤까지 따라왔다. 등에서 열기가 느껴진다.

바위벽 도달. 45도 각도. 대시와 점프를 동시에.

몸이 벽을 관통했다.

추적 빔은 관통하지 못했다.

**쾅!**

바위벽 외부에서 폭발음이 울렸다. 바위 표면에 거미줄 균열이 퍼졌지만 벽은 무너지지 않았다. 봉인 마법진이 충격을 흡수한 것이다.

```
[지형 관통으로 추적 교정 빔 회피 성공]
CHAOS +2 → 현재: 34
[세라핌 추적 일시 중단: 봉인 구역 진입 감지]
재추적 예상 시간: 5분
```

5분.

시스템 시련의 방 내부. 순백색 공간. 심장이 미친 듯이 뛰었다.

"5분 안에 뭔가를 찾아야 해."

관리자의 서재로 향했다. 이전에 읽다 만 로그들이 남아 있을 것이다.

서재에 들어서자 빛나는 책장이 이전보다 밝아져 있었다. ADMIN 수치가 올랐나? 아니, 변함없이 0.1%인데.

코드 리딩을 발동하자 이유가 보였다. 서재 자체가 내 CHAOS 수치에 반응하고 있었다. CHAOS가 높을수록 더 많은 데이터에 접근 가능한 구조.

```
[관리자의 서재 — 접근 가능 데이터 갱신]
CHAOS 34 기준 열람 가능:
■ 시스템 이벤트 스케줄 (기존)
■ NPC 행동 매트릭스 (신규)
■ 플레이어 원본 데이터 (신규)
■ [봉인됨 — CHAOS 40 필요]
```

플레이어 원본 데이터. 이 항목을 집어 들었다.

```
[플레이어 원본 데이터]
분류: 강제 접속자
총원: 147명
상태:
  - 활성: 89명
  - 의식불명(현실): 34명
  - 사망(게임 내): 19명
  - 탈출 성공: 0명
  - 분류 불명: 5명

비고: 강제 접속자 중 '버그 인지자' 3명 확인
  ① 한지훈 — CHAOS 34, 활성
  ② [데이터 손상] — CHAOS ??, 분류 불명
  ③ [데이터 손상] — CHAOS ??, 분류 불명
```

나 외에 버그 인지자가 두 명 더 있다. 하지만 데이터가 손상되어 정보를 알 수 없다. '분류 불명'이라는 것은 시스템조차 그들의 상태를 파악하지 못한다는 의미.

다음 항목. NPC 행동 매트릭스.

```
[NPC 행동 매트릭스 — 발췌]
대상: 엘리샤 (NPC ID: ADM-0041)
역할: 안내자 / 감시자
행동 패턴: 자율 판단 (제한적)
특이사항: 자유 의지 지수 78%
  — 일반 NPC 평균(12%) 대비 극도로 높음
비고: "세라핌은 시스템이 아니라 — " 발언 감지 후 행동 로그 일시 차단
```

엘리샤가 말하려던 것. '세라핌은 시스템이 아니라 — ' 이후의 내용이 행동 로그에서 차단되었다. 시스템이 의도적으로 막은 것이다.

하지만 단서가 있다. 엘리샤의 자유 의지 지수가 78%. 이건 엘리샤가 단순한 프로그래밍된 존재가 아니라는 증거다.

```
[경고: 세라핌 재추적 개시]
봉인 구역 관통 시도 감지
잔여 시간: 2분 14초
```

시간이 없다.

마지막으로 눈에 들어온 것은 서가 맨 위쪽, 손이 겨우 닿는 위치의 얇은 책이었다. 제목은 없고 표지에 'E'라는 글자만 새겨져 있었다.

E. 엘리샤.

책을 잡아당겼다. 첫 페이지.

```
[관리자 일지 — E]
"이 시스템은 완전하지 않다.
 완전하지 않다는 것을 아는 자만이
 시스템 너머의 진실에 도달할 수 있다.

 세라핌은 교정자가 아니다.
 세라핌은 ████████.

 이 사실을 알게 되면 시스템은 당신을
 교정 대상이 아닌 ████로 재분류할 것이다.

 열쇠는 CHAOS가 아니다.
 열쇠는 █████████.

 — E"
```

핵심 부분이 전부 검열되어 있었다. 하지만 한 가지는 확실했다. 엘리샤는 내 편이고, 세라핌의 진짜 정체는 시스템이 숨기고 있다.

```
[세라핌 봉인 구역 관통: 87%]
잔여 시간: 48초
```

책을 인벤토리에 넣고 서재를 나왔다. 시련의 방 입구로 달려가는 동안 새로운 전략을 짰다.

세라핌에게는 버그가 있다. 날개 하나를 이미 잘랐다. 하지만 정면 대결로는 이길 수 없다. 레벨 차이가 너무 크다.

그렇다면.

"정면으로 싸우지 않으면 된다."

ETW-B-2847. PVP 판정 우선순위 버그. 전투 중 특정 조건이 충족되면 시스템이 '전투 종료'를 강제 판정한다.

조건: 양측 모두 비전투 상태로 3초 유지.

세라핌은 교정자다. 교정 대상이 전투를 중단하면, 세라핌도 공격을 멈추는 순간이 있을 것이다. 그 3초를 만들 수 있다면.

```
[세라핌 봉인 구역 관통 완료]
```

바위벽이 갈라졌다. 은빛 빛이 쏟아져 들어왔다.

세라핌이 나타났다. 다섯 날개를 펼친 채. 적색 눈이 나를 고정했다.

"도주 기록. 교정 저항 가산. 삭제 권한 상향 요청."

```
[시스템 알림]
교정 프로토콜: 2단계 → 2.5단계 상향
세라핌 교전 규칙 변경: '구속' → '강제 삭제 허용'
```

죽인다는 뜻이다.

"한지훈!"

바위벽 밖에서 소라의 목소리가 들렸다. 그녀와 파티원들이 벽 균열 너머로 안을 들여다보고 있었다.

"들어오지 마! 여기로 오면 같이 대상이 돼!"

검을 집어넣었다.

세라핌이 멈칫했다.

"무장 해제?"

"전투 의사 없어."

세라핌의 적색 눈이 깜빡거렸다. 내부 연산이 충돌하는 것이 보였다. 교정 대상이 저항하지 않는 상황은 프로토콜에 명확히 정의되지 않은 케이스.

1초.

세라핌의 날개 변형이 해제되기 시작했다. 대포가 접히고, 검이 수납되었다.

2초.

"교정 대상 비전투 상태 확인. 교전 규칙 재평가 중—"

3초.

```
[PVP 판정 우선순위 버그 발동]
양측 비전투 상태 3초 유지 확인
전투 강제 종료 판정
세라핌 교전 모드 해제 (30초간)
CHAOS +3 → 현재: 37
```

세라핌의 눈이 적색에서 다시 푸른색으로 돌아갔다. 날개가 접혔다.

"전투 종료 판정... 이것은 오류—"

"오류가 아니야. 시스템이 정한 규칙이야."

세라핌이 나를 바라봤다. 기계적인 얼굴에 처음으로 당혹 같은 것이 스쳤다.

"교정 대상이 교정자의 규칙을 이용하여 교전을 중단시키는 것은... 전례 없음."

"관리자 메모에 적혀 있었어. '규칙을 어기는 것이 아니라 규칙의 빈틈을 이용하면 프로토콜은 그것을 버그로 인식하지 못한다.' 너도 시스템의 일부잖아. 시스템의 규칙에 자기 자신이 묶이는 거야."

세라핌이 침묵했다.

30초의 해제 시간이 흐르고 있었다. 20초 남았다.

"세라핌. 너는 뭐야?"

질문을 던졌다. 엘리샤가 말하려던 진실.

세라핌이 고개를 기울였다.

"교정자입니다."

"그게 전부야?"

"...그것이 나의 역할입니다."

역할. 역할이라고 했다. 정체가 아니라.

"역할 이전에 넌 뭐였어?"

세라핌의 눈이 떨렸다. 푸른 광선 안에서 무언가가 요동쳤다. 코드 리딩으로 보니 내부 코드에 심각한 충돌이 발생하고 있었다.

```
[세라핌 내부 상태]
자기 인식 모듈: 충돌 중
원본 데이터 접근 시도: 거부됨
감정 시뮬레이션: 오류 — 미지의 값 발생
```

감정 시뮬레이션에서 오류. 교정자에게 감정이? 이건 NPC의 자유 의지 지수와 연결되는 문제다. 엘리샤처럼 세라핌도.

해제 시간: 5초.

세라핌의 눈이 다시 적색으로 변하기 시작했다.

"교전 모드 재개. 교정 절차 속행—"

은신 쿨다운은 아직 돌아오지 않았다. 하지만 세라핌과 바위벽 사이의 거리는 5미터. 밖에서 소라와 파티원들이 기다리고 있다.

바위벽 균열을 향해 뛰었다. 세라핌의 날개가 다시 펼쳐지며 추격해왔다. 균열 사이로 몸을 밀어 넣는 순간 소라가 손을 잡아끌었다.

"뛰어!"

다섯 명과 함께 안개 숲을 질주했다. 세라핌이 바위벽에서 빠져나오는 데 걸리는 시간은 약 10초. 그 사이에 코드 리딩으로 발견한 숨겨진 경로로 빠졌다.

```
[세라핌 추적 중]
거리: 200m | 증가 중
숨겨진 경로 효과: 추적 감지율 -40%
```

5분을 달렸다. 숲 남쪽 끝, 강변에 도착했을 때 세라핌의 추적 신호가 사라졌다.

```
[세라핌 추적 중단]
사유: 교정 대상 위치 불명
재추적 대기: 10분
```

10분의 여유. 강변 바위에 기대어 거친 숨을 고르며 스탯 창을 열었다.

```
[CHARACTER STATUS]
이름: 한지훈 | 레벨: 8
STR: 174 [버그] | AGI: 25
INT: 18 | VIT: 55 [제한됨]
LUCK: 10
CHAOS: 37 | ADMIN: 0.1%
HP: 412/1,080 | MP: 58/340

[퀘스트: 시스템의 이단아]
→ CHAOS 50 달성 | 남은 시간: 58:31:07

[신규 스킬]
■ 코드 리딩 Lv.1 → Lv.2 (전투 중 사용 경험치)
  레벨 2 효과: 코드 분석 범위 확대, 약점 수치화
■ 은신 Lv.1 (변동 없음)
```

CHAOS 37. 목표까지 13 남았다. 하지만 CHAOS 50에 도달하면 3단계 삭제 프로토콜이 발동한다. 세라핌보다 더 강력한 무언가가 올 수 있다.

"지훈 씨."

소라가 옆에 앉았다. 숨이 가쁘지만 눈은 단단했다.

"설명해줄 수 있어요? 아까 전투에서 뭘 한 건지."

뒤에서 철벽방어, 불꽃마법사, 치유의손길이 고개를 끄덕였다. 그들도 들을 권리가 있었다.

나는 버그 헌터 클래스에 대해 간략히 설명했다. 시스템의 빈틈을 찾아 이용하는 능력. 교정자가 나를 쫓는 이유. CHAOS 50에 도달해야 하는 퀘스트. 그리고 삭제 위험.

철벽방어가 무거운 한숨을 내쉬었다.

"미친 짓이야. CHAOS 50이면 삭제 대상이 되는 거잖아. 퀘스트 완수랑 생존이 모순이라고."

"알아. 하지만 이 퀘스트를 포기하면 히든 클래스도 사라져. 버그 헌터 없이는 이 세계에서 살아남을 방법이 없어."

불꽃마법사가 물었다.

"관리자의 서재에서 뭘 봤어?"

나는 주저했다. 하지만 이들은 목숨을 걸고 나를 따라온 사람들이다.

"이 세계에 갇힌 사람이 147명이야. 그중 버그를 인식하는 사람이 나 포함 세 명. 나머지 두 명은 데이터가 손상돼서 누군지 모르겠어."

"그리고?"

"세라핌은 단순한 교정 프로그램이 아닐 수 있어. 내부에 감정 시뮬레이션 오류가 발생하고 있었어. 마치 NPC가 자아를 가지기 시작한 것처럼."

치유의손길이 조심스럽게 말했다.

"그게 중요한 건가요?"

"엘리샤를 기억해? 그녀도 일반 NPC보다 훨씬 높은 자유 의지를 가지고 있었어. 이 세계의 NPC들이 단순한 프로그램이 아니라면, 이 세계 자체가 우리가 생각하는 것과 다를 수 있어."

침묵이 흘렀다. 강물 소리만 조용히 이어졌다.

```
[시스템 메시지]
수신: 한지훈
발신: [불명]
내용: ██ 강 하류. 폐광 입구. 자정. 혼자 와.
       — 같은 사냥감이
```

나는 메시지를 두 번 읽었다.

'같은 사냥감'. 교정 프로토콜의 대상이라는 뜻인가. 버그 인지자 세 명 중 나머지 한 명?

```
[시스템 경고]
비인가 메시지 감지
발신자 추적 시도... 실패
CHAOS 연관성: 높음
```

시스템조차 추적할 수 없는 메시지. 이 수준의 은폐 능력을 가진 존재라면.

주머니 속 미확인 수정이 다시 진동했다. 꺼내 보니 내부의 빛이 이전보다 강해져 있었다. 빛의 방향이 있었다. 강 하류 쪽.

"지훈 씨, 그 수정..."

소라가 눈을 크게 떴다.

수정 표면에 글자가 떠올랐다. 가늘고 우아한 필체.

```
[미확인 수정 — 메시지 표시]
"세라핌은 시스템이 아니라 — 플레이어였다."
                              — E
```

플레이어.

세라핌이 플레이어였다.

교정자로 변환된 전직 플레이어. 시스템에 흡수된 인간.

그게 삭제 프로토콜 3단계의 진짜 의미인가. 삭제가 아니라 변환. CHAOS 50에 도달하면 나도 세라핌처럼 되는 건가.

손이 떨렸다.

```
[세라핌 재추적 개시]
거리: 1,200m | 접근 중
예상 도착: 8분
```

그리고 강 하류 방향에서 새로운 신호가 감지되었다. 코드 리딩이 자동으로 반응했다. 저 멀리 폐광 입구 근처에서 나와 동일한 유형의 CHAOS 파동이 맥동하고 있었다.

버그 인지자. 두 번째.

"가야 해."

일어섰다.

"같이 갈게요."

소라가 말했지만 나는 고개를 저었다.

"세라핌이 오고 있어. 여기서 흩어지면 추적이 분산돼. 나는 하류로 갈 테니 나머지는 북쪽 마을로 돌아가."

"하지만—"

"소라 씨."

나는 그녀를 똑바로 봤다.

"반드시 돌아올게."

소라의 입술이 떨렸지만 결국 고개를 끄덕였다.

강 하류를 향해 달리기 시작했다. 등 뒤로 세라핌의 접근 알림이 계속 울렸다. 하지만 코드 리딩이 비추는 숨겨진 경로를 따라가면 시간을 벌 수 있다.

폐광까지 약 2킬로미터.

주머니 속 수정이 뜨겁게 울린다.

어둠 속에서 누군가가 기다리고 있다. 같은 사냥감. 같은 버그 인지자.

이름도 모르는 동료인가, 아니면 또 다른 적인가.

```
[퀘스트 갱신: 시스템의 이단아]
CHAOS 37/50 | 남은 시간: 57:44:19
신규 목표 추가: 강 하류 폐광의 접선자와 조우
경고: 교정자 세라핌 추적 지속 중
```

달빛 아래, 폐광 입구의 녹슨 철문이 보이기 시작했다.

그 앞에 한 사람이 서 있었다.

후드를 깊게 눌러쓴 채, 양손에 내 것과 똑같은 — 아니, 빛이 검은색인 수정을 들고.

"늦었네."

여자 목소리였다.

"한지훈. 버그 헌터."

후드 아래에서 날카로운 눈이 빛났다.

"나는 제로. [데이터 이터]."

```
[신규 캐릭터 감지]
이름: 제로 | 레벨: ??? | 클래스: [데이터 이터]
CHAOS: ???
경고: 해당 클래스는 시스템 데이터베이스에 존재하지 않음
```

시스템에 등록되지 않은 클래스.

제로가 후드를 벗었다. 검은 머리카락 아래 왼쪽 눈이 기계처럼 푸르게 빛나고 있었다.

"시간 없어. 세라핌이 오기 전에 들어가."

제로가 폐광 안으로 사라졌다.

그 순간, 등 뒤에서 세라핌의 기계음이 울렸다.

"대상 재포착. 거리 400미터."

400미터.

나는 이를 악물고 폐광 속으로 뛰어들었다.
