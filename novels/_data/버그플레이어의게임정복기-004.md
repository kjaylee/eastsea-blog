## 교정자의 버그

세라핌의 여섯 날개가 펼쳐진 순간, 시간이 느려졌다.

아니, 느려진 게 아니다. 코드 리딩이 극한까지 가동되면서 내 인지 속도가 올라간 것이다. 세라핌의 몸체를 구성하는 수천 줄의 코드가 형광색으로 눈앞에 떠올랐다.

그리고 그 사이, 한 줄의 붉은 라인.

```
[코드 리딩 Lv.1 — 분석 결과]
대상: 교정자 세라핌
코드 총량: 8,247줄
이상 코드: 1줄 (라인 4,091)
내용: [접근 중... MP 소모 가속]
경고: 현재 MP로는 완전 해석 불가
MP: 220/220 → 170/220 (소모 중)
```

완전 해석은 무리다. 하지만 위치는 확인했다. 라인 4,091. 세라핌의 좌측 세 번째 날개 접합부. 거기에 붉은 코드가 깜빡이고 있었다.

"대상 한지훈, 최후 경고입니다."

세라핌의 음성이 금속을 긁는 듯한 울림을 동반했다. 푸른 광선이 양 눈에서 수렴하며 조준 자세를 취했다.

"저항 시 — "

"잠깐."

내가 손을 들었다.

"교정 프로토콜 2단계, 구속 명령이지? 삭제는 3단계부터잖아."

세라핌의 동작이 0.3초 멈췄다. 시스템 명령어를 정확히 인용하면 교정자는 반드시 검증 루틴을 거치도록 설계되어 있다. QA 테스터 3년의 감이 맞았다.

```
[세라핌 — 명령 검증 중]
프로토콜 2단계 확인... 구속 우선, 삭제는 저항 시에만 허용
판정: 대상의 발언은 정확함
행동 수정: 구속 모드 유지
```

"정확합니다. 그러나 저항 판정은 제 재량입니다."

0.3초. 그 사이에 나는 뒤를 돌아봤다. 박소라, 철벽방어, 불꽃마법사, 치유의손길. 네 명 모두 경직된 채로 서 있었다.

"다들 들어. 내가 신호를 보내면 남쪽으로 전력 질주해. 코드 리딩으로 확인한 안전 경로야."

"지훈 씨, 혼자 남을 셈이에요?" 소라의 목소리가 떨렸다.

"2분이면 돼."

"그 2분 안에 죽을 수도 있잖아요!"

"죽진 않아. 삭제 권한이 없으니까."

거짓말이다. 세라핌이 '저항'으로 판정하면 삭제도 가능하다. 하지만 그건 말하지 않았다.

주머니 속 미확인 수정이 손바닥을 데울 듯 뜨거웠다. 엘리샤의 끊긴 말.

*"세라핌은 시스템이 아니라 — "*

수정을 꺼냈다. 반투명한 표면 안에서 보라색 빛이 맥동했다.

```
[미확인 수정 — 활성화 조건 충족]
발동 조건: 교정자와의 근접 조우
효과: ???
사용하시겠습니까? [Y/N]
```

"Y."

수정이 산산조각 나며 보라색 파동이 퍼졌다. 세라핌이 순간적으로 후퇴했다. 그리고 내 눈앞에 새로운 시스템 창이 떴다.

```
[엘리샤의 기록 — 단편 #041]
"세라핌은 시스템이 만든 프로그램이 아니라,
 한때 이 세계에 살았던 플레이어의 잔존 데이터다.
 코드명 S-7. 본명 미상. 첫 번째 버그 헌터.
 시스템에 포획되어 교정자로 변환됨.
 약점: 원래 자아의 기억 조각.
 라인 4,091에 봉인된 플레이어 ID가 잠들어 있다."
```

심장이 뛰었다.

세라핌은 프로그램이 아니다. 사람이었다. 나보다 먼저 이 세계에 갇힌 버그 헌터. 시스템이 포획해서 무기로 만든 것이다.

"지금이야! 가!"

소라에게 외쳤다. 네 명이 남쪽으로 달리기 시작했다. 세라핌의 시선이 잠시 그쪽으로 향했지만, 즉시 나에게 돌아왔다.

"우선 대상은 한지훈. 부차 대상 추적은 후순위."

"그래, 나만 보면 돼."

은신 스킬을 발동했다.

```
[은신 Lv.1 활성화]
10초간 투명화 | 쿨다운 60초
MP: 170 → 140
```

몸이 투명해지는 순간, 왼쪽으로 구르며 위치를 옮겼다. 세라핌의 추적 시선이 0.5초 흔들렸다.

"감지 범위 재조정 중..."

10초. 그 안에 해야 할 일이 있었다.

인벤토리에서 교정자의 코어를 꺼냈다. 시련의 방에서 레벨 25 교정자를 잡고 얻은 아이템. 버그 저항 +30%. 하지만 지금 필요한 건 다른 용도다.

교정자의 코어는 교정자 계열 존재의 공명 매개체로 사용할 수 있다. 같은 계열 코드로 이루어진 존재끼리는 코어를 통해 데이터 간섭이 가능하다. 베타 테스트 당시 ETW-B-4102 리포트에 기록한 내용이다.

코어를 손에 쥐고 코드 리딩을 집중했다. 코어에서 발산되는 교정자 파장이 세라핌의 라인 4,091과 공명하기 시작했다.

```
[교정자의 코어 — 공명 발동]
대상: 세라핌 | 간섭 지점: 라인 4,091
접근 중... 38%... 61%... 
경고: 은신 해제까지 3초
```

은신이 풀리기 직전, 라인 4,091의 데이터가 흘러들어왔다.

```
[봉인된 플레이어 데이터]
ID: S-7_UNKNOWN
원래 클래스: [버그 헌터] (1세대)
포획 일시: 에테르 월드 알파 테스트 기간
봉인 상태: 99.7% — 자아 잔존율 0.3%
접근 키워드: [해제 불가 — CHAOS 40 필요]
CHAOS +4 → 현재: 32
```

CHAOS 40이 필요하다. 현재 32. 아직 8 포인트가 부족하다.

은신이 해제됐다. 세라핌이 즉시 위치를 잡았다.

"감지 완료. 구속 실행."

여섯 날개에서 은빛 사슬이 발사됐다. 여섯 가닥의 사슬이 내 팔다리와 몸통을 향해 쇄도했다.

"스탯 중복 적용 — 트리거!"

장비를 순서대로 해제하고 재장착했다. 반지 → 손목보호대 → 검.

```
[시스템 충돌 감지]
스탯 중복 적용 발동
STR: 87 → 174 | AGI: 25 → 50
CHAOS +3 → 현재: 35
```

AGI 50으로 올라간 순간, 사슬 네 개를 피했다. 하지만 두 개가 왼팔과 오른쪽 허벅지를 감았다.

**-312!**

```
[교정 사슬 — 디버프 적용]
이동 속도 -60% | 스킬 사용 불가 (10초)
HP: 680 → 368
```

스킬 사용 불가. 코드 리딩도, 은신도 못 쓴다.

세라핌이 다가왔다. 기계적인 걸음. 푸른 눈에서 스캔 광선이 내 몸을 훑었다.

"비정상 스탯 확인. CHAOS 수치 35. 교정 대상 확정."

사슬이 조여들었다. 왼팔의 감각이 사라져가고 있었다.

"잠깐, 질문 하나만."

"비정상 플레이어의 발언권은 제한됩니다."

"교정 프로토콜 조항 7-B. 대상이 자발적 교정에 동의할 의사를 표명할 경우, 최소 30초의 의사 표현 시간을 부여해야 한다."

세라핌이 다시 멈췄다. 0.5초간 조항 검증.

```
[프로토콜 조항 7-B 확인]
... 유효.
의사 표현 시간 부여: 30초
```

"30초 시작합니다."

"너 이름이 뭐야?"

세라핌의 코드가 한순간 출렁였다. 미세하지만 분명한 파동. 라인 4,091 부근에서 봉인된 데이터가 맥동한 것이다.

"명칭은 교정자 세라핌입니다."

"그건 코드네임이잖아. 진짜 이름. 시스템에 잡히기 전에 불렸던 이름."

세라핌의 여섯 날개가 떨렸다. 기계적 안정성이 0.1초 흔들렸다.

"해당 질문은 교정 업무와 무관합니다."

"무관하지 않아. 너는 교정자가 아니라 플레이어야. S-7. 알파 테스트 때 이 세계에 들어온 첫 번째 버그 헌터."

라인 4,091이 격렬하게 반응했다. 코드 리딩이 비활성화된 상태에서도 느껴질 정도의 진동.

```
[세라핌 — 내부 충돌 감지]
봉인 데이터 활성화 시도... 차단됨
자아 잔존율: 0.3% → 0.7%
시스템 경고: 교정자 안정성 저하
```

"당신은... 무슨 말을..."

세라핌의 음성이 한 순간 기계음에서 인간의 것으로 바뀌었다. 아주 짧은 찰나.

"12초 남았습니다."

금속음이 다시 돌아왔다. 하지만 나는 들었다. 사슬의 조임이 미세하게 약해졌다.

"기억나지? 네가 처음 이 세계에 들어왔을 때. 버그를 발견하고, 시스템과 싸웠던 거. 나랑 같은 일을 했던 거야."

"5초."

"너를 꺼내줄게. 봉인을 풀어줄게. 그러니까 —"

"시간 종료. 구속을 재개합니다."

사슬이 다시 조여들었다. 하지만 세라핌의 동작에는 0.2초의 지연이 있었다. 내부 충돌이 처리 속도를 늦추고 있는 것이다.

그 0.2초를 이용했다.

치유의손길이 남기고 간 긴급 회복 포션. 인벤토리에서 꺼내 마셨다.

```
[긴급 회복 포션 사용]
HP: 368 → 528 | 디버프 해제 시간 -5초
```

디버프 해제까지 5초. 버틸 수 있다.

세라핌의 광선이 나를 향해 수렴했다.

"저항 판정 — 확정."

```
[경고: 교정자 세라핌 — 삭제 모드 전환]
저항 확인 → 프로토콜 2단계 예외 조항 발동
제한적 삭제 권한 승인
```

"삭제 시퀀스 개시."

여섯 날개에서 순백색 에너지가 모이기 시작했다. 공기가 진동했다. 주변의 나무가 흔들리고, 안개가 밀려났다.

```
[삭제 광선 — 차지 중]
예상 데미지: 즉사급
발사까지: 8초
```

8초.

디버프 해제까지 5초.

스킬 사용 가능까지 5초.

해제되는 순간, 은신을 쓰면 회피가 가능하다. 하지만 그것만으론 부족하다. 세라핌은 실시간 추적 중이다. 도망쳐봤자 다시 쫓아온다.

근본적인 해결이 필요했다.

교정자의 코어를 다시 꺼냈다. 코어를 통한 간섭으로 봉인된 자아를 자극하면 — 세라핌의 시스템이 내부 충돌을 일으킨다. 하지만 CHAOS 40이 필요하고, 지금은 35다.

5 포인트.

3초 남았다.

내 머릿속에서 버그 리포트 ETW-B-1847이 떠올랐다. 사망 직전 캐릭터의 스탯이 역산되는 버그. 정확히는 — HP가 10% 이하로 떨어지는 순간, 시스템이 사망 판정을 먼저 실행하고 실제 데미지를 나중에 적용하는 타이밍 오류.

이 버그를 역이용하면 — '가사(假死) 상태'를 만들 수 있다.

HP를 의도적으로 10% 이하로 떨어뜨리면, 시스템이 사망을 판정하는 0.1초 동안 CHAOS가 급등한다. 시스템 오류니까.

미친 짓이다.

디버프가 해제됐다.

```
[디버프 해제 — 스킬 사용 가능]
삭제 광선 발사까지: 3초
```

나는 자신의 검을 뽑아 — 왼쪽 팔을 그었다.

**-460!**

```
HP: 528 → 68 (10%)
[시스템 판정 오류 발생]
사망 전 판정 실행 — 0.1초 딜레이
CHAOS 급상승: 35 → 41
```

세계가 빨갛게 물들었다. HP 10%. 시야 가장자리에 검은 테두리가 번졌다. 동시에 CHAOS가 41로 치솟았다.

40 이상. 봉인 해제 조건 충족.

코드 리딩 발동. 교정자의 코어를 세라핌을 향해 던졌다.

"봉인 해제 — 라인 4,091!"

코어가 세라핌의 가슴에 닿는 순간, 보라색과 은색 빛이 폭발했다.

```
[교정자 세라핌 — 내부 충돌 심화]
봉인 데이터 활성화: 라인 4,091 개방
자아 잔존율: 0.7% → 12.4%
시스템 경고: 치명적 오류 — 교정자 기능 일시 정지
삭제 광선 — 취소
```

세라핌이 공중에서 경련했다. 여섯 날개가 불규칙하게 접혔다 펴졌다를 반복했다. 기계적인 푸른 눈빛이 점멸했다.

"으... 아..."

인간의 신음이었다.

세라핌이 바닥에 무릎을 꿇었다. 은색 갑옷의 표면에 금이 가기 시작했다. 그 틈으로 약한 살빛이 보였다.

"이름... 이름이..."

"기억해. 네 이름이 뭔지."

세라핌이 — 아니, 그 안에 갇힌 사람이 고개를 들었다. 푸른 기계 눈과 갈색 인간 눈이 반반씩 겹쳐 보였다.

"나는... 나는..."

```
[시스템 긴급 개입]
교정자 안정화 프로토콜 발동
봉인 재강화 중... 12.4% → 8.1%
교정자 의식 강제 차단 시도
```

시스템이 다시 봉인을 씌우고 있다. 12.4%에서 8.1%로 되돌리고 있었다.

"버텨! 밀려나지 마!"

세라핌의 몸이 부들부들 떨렸다. 은색 갑옷이 재결합하며 금이 메워지고 있었다.

그때, 안개 속에서 은빛 광선이 날아왔다.

세라핌의 등에 꽂혔다.

아니, 꽂힌 게 아니었다. 은빛 빛이 세라핌의 봉인 코드를 — 잠시 동안 멈춰 세웠다.

"거기까진 제가 해줄 수 있는 전부예요."

안개 너머, 반투명한 홀로그램. 엘리샤였다. 그녀의 형상은 흔들리며 노이즈가 끼어 있었다. 원격 접속. 제한된 데이터 전송.

"엘리샤!"

"시간이 없어요, 지훈 씨. 봉인 정지는 8초가 한계예요. 그 안에 이 좌표로 이동하세요."

눈앞에 좌표가 떴다.

```
[좌표 수신: (2104, 3847)]
위치: 안개 숲 최심부 — 시스템 사각지대
설명: GM 감시가 도달하지 못하는 영역
도착 예상 시간: 은신 + 전력 질주 시 약 4분
```

"세라핌은?"

"봉인이 재강화되면 일시 기절 상태에 들어가요. 약 30분. 그 안에 멀리 가세요."

"하지만 이 사람을 이대로 — "

"지금은 구할 수 없어요."

엘리샤의 목소리가 단호했다. 하지만 눈빛에는 고통이 서려 있었다.

"CHAOS 50 이상, ADMIN 1% 이상이면 봉인을 완전히 해제할 수 있어요. 지금의 지훈 씨론 부족해요."

```
[현재 상태]
CHAOS: 41 | ADMIN: 0.1%
목표: CHAOS 50+ / ADMIN 1%+
```

세라핌의 떨림이 멈춰가고 있었다. 눈빛이 다시 푸른 기계색으로 돌아가고 있었다.

나는 이를 악물었다.

"반드시 돌아올게."

은신을 발동하고 달렸다.

```
[은신 Lv.1 활성화]
10초간 투명화 | 잔여 MP: 90
```

등 뒤에서 세라핌이 바닥에 쓰러지는 소리가 들렸다. 기절 상태 진입. 30분.

전력으로 달렸다. HP 68. MP 90. 빈사 상태나 다름없었다.

안개가 짙어졌다. 평범한 안개가 아니었다. 시스템 데이터가 물리적으로 응축된 영역. 코드 리딩을 쓰지 않아도 공기 자체가 코드의 냄새를 풍겼다.

4분 후, 좌표에 도착했다.

```
[시스템 사각지대 진입]
위치: (2104, 3847)
상태: GM 감시 불가 영역
교정자 추적 — 신호 소실
안전 판정: 일시적
```

추적이 끊겼다.

무릎이 꺾였다. 풀밭 위에 쓰러졌다. 호흡이 거칠었다. 왼팔에서 흐르는 피를 오른손으로 눌렀다. 자해로 깎은 HP. 미친 짓이었지만, 결과적으로 살아남았다.

```
[자연 회복 활성화]
HP: 68 → 분당 +5 회복
완전 회복 예상: 약 2시간
```

하늘을 올려다봤다. 안개 너머로 별이 보였다. 이 세계의 별은 데이터 노드라고 한다. 반짝이는 하나하나가 시스템의 연산 장치.

"세라핌... 아니, S-7."

그 사람도 나와 같았다. 테스터였을지, 일반 유저였을지 모르지만 — 이 세계에 갇히고, 시스템과 싸우다가, 결국 잡혀서 무기가 되었다.

엘리샤가 알고 있었다. 처음부터.

"세라핌은 시스템이 아니라 —" 포획된 플레이어.

주먹을 쥐었다. CHAOS 41. 목표까지 9. 그리고 ADMIN을 0.1%에서 1%로 올려야 한다.

```
[퀘스트 갱신: 시스템의 이단아]
→ CHAOS 50 달성 | 남은 시간: 51:22:07
→ 추가 목표: 세라핌 해방 (CHAOS 50+ / ADMIN 1%+)
```

51시간. 2일 조금 넘게 남았다.

소라에게서 메시지가 왔다.

```
[파티 채팅]
박소라: 지훈 씨, 살아 있어요? 남쪽 안전 경로로 벨크로스 외곽까지 왔어요
철벽방어: 응답 바람. 안 되면 좌표 알려줘
불꽃마법사: 교정자 추적은 여기까진 안 온 것 같아
치유의손길: 회복 포션 두 개 남아있습니다
```

대답을 입력했다.

```
[한지훈 → 파티 채팅]
살아있다. 세라핌은 일시 기절.
현재 시스템 사각지대. 위치 공유는 위험하니 생략.
24시간 뒤 벨크로스 남문에서 합류. 그때까지 레벨링.
그리고 — 세라핌에 대해 알아낸 게 있다. 만나서 말해줄게.
```

보내고 나서 인벤토리를 확인했다. 교정자의 코어는 세라핌에게 던져서 소실됐다. 남은 아이템은 초급 검, 가죽 갑옷, 손목보호대, 스탯 반지. 그리고 시련의 방에서 가져온 관리자 서재의 단서들.

코드 리딩을 발동했다. 사각지대 내부를 탐색했다.

```
[코드 리딩 — 사각지대 탐색]
감지된 이상 요소: 2건
① 숨겨진 NPC 반응 (약함, 지하 3m)
② 시스템 로그 잔해 (부분 손상, 해독 가능)
MP: 90 → 60
```

숨겨진 NPC. 시스템조차 감시하지 못하는 영역에 NPC가 있다? 일반 NPC가 아닐 것이다.

"먼저 로그부터."

시스템 로그 잔해에 접근했다. 땅 위에 흩어진 반투명한 파편들. 깨진 유리처럼 보이지만, 손을 대면 텍스트가 떠올랐다.

```
[손상된 시스템 로그 — 부분 복원]
...알파 테스트 참가자 12명 중 9명 로그아웃 성공...
...3명 잔류: S-7, S-11, S-12...
...S-7: 교정자로 변환 완료...
...S-11: [데이터 손상 — 해독 불가]...
...S-12: 시스템과 합의. ADMIN 5% 부여. 조건부 자유...
...S-12 현재 위치: [데이터 손상]...
...관리자 서재 관리 담당으로 배정...
```

S-12. 시스템과 합의해서 조건부 자유를 얻은 플레이어. 관리자 서재 관리 담당.

그리고 ADMIN 5%.

내가 서재에서 봤던 메모 — *"핵심은 규칙의 해석이다. — E."*

E. 엘리샤.

엘리샤가 S-12?

NPC가 아니라 — 플레이어?

```
[정보 갱신]
엘리샤 = S-12 (추정)
알파 테스트 잔류자
시스템과의 합의로 ADMIN 5% 보유
NPC로 위장하여 활동 중
```

퍼즐이 맞춰지고 있었다. 엘리샤는 처음부터 나를 관찰한 게 아니라, 같은 처지의 사람을 기다리고 있었던 것이다. 새로운 버그 헌터를.

잠깐. S-11은? 세 번째 잔류자.

로그의 해당 부분은 손상되어 읽을 수 없었다. 하지만 불안한 예감이 들었다. S-7은 교정자가 되었고, S-12는 시스템과 합의했다. S-11은?

"지하의 NPC가 혹시..."

땅을 파기 시작했다. 검으로 흙을 걷어내자, 30센티미터 아래에 돌계단이 나타났다. 이끼 낀 고대 석조. 계단은 어둠 속으로 내려갔다.

코드 리딩으로 확인하니 아래에서 미약하지만 확실한 생체 반응이 감지됐다. NPC — 아니, 어쩌면 또 다른 플레이어의 잔존.

```
[경고: HP 73 / MP 60]
전투 가능 상태 아님
권장 행동: 대기 및 회복
```

시스템은 대기를 권장했다. 하지만 남은 시간은 51시간이고, 세라핌의 기절은 30분뿐이다. 기절에서 깨어난 세라핌이 사각지대를 찾아올 가능성도 있다.

쉴 시간이 없다.

계단을 내려갔다.

어둠이 깊어졌다. 10단, 20단, 30단. 공기가 차가워지고, 벽면에 빛나는 이끼가 약한 녹색 빛을 냈다.

지하 공간이 펼쳐졌다.

원형 방. 직경 약 10미터. 중앙에 석조 의자 하나. 그 위에 누군가 앉아 있었다.

후드를 쓴 인물. 몸에 반투명한 사슬이 감겨 있었다. 시스템 사슬이 아닌, 자발적으로 건 봉인처럼 보였다.

내가 들어서자, 후드 아래에서 목소리가 흘러나왔다.

"오래 걸렸군."

낡은 목소리. 오래 쓰지 않은 성대의 진동.

"기다리고 있었다, 새로운 버그 헌터."

```
[미확인 NPC / 플레이어 감지]
이름: ???
레벨: ???
특이사항: ADMIN 수치 감지됨 — 수치 불명
경고: 코드 리딩으로 해석 불가
```

코드 리딩이 통하지 않는다. 이 존재의 코드 레벨이 내 스킬 레벨을 압도적으로 넘어선다는 의미.

"누구세요?"

후드가 벗겨졌다.

회색 머리카락. 깊게 패인 눈. 그러나 눈동자는 날카롭게 빛났다.

"S-11."

심장이 뛰었다.

"알파 테스트 잔류자 3인 중 마지막. 네가 엘리샤한테 들었는지 모르겠지만 — 나는 시스템에 잡히지도, 합의하지도 않았어. 도망쳤지."

S-11이 사슬 감긴 손을 들어 벽을 가리켰다. 벽면 전체에 코드가 새겨져 있었다. 수천 줄의 분석 데이터.

"20년 동안 여기서 시스템을 분석했다. 탈출 방법을 찾기 위해."

"20년...?"

"알파 테스트부터니까. 현실 시간으로는 모르겠다만, 이 세계 기준으론 20년이야."

S-11이 나를 바라봤다. 오래된 피로와 날카로운 의지가 공존하는 눈빛.

"그리고 마침내 방법을 찾았어. 시스템을 이기는 게 아니라, 시스템을 다시 쓰는 방법을."

벽의 코드 중 한 부분이 붉게 빛났다.

```
[S-11의 연구 노트 — 핵심 발견]
"시스템 코어는 이 세계의 최심부, '원점(Origin)'에 존재한다.
 원점에 접근하려면 ADMIN 10% 이상이 필요하다.
 원점에서 코어를 재프로그래밍하면
 ── 이 세계의 모든 법칙을 다시 쓸 수 있다.
 교정자를 해방시킬 수도, 탈출 경로를 만들 수도 있다.
 단, 접근에는 조건이 있다."
```

"조건이 뭔데요?"

S-11이 씁쓸하게 웃었다.

"혼자서는 불가능해. 세 명의 버그 헌터가 동시에 원점에 접속해야 해. 시스템이 만든 삼중 잠금이거든."

세 명.

S-7 — 세라핌. 교정자로 갇혀 있다.
S-12 — 엘리샤. 시스템과 합의해서 제한적 자유.
S-11 — 눈앞의 이 사람. 도망쳐서 숨어 있다.

"당신들 셋이 모이면 되는 거잖아요."

"S-7은 포획됐어. 봉인 해제가 먼저야. 그리고 엘리샤는 시스템과의 합의 때문에 직접 행동할 수 없어. 합의 조건에 '원점 접근 금지'가 있거든."

"그러면..."

S-11이 나를 가리켰다.

"새로운 버그 헌터가 필요한 이유야. 엘리샤 대신 세 번째 자리를 채울 사람."

```
[히든 퀘스트 갱신]
『시스템의 이단아』 → 『원점으로의 길』
목표: 세 명의 버그 헌터로 원점(Origin) 접근
① 세라핌(S-7) 봉인 해제 — CHAOS 50+ / ADMIN 1%+
② S-11과의 동맹 — 조건: ???
③ 한지훈의 자격 증명 — 조건: ADMIN 5%+
보상: ???
경고: 실패 시 영구 삭제
남은 시간: 50:44:19
```

영구 삭제.

침을 삼켰다. 하지만 물러설 곳은 없었다.

"해요. 세 번째 자리, 내가 채울게요."

S-11이 처음으로 제대로 웃었다.

"좋아. 그럼 먼저 ADMIN부터 올려야 해. 0.1%로는 아무것도 못 하지."

S-11이 주먹을 쥐자, 벽의 코드가 재배열되며 새로운 문이 나타났다.

"여기 아래에 시스템이 폐기한 옛 관리자 노드가 있어. 20년 동안 내가 반쯤 해킹해놨지. 거기서 ADMIN을 뽑을 수 있다."

문 너머로 검은 공간이 보였다. 데이터의 바다. 코드로 이루어진 심연.

"단, 경고 하나."

S-11의 눈이 날카로워졌다.

"이 노드에 접속하면, 시스템이 즉시 감지해. 사각지대의 위치가 노출돼. 세라핌뿐 아니라 — 더 높은 등급의 교정자가 올 수도 있어."

```
[경고: 관리자 노드 접속 시 위험도]
사각지대 위치 노출
교정 프로토콜 3단계 조기 발동 가능
예상 대응 병력: 세라핌 + 상위 교정자 1기 이상
```

3단계. 삭제 프로토콜.

HP 73. MP 60. 레벨 8.

말도 안 되는 상황이었다.

"어차피 시간이 없잖아요."

나는 검을 쥐었다.

"들어갑니다."

S-11이 끄덕였다.

문이 열렸다.

검은 심연이 우리를 삼켰다.

그리고 — 저 멀리, 심연의 끝에서 무언가가 깨어나는 소리가 들렸다.

기계음이 아니었다.

숨소리였다.

```
[시스템 공지 — 전 서버]
비정상 활동 급증 감지.
교정 프로토콜 최종 단계 검토 개시.
대상: 한지훈, 미확인 개체 2건.
모든 교정자에게 명령 — 안개 숲 집결.
```

*제5화: 「원점(Origin)」 — 근일 공개*
