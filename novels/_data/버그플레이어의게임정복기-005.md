## 데이터 이터

폐광 입구를 넘는 순간, 공기가 달라졌다.

안개 숲의 축축한 냄새 대신 쇳가루 냄새가 폐를 찔렀다. 바닥은 젖어 있었고, 벽은 오래된 곡괭이 자국으로 갈라져 있었다.

뒤에서 세라핌의 기계음이 멀게 울렸다.

"대상 재포착. 거리 400미터."

나는 이를 악물고 달렸다.

전방, 어둠 속에서 푸른 빛이 한 번 깜빡였다. 제로의 왼쪽 눈이었다.

"이쪽."

낮고 짧은 목소리.

나는 빛을 따라 첫 번째 분기점으로 뛰어들었다.

코드 리딩을 켰다.

```
[코드 리딩 Lv.2 — 환경 스캔]
위치: 폐광 B-7
감지 결과:
- 주 갱도 1개
- 분기 갱도 4개
- 시스템 코드 밀도: 극저
특이사항: 시스템 관할 공백 구역
```

관할 공백. 시스템의 눈이 닿지 않는 장소.

그래서 제로가 여기로 사라진 거다.

"멈춰."

제로가 손을 들어 나를 세웠다.

발밑 2미터 앞에서 바닥이 끊겨 있었다. 수직 낙하 구간. 아래는 칠흑 같은 어둠.

"밑으로 내려가야 해."

"깊이가 안 잡혀."

"잡힐 리 없지. 시스템이 버린 구역이니까."

제로가 왼쪽 눈을 빛냈다. 그러자 허공에 반투명 계단이 생겨났다. 깨진 코드 조각들이 서로 엮여 발판으로 굳는 광경.

```
[미등록 스킬 감지]
시전자: 제로
스킬: 데이터 구조화
효과: 잔존 데이터를 물리 구조물로 변환
```

"데이터를... 발판으로 만든다고?"

"기본 중 기본. 따라와."

우리는 40미터 아래로 내려갔다.

최하층은 거대한 공동이었다. 천장 틈에서 푸른빛이 새어 들어오고, 중앙에는 둥근 수정 기둥 같은 것이 박혀 있었다.

코드 리딩이 즉시 반응했다.

```
[핵심 노드 반응]
구역: 폐광 B-7 최하층
대상: 시스템 원형 노드(폐기)
상태: 91% 비활성
잔여 기능: 기록 보관, 로그 백업
접근 권한 요구: 관리자 1.0%
현재 권한: 0.1% (접근 불가)
```

관리자 권한이 없어 난 접근할 수 없었다.

제로는 다르게 행동했다.

그녀가 노드에 손을 올리자 표면에서 푸른 코드 실이 뽑혀 나와 손목을 타고 올라갔다. 왼쪽 눈의 기계광이 강해졌다.

```
[데이터 이터 — 흡수 진행]
1단계: 구조 해석
2단계: 유효 데이터 추출
3단계: 자체 스킬로 변환
현재 추출량: 0.27%
```

5초. 10초.

코드 실이 끊겼다. 제로가 숨을 길게 내뱉었다.

"이걸로 추격전은 버틸 수 있어."

"정체가 뭐야, 너."

질문이 튀어나왔다.

"미등록 클래스. [데이터 이터]."

"그건 나도 알아. 시스템에 등록되지 않은 클래스가 어떻게 존재하냐고."

제로가 잠시 침묵했다.

"질문 바꿀게. 넌 서재에서 뭘 봤지?"

"세라핌이 원래 플레이어였다는 기록."

"반만 맞아."

그녀는 노드에서 깨진 로그 조각을 뽑아 공중에 띄웠다.

```
[복원 로그 — 교정 프로토콜 초안]
대상 조건: 혼돈 수치 50 도달 플레이어
실행 단계:
1) 삭제 판정
2) 버그 인지 능력 분리
3) 분리 능력 + 교정 코드 결합
4) 인격 억압 후 교정자 생성
비고: 원본 인격 잔존률 0.3%
```

차가운 숫자가 머리를 때렸다.

삭제가 아니라 변환.

혼돈 50에 도달하면 플레이어는 사라지는 게 아니라 교정자로 재가공된다.

"...그럼 내 퀘스트는 함정이네."

```
[퀘스트: 시스템의 이단아]
목표: 혼돈 50 달성
현재: 41/50
남은 시간: 54:03:11
```

제로가 고개를 끄덕였다.

"나도 같은 퀘스트를 받았어. 그리고 중간에 버렸다."

"버리면 클래스가 사라진다고 했잖아."

"사라졌지."

그녀가 왼쪽 눈가를 두드렸다.

"대신 시스템이 회수하려던 인터페이스를 내가 역으로 잡아먹었어. 그걸 기반으로 만든 게 데이터 이터야."

```
[정보 갱신]
데이터 이터 = 시스템 미등록 자체 구축 클래스
구축 자원: 회수 중이던 인터페이스 + 폐기 데이터
장점: 시스템 외부에서 성장 가능
약점: 시스템이 존재 자체를 교정 대상으로 지정
```

논리는 완벽했다.

시스템이 만든 클래스가 아니니 시스템이 통제할 수 없고, 통제할 수 없으니 반드시 지운다.

"그래서 세라핌이 널 쫓는 거군."

"정확히는 서로 쫓고 있어."

"무슨 뜻이야?"

"세라핌 안에 최초 플레이어 데이터가 남아 있어. 그걸 꺼내면 교정 프로토콜을 무너뜨릴 수 있어."

"꺼낸다? 복원한다는 거야, 흡수한다는 거야?"

"둘 다 가능하게 만드는 게 내 목표였어."

'였어'라는 말이 걸렸다.

제로는 눈을 피했다.

"혼자서는 불가능했거든. 교정자 코드를 해석할 사람이 필요했어."

"코드 리딩."

"그래. 너."

거래 제안이었다.

내가 세라핌 코드를 읽어 약점을 도출하면, 제로가 그 지점을 먹어 프로토콜을 찢는다.

성공하면 대안 경로가 열린다.

```
[복원 로그 — 예외 조항]
조건: 교정 프로토콜 무력화 성공자
보상: 혼돈 50 미도달 상태에서도 히든 클래스 완성 경로 개방
상태: 검증 사례 없음
```

검증 사례 없음.

정말 되는지 모른다.

하지만 혼돈 50은 확정 사망 루트다.
선택지는 사실상 하나뿐이었다.

"좋아. 하자. 대신 조건이 있어."

"뭐."

"세라핌 안의 원본 인격, 살릴 수 있으면 살린다. 그냥 전부 먹어치우지 마."

제로의 눈빛이 흔들렸다.

"...장담은 못 해."

"장담 말고 약속. 시도는 한다."

잠깐의 정적 끝에 제로가 고개를 끄덕였다.

"시도한다."

그 순간, 폐광 전체가 요동쳤다.

천장에서 돌가루가 쏟아지고, 멀리서 금속 찢어지는 소리가 들렸다.

```
[경고]
외부 강제 진입 감지
원인: 교정자 세라핌
관할 공백 붕괴 예상: 3분 40초
```

"예상보다 빨라."

제로가 욕설을 삼켰다.

"비상 통로로—"

말이 끝나기도 전에 벽면에 붙어 있던 폐기 코드들이 떨리기 시작했다.

아까 제로가 말했던 잔존 데이터.

진동과 소리에 반응해 깨어난 것이다.

벽에서 유리 조각처럼 떨어져 나온 코드 덩어리들이 서로 엉기더니 사람 형상을 만들었다. 얼굴 없는 반투명 형체 열두 개.

```
[적 개체 등장]
명칭: 잔존 데이터 집합체
수량: 12
특성: 물리 저항 극대, 코드 공격 우선
약점 분석: 진행 중
```

첫 개체가 달려들었다.

제로가 손을 뻗어 가슴을 관통했다.

"먹어."

형체가 붕괴하며 푸른 실로 분해되어 제로에게 흡수됐다.

```
[흡수 성공]
획득 단편: 위상 이동 코드(미완성)
```

하지만 하나씩 먹어선 속도가 안 맞다.

"한지훈, 약점!"

코드 리딩을 밀어붙였다.

시야가 파란 선으로 갈라졌다.

```
[약점 도출 완료]
핵심 라인: 개체 중심부 1개
파괴 조건: 코드 간섭 공격으로 핵심 라인 절단
주의: 일반 물리 공격 무효
```

일반 검으론 못 죽인다.

나는 인벤토리에서 교정자의 코어를 꺼내 검 손잡이에 결속했다. 은빛 문양이 칼날을 타고 번졌다.

```
[무기 상태 갱신]
한손검 + 교정자 코어 결속
효과: 코드 계열 대상 타격 가능
```

두 번째 개체가 뛰어들었다.

나는 핵심 라인을 겨냥해 찔렀다.

**-9,999 (핵심 절단)**

즉사.

세 번째. 네 번째.

제로는 흡수로 셋을 처리했고, 나는 핵심 절단으로 넷을 끊었다.

```
[전투 현황]
처리: 7/12
남음: 5
혼돈: 41 → 43
```

혼돈이 또 올랐다.

43.

3단계 경계선 45까지 이제 2.

"좋아할 일이 아니네..."

세라핌 진입 알림이 연달아 울렸다.

```
[교정자 세라핌]
거리: 120m → 90m → 60m
상태: 강제 관통 완료 직전
```

남은 다섯을 정면 상대하면 시간 초과다.

나는 계산을 바꿨다.

"제로! 3체만 더 처리하면 돼."

"왜 3체?"

"혼돈 45 넘기면 즉시 3단계가 떠. 지금 43이야."

하나당 평균 +1. 운 나쁘면 +2.

다섯을 전부 잡으면 45를 넘길 확률이 높다.

"둘은 유인해서 떨군다!"

제로가 바로 이해했다.

우리는 개체 셋만 집중 타격했다.

**-9,999!**

**흡수 성공!**

**-9,999!**

```
[전투 현황]
처리: 10/12
잔존: 2
혼돈: 44
```

44.

간신히 선을 밟지 않았다.

"비상 통로로!"

우리가 달려 나가려는 찰나, 천장이 찢어지며 은빛 날개가 내려왔다.

세라핌.

붉은 눈동자가 우리를 동시에 겨냥했다.

"교정 대상 확인. 한지훈. 제로."

날개 다섯 개가 각기 다른 형태로 변환됐다. 창, 그물, 검, 방패, 그리고 대포.

```
[경고: 추적 교정 빔 충전]
발사까지: 2.4초
```

"막을 수 없어!"

"막는 게 아니야."

제로가 왼쪽 눈을 폭주시키듯 빛냈다. 앞벽의 폐기 데이터가 순식간에 비워지며 구멍이 뚫렸다.

"뚫고 나가!"

우리는 구멍으로 몸을 던졌다.

뒤에서 교정 빔이 발사됐다. 열풍이 등을 태웠고, 갱도 전체가 진동했다.

굴러 떨어진 곳은 경사진 상승 통로.

출구까지 약 80미터.

나는 스킬 창을 열었다.

```
[은신 Lv.2]
지속: 15초
재사용 대기: 사용 가능
```

"출구에서 은신 걸면 나는 빠져."

"알아. 대신 세라핌 주의를 내가 끈다."

"너 혼자 위험해."

"이미 2년째 위험했어. 새삼스럽게."

제로가 씁쓸하게 웃었다.

2년.
시스템 바깥에서 살아남기 위해 폐광에 숨어 지낸 시간.

출구 달빛이 보였다.

우리는 멈추지 않고 동시에 뛰쳐나갔다.

나는 즉시 은신을 발동했고, 제로는 반대 방향으로 푸른 잔광을 뿌리며 전력 질주했다.

세라핌이 출구를 뚫고 나와 제로를 향해 회전했다.

"우선 교정 대상: 제로. 추적 개시."

성공.

세라핌은 동쪽으로 사라졌고, 나는 감지망 밖 서쪽 숲으로 빠져나왔다.

15초가 끝났을 때 이미 300미터 이상 거리를 벌린 상태였다.

숨을 고르며 상태창을 확인했다.

```
[상태 갱신]
이름: 한지훈 | 레벨: 9
힘: 174[버그] | 민첩: 27 | 지능: 20 | 체력: 58
혼돈: 44
관리자 권한: 0.1%

[퀘스트: 시스템의 이단아]
- 기본 경로: 혼돈 50 달성
- 대안 경로: 교정 프로토콜 무력화 (미검증)
남은 시간: 53:12:44
```

혼돈 44.

단 1만 더 올라가도 3단계 경보가 울린다.

이제 전투를 함부로 할 수 없다.

나는 곧장 마을로 들어가지 않고, 강가의 폐 펌프장 쪽으로 경로를 꺾었다. 혹시 세라핌이 잔여 추적선을 남겼다면 파티를 위험에 노출시킬 수 있었기 때문이다.

코드 리딩을 다시 켰다.

```
[추적선 검사]
세라핌 잔류 표식: 2개 감지
- 표식 A: 열 신호 추적용
- 표식 B: 교정 파장 표지용
권장 조치: 수중 통과로 제거
```

나는 강으로 뛰어들어 허리까지 잠긴 채 30미터를 이동했다. 물속에서 표식이 하나씩 꺼졌다.

```
[표식 제거 완료]
잔여 추적선: 0
```

물에서 올라오자, 왼쪽 손목의 코어가 희미하게 진동했다. 방금 전 전투에서 교정 코드와 과부하 충돌이 있었던 모양이다.

```
[교정자 코어 — 임시 진단]
동기화율: 48%
과부하 흔적: 있음
잠금 해제 가능 기능: 1개 (조건부)
조건: 교정자 핵심 구조 데이터 확보
```

조건부 기능 잠금 해제.

세라핌 구조 데이터만 확보하면 코어를 한 단계 진화시킬 수 있다는 뜻이었다.

그때, 인벤토리 안에서 폐광 노드 조각이 스스로 빛났다. 제로가 마지막에 던져 준 작은 파편이다. 파편 표면에 데이터 문자열이 떠오르더니, 자동으로 내 시야에 펼쳐졌다.

```
[제로 전송 로그 — 압축본]
제목: 세라핌 해체 설계(초안)
핵심 요약:
1) 세라핌 외곽 교정 코드 5중 보호막
2) 내핵 진입 조건: 동시 간섭 2종 필요
   - 코드 해석 간섭(버그 헌터)
   - 데이터 침식 간섭(데이터 이터)
3) 실패 시 역교정 폭주 확률 72%
4) 성공 시 원본 인격 분리 가능성 31%
메모: "31%면 낮아 보여도, 지금까지 가장 높은 수치야."
```

31%.

확률로만 보면 형편없다. 하지만 0%보다 크다. 이 세계에서 내가 본 수치 중, 처음으로 '가능성'이라고 부를 만한 숫자였다.

추가 로그가 이어졌다.

```
[제로 메모 — 비공식]
"한지훈, 네가 조건을 걸었던 거 기억해.
 살릴 수 있으면 살린다.
 그래서 흡수 알고리즘을 바꿨어.

 기본 흡수 모드: 데이터 분해 후 통합
 수정 흡수 모드: 인격 패턴 분리 우선

 문제는 계산량이 세 배라 전투 중엔 불안정해.
 네 코드 리딩이 틀리면 둘 다 죽어."
```

짧은 문장이었지만, 제로의 선택이 보였다.

그녀는 정말로 '먹기' 대신 '분리'를 시도하려고 한다.

나는 코어를 주먹에 쥐었다.

"31%를 50%로 올리면 된다."

방법은 단순하다.
세라핌의 보호막 5중 중 최소 2중을 사전에 무력화할 버그를 찾아내면 된다.

즉, 다음 목표는 도망이 아니다.
분석이다.

```
[임시 목표 설정]
목표 1: 세라핌 외곽 보호막 버그 탐색
목표 2: 교정자 코어 기능 잠금 해제
목표 3: 혼돈 45 도달 회피
```

모든 경로가 숫자로 정리되자 숨이 조금 가라앉았다.

나는 젖은 옷을 털고 다시 북쪽 마을로 향했다.

북쪽 마을 불빛이 보였다.

모닥불 주변에 익숙한 얼굴들이 있었다. 소라, 철벽방어, 불꽃마법사, 치유의손길.

소라가 먼저 달려왔다.

"지훈 씨! 어디 다친 데는—"

그때 주머니 속 수정이 얼음처럼 차가워졌다.

나는 멈춰 서서 수정을 꺼냈다.

표면에 새로운 문장이 떠올랐다.

```
[미확인 수정 메시지]
"세 번째 버그 인지자는 이미 네 파티 안에 있다."
                              — E
```

심장이 한 번 멈췄다.

세 번째 버그 인지자.

147명 중 단 셋.

나, 제로, 그리고 나머지 하나.

그 하나가 내 눈앞 네 명 중 한 명이라는 뜻이었다.

소라가 내 표정을 보고 멈칫했다.

"무슨 일 있어요?"

나는 웃는 척했다.

"아니. 잠깐 시스템 알림이 떠서."

거짓말을 하면서도 시선은 파티원들을 훑었다.

누가 세 번째인가.
누가 이미 시스템의 균열을 보고 있는가.

바로 그 순간, 또 하나의 경고창이 열렸다.

```
[긴급]
교정자 세라핌 — 추적 대상 전환 준비
다음 우선 교정 대상 산정 중...
조건: 혼돈 45 이상 대상 존재 시 즉시 3단계 발동
```

그리고 내 시야 우측 하단, 파티원 한 명 위에만 보이는 미세한 표식.

보라색 점멸.

코드 리딩 자동 반응이었다.

```
[숨김 속성 감지]
대상: 파티원 ███
분류: 버그 인지자
상세 열람: 차단됨
```

누군지 이름이 가려져 있었다.

하지만 확실했다.

세 번째가 있다.

그리고 그 사람이 고개를 들며 나를 정면으로 바라봤다.

낯익은 미소.

"지훈 씨."

소라의 목소리였다.

"이제... 나한테도 설명해줄 때가 된 것 같네요, 그렇죠?"
