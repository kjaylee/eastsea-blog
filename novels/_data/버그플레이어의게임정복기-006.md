## 6화. 경계선의 동맹

"지훈 씨."

모닥불 너머에서 소라가 나를 똑바로 봤다.

"이제 저한테도 설명해줄 때가 됐죠?"

나는 대답 대신 파티원 얼굴을 순서대로 훑었다. 철벽방어, 불꽃마법사, 치유의손길. 다들 의문은 있지만 아직 상황을 모른다는 표정이었다.

문제는 내 시야 우측 하단에만 보이는 시스템 표식이었다.

보라색 점멸.

숨김 속성: 버그 인지자.

대상 이름은 차단되어 있었지만, 지금 소라가 먼저 말을 꺼낸 시점과 정확히 겹쳤다.

"일단 여기선 말 못 해."

내가 낮게 말했다.

"북쪽 폐펌프장으로 이동한다. 파티 채널 봉인 모드, 외부 송신 차단."

철벽방어가 바로 반응했다.

"긴급 프로토콜?"

"응. 지금부터는 농담 없다."

---

폐펌프장 지하실.

나는 코어를 바닥에 내려놓고 임시 방벽 코드를 펼쳤다.

```
[임시 보안 루틴 실행]
- 외부 감시 차단: 62%
- 교정 파장 난독화: 41%
- 지속 시간: 00:18:00
```

완벽하진 않다. 하지만 세라핌이 바로 좌표를 찍고 내려꽂을 정도로 허술하진 않다.

소라가 먼저 입을 열었다.

"저도 시스템 창이 보여요. 원래는 '직감' 정도였는데, 어제부터 선이 보이기 시작했어요."

나는 숨을 고르며 물었다.

"어떤 선?"

"스킬 구조선. 어디를 건드리면 무너지는지, 어디를 막으면 버티는지."

정확히 버그 인지자의 시야다.

내가 코드 리딩을 켰다.

```
[대상 스캔: 소라]
직업: 정찰지원가 Lv.12
숨김 속성: BUG AWARE (잠금 해제 37%)
파생 재능: 구조 직감
위험도: 중
```

맞다.

세 번째 인지자는 소라였다.

철벽방어가 어이없다는 듯 물었다.

"잠깐. 그럼 우리 파티에 핵유저가 둘이라는 거냐?"

"핵유저 아니고 생존자." 내가 잘라 말했다.

"그리고 한 명 더 있다. 제로."

세 사람 표정이 동시에 굳었다.

나는 지금까지 숨긴 정보를 압축해서 풀었다. 혼돈 50 함정 퀘스트, 플레이어를 교정자로 변환하는 프로토콜, 세라핌의 본질, 폐광에서 본 로그, 제로와의 임시 동맹까지.

설명이 끝나자 치유의손길이 손을 떨며 중얼거렸다.

"그럼 우리는… 시스템이 필요하면 지워버리는 부품이라는 거네."

"그래서 부품에서 변수로 올라가야 해." 

나는 바닥에 간이 도식을 그렸다.

세라핌 공략 핵심은 두 가지.

1) 외곽 교정 코드 5중 보호막 해체
2) 내핵 진입 시 동시 간섭 2종 입력

제로가 맡는 건 데이터 침식.

내가 맡는 건 코드 해석.

그리고 오늘 추가된 변수, 소라의 구조 직감은 보호막 해체 확률을 끌어올릴 수 있다.

소라가 손을 들었다.

"제가 뭘 하면 돼요?"

"실전 투입 전에 검증부터."

---

검증 장소는 폐펌프장 하층 유지보수 통로.

오래된 자동포탑 두 대가 남아 있었다. 레벨은 낮지만 교정 코드 기반 방어 루틴을 갖춘 테스트용 표적이다.

나는 명령을 내렸다.

"소라, 구조선 보여주는 대로 말해. 공격은 내가 맞춘다."

포탑이 기동했다.

```
[적 개체: 수리 포탑 Mk.1 x2]
특성: 물리 저항 70%
약점: 미확인
```

소라가 눈을 가늘게 뜨더니 빠르게 외쳤다.

"왼쪽 포탑, 코어 아래 7시 방향! 선 두 겹 겹쳐요!"

나는 반 박자 늦게 찔렀다.

**-3,420 (구조 붕괴)**

한 방에 기능 정지.

철벽방어가 휘파람을 불었다.

"와, 진짜 보이네."

두 번째 포탑이 회전하며 난사했다. 소라가 몸을 굴리며 다시 좌표를 찍었다.

"오른쪽 상단! 돌출 렌즈 바로 뒤!"

불꽃마법사가 화염구를 꽂았다.

**-2,980 (핵심 라인 단선)**

두 포탑이 동시에 멈췄다.

성능 확인 완료.

```
[파티 시너지 분석]
코드 리딩 + 구조 직감
약점 도출 속도: +47%
보호막 해체 예상 성공률: 31% → 49%
```

49%.

제로 로그 기준 31%에서 크게 올랐다.

살릴 수 있을 확률도, 살아남을 확률도 동시에 올라간다.

문제는 다른 쪽이었다.

테스트 직후, 내 혼돈 수치가 튀었다.

```
[경고]
혼돈: 44 → 45
임계치 도달
3단계 교정 감시 활성화
```

심장이 서늘해졌다.

끝내 선을 밟았다.

이제 세라핌은 '탐색'이 아니라 '즉시 교정' 모드로 전환할 수 있다.

치유의손길이 내 창을 보고 얼굴이 하얘졌다.

"45면… 바로 잡으러 오는 거 아냐?"

"올 확률 높아. 그래서 다음 단계로 바로 넘어간다."

나는 코어를 다시 장착하며 말했다.

"오늘 밤, 세라핌 외곽 보호막 하나를 강제로 뜯는다."

철벽방어가 눈을 크게 떴다.

"지금? 준비도 덜 됐는데?"

"준비가 덜 된 게 아니라 시간이 없다."

내가 상태창을 띄웠다.

```
[긴급 카운트]
교정자 세라핌 — 우선 교정 대상 재산정 중
예상 갱신: 00:06:12
```

6분.

6분 안에 우리가 먼저 좌표를 잡아야 한다.

---

목표는 북서쪽 감시탑 잔해.

세라핌이 주기적으로 스캔 비콘을 던지는 지점이다. 본체를 직접 상대하는 대신, 보호막 연산에 연결된 보조 노드를 먼저 파괴하는 계획.

이동 중 소라가 내 옆에서 물었다.

"지훈 씨, 하나만."

"말해."

"만약 진짜로 세라핌 안에 원래 플레이어 인격이 남아 있으면… 우리, 살릴 수 있어요?"

나는 잠시 침묵했다.

"확률은 낮아."

"몇 퍼센트요?"

"현재 49."

소라는 짧게 웃었다.

"그럼 할 만하네요. 수학은 못하지만, 반반은 알거든요."

이상하게 그 말이 긴장을 조금 풀어줬다.

---

감시탑 잔해에 도착하자 코드 안개가 층처럼 쌓여 있었다.

보조 노드 다섯 개가 반원형으로 떠 있고, 중앙에는 붉은 렌즈가 회전 중이었다.

```
[세라핌 외곽 연산 노드]
연결 수: 5
권장 파괴 방식: 동시 간섭 3개 이상
실패 시: 역추적 파장 발생
```

나는 파티를 배치했다.

철벽방어 전면 고정.

불꽃마법사 좌측 고열 간섭.

치유의손길 후방 안정화.

소라는 내 옆에서 구조선 호출.

"카운트 셋에 시작한다."

하나.

둘.

셋.

소라가 외쳤다.

"오른쪽 두 노드, 같은 축! 지금!"

나는 코어 검으로 첫 번째 노드의 라인을 끊었다.

불꽃마법사의 화염이 두 번째 노드에 겹쳤다.

철벽방어가 세 번째 노드를 방패로 찍어 눌렀다.

노드가 연쇄적으로 흔들렸다.

```
[연산 불안정]
보호막 동기화율: 100% → 76% → 61%
```

성공한다.

그때 상공이 찢어지는 소리가 났다.

은빛 날개.

세라핌 본체가 예상보다 훨씬 빨리 강하했다.

```
[긴급]
교정자 세라핌 출현
우선 교정 대상 업데이트 완료
1순위: 한지훈
2순위: 소라
```

소라의 얼굴이 굳었다.

"왜 제가 2순위예요?"

"버그 인지자 판정이 떴으니까."

세라핌의 눈이 붉게 점멸했다.

"교정 대상 2체 확인. 동시 회수 모드 전환."

나는 이를 악물고 외쳤다.

"소라, 지금 당장 뒤로 빠져!"

하지만 세라핌의 다섯 날개가 동시에 펼쳐지더니, 창과 그물 형태로 분리되어 우리를 덮쳤다.

그리고 마지막 경고창이 눈앞에 터졌다.

```
[치명 경고]
역교정 파장 활성화
범위 내 BUG AWARE 대상 강제 동기화 시작
남은 시간: 00:00:09
```

9초.

9초 안에 못 끊으면, 소라는 플레이어가 아니라 교정자 재료로 빨려 들어간다.

소라가 나를 바라보며 떨리는 목소리로 말했다.

"지훈 씨… 저, 머릿속이 하얘져요."

세라핌의 붉은 눈이 더 밝아졌다.

나는 코어를 쥔 손에 전력을 실었다.

선택지는 하나였다.

내 혼돈을 더 올려서라도, 지금 이 파장을 강제로 찢어야 한다.

[다음 화에 계속]
