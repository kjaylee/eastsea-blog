## 10화. 집단 회수 예고

파란 전류가 솟아오른 구금장 안에서,
나는 이를 악물고 콘솔 케이블을 붙잡았다.

바깥에서는 소라의 얼굴을 한 관리자 개체가 손을 들고 있었고,
열두 개 캡슐에서 나온 회수 유닛들이 동시에 우리를 향해 걸어왔다.

"한지훈."
소라-관리자가 차갑게 말했다.
"우선 포획 대상 1번으로 지정합니다."

"지정은 네가 하고 해제는 내가 한다."
나는 콘솔 창을 밀어 올렸다.
"유진! 오른쪽 세 번째 고리! 검은빛!"

유진이 상부 파이프를 밟고 점프했다.
"봤어!"

탕, 탕, 탕!

총탄 세 발이 코어 둘레를 스쳤다.
파랑 고리,
흰 고리,
그리고 마지막 검은 고리.

처음 두 발은 반사막에 튕겼다.
세 번째는 하부 격자 반동을 타고 각도가 꺾였다.

쨍!
검은 고리 표면에 실금이 갔다.

소라-관리자의 눈동자가 미세하게 떨렸다.

```
[서브링 무결성 손상]
제3 링 87%
```

"좋아! 한 번 더!"

유진이 방아쇠를 당기려는 순간,
회수 유닛 둘이 동시에 그녀를 향해 돌진했다.

치유의손길이 방어막을 겹쳐 세웠다.
"유진 뒤!"

금속 팔이 방어막을 긁었다.
불꽃이 튀며 실드 수치가 쭉 깎였다.

```
[치유의손길 실드]
내구도: 62% → 41%
```

"버티는 건 내가 해!"
치유의손길이 외쳤다.
"넌 쏴!"

유진이 몸을 회전시켜 공중에서 사격각을 다시 맞췄다.
그러나 이번엔 소라-관리자가 먼저 움직였다.

그녀가 허공에 창 하나를 띄웠다.

```
[보조 관리자 명령]
탄도 보정 차단
```

유진의 조준선이 갑자기 흔들렸다.
탄이 고리를 비켜가 천장 렌즈를 깨뜨렸다.

"젠장, 에임이 끌린다!"

나는 구금장 안쪽에서 바닥을 두드렸다.
"소라! 듣고 있으면 답해!"

노이즈뿐이던 서브채널이 한 번 번쩍였다.

[…훈]

"좋아, 살아 있어."
나는 즉시 명령어를 추가했다.

```
inject channel_boost --target sora.local --unsafe
```

오류창이 떠올랐다.

```
[권한 부족]
영점 관리자 승인 필요
```

"승인?"
나는 웃었다.
"그럼 강제로 서명 위조하지 뭐."

```
forge_sign --profile SORA-00 --lag 1frame
```

콘솔이 붉게 물들었다.

```
[경고]
위조 서명 감지 시 즉시 전체 격리
```

"지훈!"
치유의손길이 소리쳤다.
"너 과부하 수치!"

HUD 우측이 비명을 질렀다.

```
[CHAOS 과부하]
현재치: 96
인지 안정도: 38%
```

시야 가장자리가 검게 녹았다.
귓속에서 팬 소음 같은 잡음이 커졌다.

괜찮다.
여기서 멈추면 끝이다.

나는 손등을 깨물며 의식을 붙잡았다.

"유진, 세 번째 고리 포기."

"뭐?"

"고리 끊는 대신, 고리를 잠깐 멈춰."

"어떻게!"

"네가 잘하는 걸로."

유진은 0.3초만에 알아들었다.
그녀가 벨트에서 짧은 막대 탄두를 꺼냈다.

EMP 미니 스파이크.

"3초밖에 안 간다!"

"3초면 돼!"

유진이 스파이크를 던졌다.
스파이크는 코어 하단 격자에 박히며 퍼런 원을 퍼뜨렸다.

우웅—

검은 고리 회전이 느려졌다.
정지까지 1초.

"지금!"

나는 위조 서명을 강행했다.

```
forge_sign --force
```

```
[임시 승인]
SORA-00(지연 프레임) 확인
```

"먹혔다!"

곧바로 다음 명령.

```
detach ring3 --manual
```

금속 마찰음과 함께 코어 셋째 고리가 반쯤 이탈했다.
소라-관리자의 동공이 크게 흔들렸다.

그녀 입에서 동시에 두 목소리가 나왔다.

"포획 절차를—"
"지훈, 지금!"

로컬 소라였다.

나는 구금장 전류를 무시하고 양손으로 셋째 고리를 잡아 비틀었다.
손바닥 피부가 타는 냄새가 올라왔지만 놓지 않았다.

카직!

고리가 완전히 빠져나오며 바닥에 굴렀다.

```
[제3 링 분리 완료]
로컬 인격 우선권 상승: 3% → 29%
```

소라가 비틀거리며 무릎을 꿇었다.

"하…"
그녀가 자기 목을 붙잡았다.
"내 목소리… 들려."

"좋아."
나는 전류 속에서 웃었다.
"이제 네가 명령해."

소라의 눈빛이 잠깐 맑아졌다.
그녀는 허공 창을 떨리는 손으로 열었다.

```
SORA-00 LOCAL OVERRIDE
명령 대상: 보조 관리자 유닛
```

"멈춰."
소라가 낮게 말했다.

아홉 회수체의 걸음이 동시에 멈췄다.
하지만 완전 정지는 아니었다.
가슴 문구가 붉게 깜빡였다.

```
[명령 충돌]
GLOBAL 관리자 우선권 대기 중
```

"시간 없어!"
유진이 외쳤다.
"글로벌이 덮어쓰기 전에 튀자!"

치유의손길이 내 구금장 바깥 노드를 두드렸다.
"지훈, 전류 차단 2초만 줄게!"

그녀가 회복 스킬을 공격적으로 뒤틀어 전류 회로를 과포화시켰다.
파란 불꽃이 튀며 창살 전류가 잠깐 꺼졌다.

나는 몸을 구겨 밖으로 빠져나왔다.
무릎이 바닥에 닿자마자 소라를 부축했다.

"걸을 수 있어?"

"반반."
소라가 헛웃음 섞인 숨을 뱉었다.
"너처럼 사기로 버틴다."

"좋아. 우리 팀답네."

그 순간,
천장 전체에 붉은 경보가 점등됐다.

```
[GLOBAL ADMIN 접속]
세라핌 코어 연결
집단 교정 준비 단계 진입
```

소라-관리자의 입에서 다시 기계음이 새어 나왔다.
"수동 오버라이드 취소."

"아직 아니야."
소라가 이를 악물었다.
"내 몸에서 꺼져."

그녀가 양손을 모아 마지막 명령을 찍었다.

```
LOCAL FINAL CMD
출구 경로 7번 개방
유닛 우선순위: 지연
```

회수실 서쪽 벽이 열리며 좁은 정비 통로가 드러났다.

"전원 7번으로!"
내가 외쳤다.

우리는 거의 굴러가듯 통로로 뛰어들었다.
뒤에서는 회수 유닛들이 다시 재가동하는 금속음이 밀려왔다.

---

정비 통로는 사람 하나 겨우 지나갈 폭이었다.
천장은 낮았고,
바닥은 냉각수가 미끈거렸다.

우리는 일렬로 달렸다.
맨 앞은 유진,
중간은 소라와 나,
뒤는 치유의손길.

"후방!"
치유의손길이 외쳤다.
"유닛 세 기 붙었어!"

유진이 뒤돌아보지도 않고 수류탄 핀을 뽑았다.
"귀 막아!"

쾅!

좁은 통로에서 충격파가 두 배로 울렸다.
유닛 하나가 뒤집혔고,
다른 둘은 벽에 박혔다.

하지만 곧바로 복구 알림이 들렸다.

```
[자체 수복 모듈 작동]
```

"진짜 끈질기네!"

"수복 코어가 살아 있는 동안은 무한 리스폰이야."
나는 숨을 몰아쉬며 답했다.
"중앙 코어를 완전 꺼야 끝나."

"그럼 왜 지금 도망치냐고!"

"전술적 후퇴라고 불러줘. 멋있잖아."

소라가 중간에서 헐떡이며 웃었다.
"둘 다 시끄러워… 근데 덕분에 정신 잡혀."

통로 끝에 삼거리.
왼쪽은 막다른 환기실,
오른쪽은 보안 게이트,
정면은 폐쇄 표지.

나는 정면을 가리켰다.
"폐쇄가 정답."

유진이 물었다.
"근거."

"이 게임에서 폐쇄 표지는 늘 ‘운영자용 통로’였어. 일반 유저가 못 오게 하려고 붙여둔 거지."

"아주 개발자들의 악취가 난다."

"칭찬으로 듣겠다."

나는 폐쇄 패널을 뜯어 전선을 두 가닥 교차했다.

찌직.

문이 열렸다.

우리가 들어간 곳은 오래된 서버 챔버였다.
수백 개의 꺼진 랙,
중앙에는 작은 비상 전원 코어 하나.

그리고,
벽면 전체에 걸린 프로필 카드들.

사람 이름.
레벨.
접속 시간.
상태.

상태 칸 대부분이 같은 단어였다.

[보류]

소라가 벽을 짚고 숨을 고르며 중얼거렸다.
"이게… 강제 접속자 대기실?"

나는 카드 하나를 떼어 확인했다.

ID: P-11392
상태: 보류
예정 처리: 집단 회수

"젠장."
유진이 욕설을 내뱉었다.
"이게 다 사람이라고?"

"일부는."
나는 더 안쪽 랙을 켰다.
"그리고 일부는 이미 회수된 빈 껍데기."

치유의손길이 얼굴을 굳혔다.
"명단 뽑을 수 있어?"

"해봐야지."

나는 비상 코어에 케이블을 연결했다.
소라가 옆에서 떨리는 손으로 보조 입력을 도왔다.

```
query pending_users --phase group_recall --limit 200
```

출력창이 느리게 올라왔다.

1, 2, 3…

147에서 멈췄다.

```
총 대상: 147
```

유진이 숨을 멈췄다.
"진짜로 147명이야."

치유의손길이 물었다.
"우리가 다 구할 수 있어?"

나는 잠깐 대답하지 못했다.
147명.
지금 팀 인원은 넷.
자원은 바닥.

"한 번에 전부는 못 해."
나는 솔직히 말했다.
"그래서 동선을 다시 설계해야 해. 생존 우선 순위를 정하고, 거점부터 나눠야 해."

유진이 턱을 괴고 고개를 끄덕였다.
"좋아. 또 네 미친 지도 시간이다."

소라가 벽에 기대 선 채 말했다.
"나도 도울게. 로컬 권한 아직 29% 남았어. 전부는 못 해도, 대상 스캔 우선순위 정도는 바꿀 수 있어."

치유의손길이 웃었다.
"좋아, 관리자님."

"관리자 말고 플레이어."
소라가 정정했다.
"그거 약속했잖아."

"맞다."
나는 곧바로 고개를 끄덕였다.
"플레이어 박소라, 임무 배정한다."

---

우리는 서버 챔버 바닥에 임시 지도를 펼쳤다.
부서진 패널을 모아 삼각형으로 세우고,
각 구역에 코드명을 붙였다.

A구역: 지하 시장 폐허
B구역: 하수 관문
C구역: 버려진 튜토리얼 마을
D구역: 북측 통신탑

유진이 물었다.
"왜 C구역이 우선?"

"초보자 스폰 지역이라 저레벨 접속자가 많아."
나는 명단을 훑었다.
"집단 회수 시작되면 저기부터 빨려간다."

치유의손길이 메모를 했다.
"그럼 우리가 해야 할 건?"

"첫째, 안전지대 분산.
둘째, 가짜 관리자 비콘 설치.
셋째, 회수 경로를 유도해 빈 구역으로 보내기."

유진이 휘파람을 불었다.
"쉽게 말해, 시스템 상대로 어그로 끌기네."

"내 전문이지."

소라가 조심스럽게 말했다.
"나… 하나 더 할 수 있어."

"뭔데?"

"회수 대상 태그를 묶음으로 위장하는 거."

나는 눈을 크게 떴다.
"그게 가능해?"

"완벽하진 않아."
소라가 손가락을 떨며 허공 창을 띄웠다.
"하지만 147명을 147개 점으로 보게 하지 않고, 23개 점으로 겹쳐 보이게 만들 수 있어."

유진이 웃었다.
"좋네. 숨바꼭질 난이도 상승."

치유의손길이 심각하게 물었다.
"부작용은?"

소라는 입술을 깨물었다.
"내 인격 점유율이 다시 깎일 수 있어."

나는 즉시 말했다.
"그건 안 돼."

"지훈."
소라가 내 눈을 똑바로 봤다.
"나 혼자 안전하면 결국 다 회수돼. 그게 더 최악이야."

나는 대꾸하지 못했다.
맞는 말이었다.
너무 맞아서 싫었다.

유진이 조용히 탄창을 끼웠다.
"우리 팀 룰 기억하지?"

"뭔데."

"한 명이 영웅 놀이하면 나머지 셋이 강제로 끌어내린다."

치유의손길이 고개를 끄덕였다.
"그러니까 소라가 하더라도, 보호 루틴은 우리 셋이 다 붙여."

나는 한숨을 삼켰다.
"…좋아. 대신 새 룰 추가." 

나는 부서진 패널에 굵게 적었다.

### 생존 재편 7원칙
1) 단독 이동 금지
2) 이름 대신 코드 호출
3) 30분마다 인격 체크
4) 회수 비콘 3중 미끼
5) 저레벨 우선 구출
6) 과부하 발생 시 즉시 교대
7) 영웅 행위 금지

유진이 마지막 줄을 보고 웃었다.
"이건 네가 제일 못 지킬 듯."

"그래서 맨 위가 아니라 맨 아래에 썼다."
