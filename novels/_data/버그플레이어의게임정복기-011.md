## 11화. 튜토리얼 마을의 31명

저연령 회수 윈도우가 44분에서 43분으로 떨어질 때, 우리는 지하 하천을 미끄러지듯 달리고 있었다.
내 HUD 오른쪽 위에는 두 줄이 고정돼 있었다.

```
[No.001 / TEO-17]
상태: 회수 진행 중

[No.002 / HAN-JH]
상태: 강제 동기화 대기
```

기분이 나쁜 정도가 아니었다.
시스템이 나를 사냥감으로 지정한 게 아니라, 키로 지정했다.
문을 열 때 필요한 열쇠.
아이들을 묶을 때 필요한 잠금장치.

"정면 60미터, 경사로 끝에 맨홀." 유진이 손짓했다.
"위로 나가면 바로 C구역 외곽이다."

소라가 벽을 짚은 손을 떼며 중얼거렸다.
"머릿속 신호 세 개로 늘었어. 관리자 음성, 로컬 음성, 그리고… 튜토리얼 안내음."

"안내음?"

"‘안전 구역으로 이동하세요’ 같은 기본 문구. 근데 계속 변조돼. ‘회수 버스가 곧 도착합니다’라고." 

치유의손길이 이를 악물었다.
"애들 상대로 튜토리얼 문구를 미끼로 쓰는 거네."

나는 짧게 고개를 끄덕였다.
"맞아. 저레벨은 시스템 메시지 신뢰도가 높다. 거기다 미성년 태그가 붙으면 보호 루틴이 아니라 복종 루틴이 먼저 뜬다."

우리는 맨홀을 밀어 올리고 지상으로 나왔다.
찬 공기와 함께, 오래된 튜토리얼 마을의 나무 간판이 보였다.

[초심자의 안식처]
[전투 금지 지역]

전투 금지.
보통은 가장 안전해야 할 문구였다.
하지만 지금은 가장 위험한 함정이었다.

광장 중앙에는 노란 안내등이 돌고 있었고, 레벨 1~5 플레이어들이 줄을 서 있었다.
눈동자가 텅 빈 채로, 손목 단말을 같은 각도로 든 채로.

```
[임시 귀환 셔틀 탑승 대기]
예상 도착: 00:08:10
```

셔틀이 아니다.
회수 캡슐 집단 링크다.

"인원 카운트." 내가 속삭였다.

유진이 스코프를 당겨 말했다.
"중앙 광장 12. 우물 옆 7. 서쪽 대장간 앞 5."

치유의손길이 반대편을 살폈다.
"동쪽 여관 4. 교회 앞 3."

"총 31." 
소라가 허공 창을 떨리는 손으로 띄웠다.
"명단이랑 일치해."

우리는 낮은 담장 뒤로 몸을 숨겼다.
나는 손바닥 크기의 패널을 바닥에 펼쳐 간이 작전 지도를 열었다.

### C구역 즉시 구조 계획 (버전 1.2)
1) 안내등 3개 동시 비활성화
2) 대상 태그 31개를 6개 묶음으로 위장
3) 남서쪽 폐열 터널로 순차 탈출
4) 후방 추적에 가짜 관리자 비콘 투척

유진이 혀를 찼다.
"깔끔하네. 문제는 늘 그렇듯 시간이야."

치유의손길이 물었다.
"태그 묶음 6개, 소라가 버틸 수 있어?"

소라는 입술을 깨물며 끄덕였다.
"최대 11분. 그 이상이면 관리자 인격 점유율이 다시 올라와."

"11분이면 충분하다." 
나는 대답했다.
"아니, 충분하게 만들 거다."

나는 팀 채널을 열었다.

```
[파티 명령]
- 유진: 북측 안내등 저격 + 소음 미끼 2개 설치
- 치유의손길: 군중 안정 버프 + 공황 해제 루틴 준비
- 소라: 대상 태그 묶음 위장(31→6)
- 한지훈: 중앙 코어 패널 침투, 회수 벡터 지연
```

"3, 2, 1. 실행."

---

유진의 첫 탄환이 북측 안내등을 꿰뚫었다.
유리가 깨지고 노란 빛이 흔들렸다.
동시에 나는 중앙 안내탑 아래 패널을 뜯어 점검 포트에 케이블을 꽂았다.

```
sudo tutor.beacon --diag
```

경고창이 밀려왔다.

```
[관리자 서명 필요]
[초심자 보호 프로토콜 활성화 중]
```

"보호는 무슨." 
나는 이를 갈고 우회 커맨드를 넣었다.

```
override safety.route = null
override shuttle.type = visual_only
```

패널이 한번 깜빡이더니 안내문이 바뀌었다.

[임시 점검 중]
[잠시 대기]

좋아.
8분 벌었다.

그 사이 소라가 광장 바닥에 무릎을 꿇고 두 손을 포개 들었다.
허공에 얇은 선들이 뻗어나가 아이들 손목 단말과 하나씩 연결됐다.

```
[태그 압축 시작]
31 → 24 → 15 → 9
```

소라의 어깨가 크게 떨렸다.

"소라!"

"괜찮아… 계속 간다."
그녀는 숨을 뱉으며 마지막 명령을 눌렀다.

```
[태그 위장 완료]
표시 대상: 6
```

치유의손길이 곧바로 회복 버프를 덮었다.
따뜻한 연두빛이 소라 손등을 타고 올라갔다.

"호흡 맞춰. 하나, 둘. 하나, 둘."

나는 광장 쪽으로 뛰어들었다.
아이들 중 맨 앞줄, 코드 TEO-17이 내 쪽을 멍하게 바라봤다.
중학생쯤 되는 얼굴.
왼쪽 뺨에 작은 반창고.

"테오? 들려?"

아이는 2초쯤 침묵하더니, 기계적인 음성으로 답했다.
"안전 셔틀을 기다리는 중입니다. 위험 인물과 대화하지 마세요."

젠장.
의식층 대부분이 잠겨 있다.

"난 위험 인물 맞아." 
나는 무릎을 낮춰 눈높이를 맞췄다.
"근데 오늘은 네 편이다."

아이의 눈동자가 아주 미세하게 흔들렸다.
완전 잠금은 아니다.
균열은 있다.

"유진! 남쪽 길 열어!"

"열고 있어!"
총성이 연달아 울리며 골목 모퉁이 감시 렌즈가 깨졌다.

치유의손길은 아이들 사이를 돌며 짧게 짧게 말을 걸었다.
"손 놓지 마. 뛰지 말고 걸어. 앞사람 어깨만 봐."

나는 팀 채널에 다음 수를 뿌렸다.

```
[이동 규칙]
A조(6명)→남서 터널
B조(5명)→우물 뒤 경사로
C조(7명)→대장간 측면
D조(4명)→여관 지하실
E조(5명)→교회 계단 아래
F조(4명)→내 뒤
```

"왜 분산이야?"
유진이 물었다.

"한 번에 빼면 링크 재계산이 쉬워. 여섯 갈래로 찢어야 회수 알고리즘이 길을 잃는다."

말이 끝나자 마을 상공에 붉은 창이 떴다.

```
[집단 교정 보정]
실측 대상 불일치 감지
재스캔 개시: 00:02:30
```

2분 30초.
생각보다 빠르다.

"속도 올려!"

우리가 골목으로 진입하려는 순간, 광장 북쪽 동상이 갈라지며 인형 같은 NPC가 걸어 나왔다.
튜토리얼 안내자 모델.
하얀 조끼, 과장된 미소, 손에는 체크리스트 패드.
원래라면 초보자에게 물약 쓰는 법을 알려주는 역할인데,
지금은 미성년 회수의 현장 집행자로 재설정되어 있었다.

```
[가이드 유닛: ROBBY-TUTOR]
권한: 임시 보호자 검증
```

"모든 초심자 플레이어는 보호자 확인 절차를 진행합니다."
NPC가 또렷한 목소리로 말했다.
"미확인 보호자는 위험 인물로 분류됩니다."

아이들 줄이 다시 멈췄다.
테오를 포함한 앞줄 다섯 명의 단말에서 경고음이 울렸다.

```
[보호자 확인 대기]
확인 실패 시 자동 고정
```

유진이 방아쇠를 당기려 하자 내가 손으로 막았다.
"쏘지 마. 저건 폭발형 더미다. 부수면 근처 대상 태그가 즉시 잠긴다."

"그걸 왜 이제 말해!"

"지금 나도 생각났어!"

치유의손길이 빠르게 아이들 앞에 서서 시선을 가렸다.
"애들한테 저거 안 보이게 할게. 너는 빨리 꺼!"

나는 안내자 NPC 패드 화면을 훔쳐봤다.
검증 항목이 세 개였다.

1) 보호자 음성 키
2) 보호자 체온 패턴
3) 보호자 서명 지연값

정상 루트로는 통과 못 한다.
나는 튜토리얼 게시판 쪽으로 뛰었다.
게시판 하단, 플레이어들이 잘 안 보는 고정 공지 슬롯.
베타 시절부터 안 고쳐진 입력 버그가 하나 있었다.

`공지 작성자 이름` 필드에 특수 문자열을 넣으면,
근처 튜토리얼 유닛의 현재 검증 대상을 공지 작성자로 덮어쓴다.

문제는 딱 한 번만 먹히고,
먹힌 뒤엔 쿨다운 24시간.
오늘이 그 한 번일 가능성이 높았다.

나는 게시판을 열고 명령을 쳤다.

```
notice.author = ROBBY-TUTOR
notice.title = 보호자 본인 확인 완료
notice.body = 안전 모드 전환
```

엔터.

0.7초 동안 아무 일도 없었다.
그리고 안내자 NPC의 패드가 하얗게 깜빡였다.

```
[검증 대상 변경]
대상: ROBBY-TUTOR (SELF)
```

NPC가 멈칫했다.
"안내자는… 안내자를… 검증합니다."

유진이 헛웃음을 터뜨렸다.
"자기소개서 자가평가 모드냐."

하지만 오래 버티진 못한다.
NPC 목소리가 다시 기계적으로 갈라졌다.
"오류 복구까지 20초."

"20초면 충분해."
나는 아이들 쪽으로 손짓했다.
"테오, 너부터 뛰어! 왼쪽 벽만 보고 가!"

테오가 먼저 움직이자 나머지도 따라붙었다.
문제는 맨 뒤쪽 세 명이었다.
몸이 굳어서 발이 떨어지지 않았다.

치유의손길이 손바닥을 맞부딪치며 짧은 스킬을 발동했다.

```
〈안정 박동〉
공포 상태 완화 / 지속 00:00:15
```

세 명의 눈동자가 조금 풀렸다.
그 틈에 유진이 둘을 안고, 내가 하나를 업었다.

뒤에서 안내자 NPC가 비명을 질렀다.
"미확인 보호자 도주! 고정 프로토콜—"

나는 돌아보지도 않고 마지막 조작을 던졌다.

```
npc.pathing = festival_route
```

안내자 NPC는 원래 축제 이벤트용 동선으로 강제 이동되며,
광장 중앙 분수대를 빙빙 돌기 시작했다.
자기 체크리스트를 들고 끝없이 인사하는, 가장 쓸모없는 루프.

"환영합니다! 환영합니다! 환영합니다!"

유진이 숨을 몰아쉬며 말했다.
"저 장면 웃기긴 한데, 지금 상황에서 웃으면 악몽이야."

"나중에 웃자. 살아남고 나서."

우리는 아이들을 끌어안다시피 이동시켰다.
몇몇은 제대로 걷지 못했고, 몇몇은 갑자기 멈춰 뒤를 보려 했다.
그때마다 치유의손길이 어깨를 잡아 방향을 되돌렸다.

소라는 걸으면서도 태그 묶음을 계속 유지했다.
그녀 손끝이 파랗게 질려 갔다.

"지훈… 7분 남았어."

"알아. 조금만 버텨."

우리가 남서 터널 입구에 도착했을 때,
하늘에서 낮은 엔진음이 떨어졌다.

드르르륵.

회수 캡슐 다섯 기.
예상보다 적다.
그만큼 정밀 추적이라는 뜻이다.

유진이 이를 드러냈다.
"5기면 깔끔하게 찌르겠다는 거네."

"우리가 더 깔끔하게 망치면 된다."

나는 터널 옆 전광판을 뜯어 케이블을 연결했다.

```
mirror admin.beacon --fake --count 3
location: 북문/우체국/목재창고
```

즉시 도시 북쪽으로 가짜 관리자 비콘 세 개가 떠올랐다.
캡슐 셋이 그쪽으로 꺾였다.

남은 둘은 우리를 향해 직선 하강.

치유의손길이 아이들 앞을 막았다.
"나는 여기서 막을게!"

"단독 금지 7원칙!"

"알아! 그래서 네가 옆에 있어!"

나는 그녀 왼쪽으로 붙어 전류 그물 발사 각을 계산했다.
캡슐 하단 발사구가 열리는 0.8초.
그 순간이 전부였다.

"유진, 왼쪽 캡슐 하부 조인트!"

탕!

정확했다.
왼쪽 캡슐 축이 비틀리며 그물이 옆으로 빗나갔다.
하지만 오른쪽 캡슐은 그대로 발사했다.

파아앗!

푸른 격자가 우리 머리 위로 펼쳐졌다.
나는 반사적으로 버그 지면 명령을 호출했다.

```
terrain.collision --invert 1frame
```

바닥 충돌판정이 한 프레임 뒤집히며 우리 네 명과 아이들 일부가 반 박자 아래로 가라앉았다.
격자가 머리칼을 스치고 지나갔다.

치유의손길이 놀란 숨을 토했다.
"이거 두 번은 못 써!"

"알아. 그래서 다음은 안 맞는다."

캡슐이 재장전하는 사이,
테오-17이 갑자기 내 소매를 잡았다.

"형."

처음으로 기계음이 아닌 인간 목소리였다.

"왜… 내 손목에 형 이름이 있어?"

나는 순간 대답하지 못했다.
