## 3. 개별 서비스 상세

### 3.1 Lovart.ai
> 📄 상세: `/docs/lovart-research.md` 참조

- **핵심:** 세계 최초 AI 디자인 에이전트, 엔드투엔드 워크플로우
- **기술:** MCoT 크리에이티브 추론 엔진, 멀티에이전트 아키텍처
- **가격:** Free(500크레딧) → Starter($15) → Plus($26) → Pro($72~$90)
- **강점:** 한 프롬프트로 40개 에셋 생성, ChatCanvas, 장기 메모리
- **약점:** Trustpilot 1.9/5, 크레딧 투명성 문제, 고객 지원 부실
- **평판:** 생성 퀄리티 호평 / 크레딧 소비·고객 지원 혹평

### 3.2 Canva AI (Magic Studio)
- **핵심:** 220M+ MAU, 가장 대중적인 디자인 플랫폼
- **기능:** Magic Design (프롬프트→디자인), Magic Edit, Magic Expand, Magic Morph, Dream Lab (이미지 생성), 배경 제거, AI 카피라이팅
- **기술:** 자체 AI + 파트너 모델 통합
- **가격:** Free → Pro($12.99~$15/월) → Teams($10/유저/월, 연간) → Business → Enterprise
- **강점:** 
  - 방대한 템플릿 라이브러리 (수백만 개)
  - 직관적 UX, 학습 곡선 최소
  - 팀 협업 기능 강력
  - 프레젠테이션, SNS, 인쇄물, 웹사이트까지 올인원
  - 가격 대비 성능 최고
- **약점:**
  - AI 이미지 품질이 전문 도구 대비 떨어짐
  - 고급 편집 기능 제한
  - 2024~2025 가격 인상 (Teams 퍼시트 가격제)
- **평판:** G2 4.7/5, 업계 표준 디자인 도구
- **API:** ✅ Canva Connect API
- **한국어:** ✅ 완전 지원

### 3.3 Adobe Firefly / Express
- **핵심:** 라이선스 데이터로만 학습, 상업적 안전성 최고
- **기능:** 이미지 생성, Generative Fill (PS), 벡터 생성 (AI), 텍스트 효과, 영상 생성 (beta), Structure Reference, Style Reference
- **기술:** Firefly (Adobe Stock + 라이선스 콘텐츠로만 학습)
- **가격:**
  - Express Free: 제한적 무료
  - Firefly Standard: $9.99/월 (2,000크레딧)
  - Firefly Pro: $29.99/월 (7,000크레딧)
  - Firefly Premium: $199.99/월 (50,000크레딧)
  - Creative Cloud 포함 시 100~1,000크레딧
- **강점:**
  - 상업적 안전성 (IP 면책)
  - Adobe 생태계 통합 (PS, AI, AE, PR)
  - Generative Fill 품질 최고
  - 벡터 생성 가능
- **약점:**
  - 독립 사용 시 비쌈
  - Creative Cloud 의존적
  - 이미지 스타일 다양성이 Midjourney 대비 떨어짐
- **평판:** PCMag 추천, 전문가들 사이에서 신뢰도 높음
- **API:** ✅ Firefly API (기업용)
- **한국어:** ✅ 완전 지원

### 3.4 Figma AI
- **핵심:** UI/UX 디자인 업계 표준 + AI 기능 확장
- **기능:** Figma Make (텍스트→UI), Check Designs (AI 린터), AI 자동 레이아웃, 컴포넌트 제안, 스마트 레이어 정리
- **기술:** 자체 AI 모델
- **가격:**
  - Starter: 무료 (150 AI크레딧/일, 500/월)
  - Professional: $16/유저/월 (3,000 AI크레딧/월)
  - Organization: $55/유저/월 (3,500크레딧)
  - Enterprise: $90/유저/월 (4,250크레딧)
- **강점:**
  - 실시간 협업 최강
  - 플러그인 생태계 방대
  - Dev Mode로 개발자 핸드오프
  - 디자인 시스템 관리 최고
- **약점:**
  - AI 기능은 아직 보조 수준 (완전 자동 생성은 아님)
  - 전문 디자이너 대상 (비디자이너에게 학습 곡선)
  - AI 크레딧 제한
- **평판:** G2 4.7/5, UI/UX 업계 표준
- **API:** ✅ REST API + Plugin API
- **한국어:** 🔶 (인터페이스 일부 지원)

### 3.5 Midjourney
- **핵심:** AI 이미지 생성 최고 품질
- **기능:** 텍스트→이미지, 스타일 참조, 이미지 편집, 영상 생성 (V7~), Upscale, Vary, Zoom Out
- **기술:** 자체 모델 (V7, 2025년 출시)
- **가격:**
  - Basic: $10/월 (~3.3 Fast GPU시간)
  - Standard: $30/월 (~15시간, 무제한 Relax)
  - Pro: $60/월 (~30시간, Stealth Mode)
  - Mega: $120/월 (~60시간)
  - 무료 체험 없음 (2025~)
- **강점:**
  - 이미지 품질 업계 최고
  - 아트/사진/일러스트 스타일 탁월
  - Relax 모드 무제한 생성 (Standard+)
  - 웹 에디터 (2025 추가)
- **약점:**
  - 텍스트 렌더링 불안정
  - 정확한 레이아웃 컨트롤 어려움
  - 디자인 워크플로우 도구 아님 (이미지만)
  - API 없음 (공식)
  - 한국어 프롬프트 지원 약함
- **평판:** 이미지 품질 1위 평가 다수, $5B 밸류에이션
- **API:** ❌ 공식 API 없음
- **한국어:** ❌

### 3.6 DALL-E 3 (ChatGPT / GPT Image-1)
- **핵심:** ChatGPT 통합, 자연어 프롬프트 이해력 최고
- **기능:** 텍스트→이미지, 이미지 편집, 대화형 수정, GPT Image-1 (4o 네이티브)
- **기술:** DALL-E 3, GPT Image-1 (2025 출시)
- **가격:**
  - ChatGPT Free: 제한적 생성
  - ChatGPT Plus: $20/월
  - API: $0.04~$0.08/이미지 (사이즈별)
- **강점:**
  - 프롬프트 이해력 최고 (ChatGPT 대화 맥락)
  - GPT Image-1 텍스트 렌더링 크게 개선
  - 다국어 프롬프트 완전 지원
  - API 접근 용이
- **약점:**
  - 이미지 아트 품질은 Midjourney 미만
  - 안전 필터 과도하게 엄격
  - ChatGPT Plus $20/월 필요
  - 공인 얼굴 생성 거부
- **평판:** 접근성·편의성 1위, 품질은 2~3위
- **API:** ✅ OpenAI API
- **한국어:** ✅ 완전 지원

### 3.7 Ideogram
- **핵심:** 이미지 내 텍스트 렌더링 정확도 1위
- **기능:** 텍스트→이미지, 텍스트 렌더링 특화, Canvas 편집, Magic Fill, Style/Color 제어
- **기술:** Ideogram 3.0 (자체, 2025.03 업데이트)
- **가격:**
  - Free: 주 10크레딧
  - Basic: $7/월 (400 우선 크레딧 + 일 100 저속)
  - Plus: $16/월 (1,000 우선 + 일 200 저속)
  - Pro: $40/월 (3,000 우선 + 일 500 저속)
- **강점:**
  - 텍스트 렌더링 업계 최고 (포스터, 로고, 간판 등)
  - 스타일 컨트롤 강력
  - 무료 플랜 존재
  - API 지원
- **약점:**
  - 얼굴/인물 일관성 불안정
  - 무료 플랜 매우 제한적
  - 한국어 텍스트 렌더링은 미흡
- **평판:** 텍스트 특화 1위 인정, 전반적 호평
- **API:** ✅
- **한국어:** ❌ (영문 텍스트 특화)

### 3.8 Recraft
- **핵심:** 벡터(SVG) AI 생성의 선두주자
- **기능:** 래스터/벡터 이미지 생성, 일러스트레이션, 아이콘 세트, 브랜드 스타일 적용, 텍스트 렌더링
- **기술:** Recraft V3 (자체)
- **가격:**
  - Free: 일 50크레딧 (상업용 불가, 공개)
  - Basic: $10/월 (1,000크레딧)
  - Advanced: $27/월 (4,000크레딧)
  - Pro: $48/월 (8,400크레딧)
  - API: 래스터 $0.04/장, 벡터 $0.08/장
- **강점:**
  - 벡터(SVG) 생성 품질 최고
  - 아이콘/일러스트 스타일 일관성
  - 합리적 API 가격
  - 브랜드 색상/스타일 적용 가능
- **약점:**
  - 무료 플랜 상업 사용 불가
  - 사진 리얼리즘은 Midjourney 미만
  - UI가 다소 전문적 (비디자이너에겐 복잡)
- **평판:** 디자이너 사이에서 벡터 생성 1위 인정
- **API:** ✅ (래스터/벡터 모두)
- **한국어:** ❌

### 3.9 Kittl
- **핵심:** 그래픽 디자인 + AI, POD(주문형 인쇄) 특화

  - 텍스트 레이아웃/효과 강력 (로고, 타이포그래피)
  - POD/머천다이즈 특화 워크플로우
  - 직관적 인터페이스

- **약점:**

- **한국어:** ❌

- **기술:** 자체 AI
- **가격:**
  - 로고 탐색: 무료
  - Basic Logo: $20 (일회)

## 요약

> **1인 게임+앱 개발자에게 최적 조합은 `Canva Pro ($13) + ChatGPT Plus ($20)` = 월 $33이다.**
>

>
> Lovart.ai는 "AI 디자인 에이전시" 컨셉으로 흥미롭지만, Trustpilot 1.9점과 크레딧 투명성 이슈로 **Free 플랜으로 충분히 테스트한 후** 확대 여부를 판단하는 것이 현명하다.
