## 📸 스크린샷 촬영 가이드

### 필수 스크린샷 (게임당)
1. **게임플레이 메인** — 가장 재미있어 보이는 순간 (1280x720 또는 16:9)
2. **특수 기능** — 파워업/콤보/보스전 등 차별점
3. **메뉴/UI** — 깔끔한 인터페이스 보여주기
4. **모바일 뷰** — 반응형 모바일 화면 (선택)

### 촬영 방법
```bash
# MiniPC Playwright로 자동 촬영 가능
# 또는 Chrome DevTools 수동 촬영
# 권장 해상도: 1280x720 (16:9)
```

### GIF 제작
```bash
# MiniPC에서 Playwright로 녹화 → ffmpeg로 GIF 변환
# 권장: 6-10초, 480px 너비, 15fps
ffmpeg -i gameplay.mp4 -vf "fps=15,scale=480:-1" -loop 0 gameplay.gif
```
