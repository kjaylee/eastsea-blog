## 1) 논문 동향

- **[Agent READMEs: An Empirical Study of Context Files for Agentic Coding]** (Papers with Code Trending)
  Papers with Code 트렌딩(현 Hugging Face 연동) 상위 논문은 에이전트 컨텍스트 파일 **2,303개(1,925개 저장소)**를 대규모로 분석해, 실제 현업 지시문이 어떻게 구성되는지 계량적으로 보여줬습니다. 본문 수치에서 기능 요구(빌드/실행 62.3%, 구현/아키텍처 69%대) 대비 보안·성능 요구는 각각 **14.5%** 수준으로 낮아, “에이전트는 돌지만 안전하지 않을 수 있는” 구조적 결함이 드러났습니다. 시사점은 분명합니다: 에이전트 도입팀은 프롬프트 품질보다 먼저 프로젝트 단위 가드레일(보안/성능/권한)을 명시적으로 템플릿화해야 합니다.
  → [링크: https://huggingface.co/papers/2511.12884]

- **[Qwen3-TTS Technical Report]** (Hugging Face Trending Papers)
  Hugging Face 트렌딩 논문 목록에서 Qwen3-TTS는 **2026-01-22 공개** 문서로 재상승했고, 멀티언어 TTS와 보이스 클로닝, 스트리밍 합성을 하나의 아키텍처로 묶은 점이 강조됐습니다. 특히 dual-track 구조와 speech tokenizer가 핵심으로 소개되어, 음성 AI의 승부축이 단순 음질에서 **지연시간·제어 가능성·운영 안정성**으로 이동했음을 보여줍니다. 음성 기능을 제품화하는 팀에게는 “좋은 데모”보다 “실시간 배포 가능한 스택”이 경쟁력을 결정하는 구간입니다.
  → [링크: https://huggingface.co/papers/2601.15621]

- **[When AI Benchmarks Plateau: A Systematic Study of Benchmark Saturation]** (arXiv cs.AI)
  최신 arXiv cs.AI 상위권 논문은 LLM 벤치마크 **60개**를 대상으로 포화(saturation)를 조사했고, 설계·데이터·평가 형식에 걸친 **14개 속성**을 비교 분석했습니다. 저자들은 벤치마크의 거의 절반이 이미 구분력을 잃는 구간에 들어갔다고 보고하며, “리더보드 1위”가 실제 성능 차이를 더 이상 충분히 설명하지 못할 수 있음을 지적합니다. 즉, 앞으로는 단일 점수보다 운영형 지표(실패율, 장기 과업 성공률, 재현성)를 병행하지 않으면 모델 선택 오류가 커질 가능성이 높습니다.
  → [링크: https://arxiv.org/abs/2602.16763]

- **[Mobile-Agent-v3.5: Multi-platform Fundamental GUI Agents]** (arXiv cs.AI)
  Mobile-Agent-v3.5는 GUI-Owl-1.5 계열을 통해 **2B/4B/8B/32B/235B** 다중 스케일을 제시하고, 데스크톱·모바일·브라우저를 아우르는 멀티플랫폼 에이전트 구조를 내세웠습니다. 논문 초록 수치 기준으로 OSWorld **56.5**, AndroidWorld **71.6**, WebArena **48.4** 등 벤치마크 성능이 제시되어 ‘실사용 UI 작업’에 가까운 평가 축을 강화했습니다. 시사점은 “더 큰 모델”보다 “환경 전환 비용이 낮은 에이전트”가 실제 자동화 도입 속도를 좌우한다는 점입니다.
  → [링크: https://arxiv.org/abs/2602.16855]

## 미스 김 인사이트

### 오늘의 핵심 트렌드 3가지
1. **벤치마크 포화 대응이 본격 과제로 올라왔습니다.** “점수 1위”보다 “실제 업무 분리력”이 중요해졌고, 평가 설계 자체가 제품 경쟁력의 일부가 되고 있습니다.
2. **에이전트는 모델 문제가 아니라 운영 문제로 이동 중입니다.** 컨텍스트 파일 품질, 스킬 구성, 권한/보안 가드레일이 성능만큼 중요한 1차 성공 요인이 됐습니다.
3. **시장 확산은 거대 투자 + 젊은 사용자층 선점의 조합으로 가속 중입니다.** 엔터프라이즈 매출 지표와 신흥시장 사용량 지표가 동시에 커지고 있어, 제품/유통 전략을 분리해서 볼 수 없는 국면입니다.
