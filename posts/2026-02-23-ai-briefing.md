## 한눈에 보기
오늘 흐름은 **평가체계 재설계(벤치마크 포화) + 대형 모델 운영비 최적화 + 지역별 수요 편차 확대**로 요약됩니다. 논문 쪽은 “점수 경쟁”보다 오래 버티는 평가 설계와 도메인 에이전트의 실전성으로 무게가 이동했고, 모델 쪽은 파라미터 확장과 동시에 배포 비용·컨텍스트 관리가 핵심 변수로 부상했습니다. 커뮤니티에서는 대회·개발자 생태계 모두에서 생산성보다 재현성과 비용 효율 지표가 더 자주 언급되고 있습니다. 산업 면에서는 대규모 자금 조달과 신흥시장 사용자 급증이 동시에 확인되며, 기술 경쟁이 곧바로 시장 점유율 경쟁으로 연결되는 구간입니다.

## 논문 동향

- **[When AI Benchmarks Plateau: A Systematic Study of Benchmark Saturation]** (arXiv cs.AI)
  이 논문은 주요 모델 개발사 리포트에서 추린 LLM 벤치마크 60개를 분석해, 거의 절반이 이미 포화 구간에 진입했다고 보고했습니다. 저자들은 과제 설계·데이터 구성·평가 형식 등 14개 속성을 비교했고, 특히 공개/비공개 테스트셋 여부만으로는 포화를 막기 어렵다고 지적했습니다. 시사점은 명확합니다: 이번 분기부터는 리더보드 순위 1개보다 **수명 긴 평가셋(노화 속도) + 운영형 지표(재현성·실패율)**를 함께 관리해야 모델 선택 오류를 줄일 수 있습니다.
  → [링크: https://arxiv.org/abs/2602.16763]

- **[OpenEarthAgent: A Unified Framework for Tool-Augmented Geospatial Agents]** (arXiv cs.CV)
  OpenEarthAgent는 위성영상·자연어 질의·GIS 도구 호출을 결합한 지리공간 에이전트 프레임워크를 제시했습니다. 공개된 데이터 규모는 학습 14,538건·평가 1,169건, 추론 스텝은 학습 100K+·평가 7K+로 명시되어 있어 단순 데모가 아니라 다단계 추론 운영을 겨냥한 구성입니다. 재난·도시·인프라 도메인까지 포함했다는 점에서, 범용 VLM보다 **도메인 툴체인 결합형 에이전트**가 실제 B2B 도입 속도를 더 빠르게 가져갈 가능성이 큽니다.
  → [링크: https://arxiv.org/abs/2602.17665]

- **[Calibrate-Then-Act: Cost-Aware Exploration in LLM Agents]** (Hugging Face Daily Papers)
  Hugging Face daily papers 상위 항목(2602.16699)은 LLM 에이전트가 행동 전에 불확실성을 보정(calibration)해 탐색 비용을 줄이는 접근을 제안합니다. 같은 피드에서 해당 논문은 상대적으로 높은 upvote(12)를 기록하며, 단순 정확도보다 비용-행동 균형 문제에 관심이 모이고 있음을 보여줍니다. 실무적으로는 에이전트 설계 시 “정답률 최대화”만이 아니라 **행동 1회당 기대비용**을 KPI에 넣는 구조가 점점 필수가 됩니다.
  → [링크: https://huggingface.co/papers/2602.16699]

- **[Papers with Code 트렌딩의 Hugging Face 연동 고착]** (Papers with Code Trending)
  현재 `paperswithcode.com/trending`은 HTTP 302로 `huggingface.co/papers/trending`으로 리디렉션됩니다. 즉, 연구 트렌드 입구가 사실상 단일 허브로 수렴하고 있어 수집 파이프라인 단순화에는 유리하지만, 소스 다변성 관점에선 편향 리스크가 커질 수 있습니다. 운영 시사점은 하나입니다: 자동 브리핑은 허브 1개에 의존하지 말고 **arXiv 원문·커뮤니티 반응·산업 뉴스를 병렬로 붙여 교차검증**해야 합니다.
  → [링크: https://paperswithcode.com/trending]

## 모델/도구 릴리즈

- **[Qwen3.5-397B-A17B]** (Hugging Face Trending Models API)

## 미스 김 인사이트

### 오늘의 핵심 트렌드 3가지
1. **평가 지표의 수명 관리가 메인 이슈로 전환**됐습니다. 고득점 자체보다 “얼마나 오래 구분력을 유지하는가”가 투자·도입 의사결정의 핵심 변수입니다.
2. **대형 모델 경쟁의 승부처가 운영비/배포비로 이동**하고 있습니다. 모델 확장 자체보다 멀티모달 비용과 도구 호출 실패율을 얼마나 통제하느냐가 실적을 가릅니다.
3. **지역별 채택 속도 차이가 수익화 타이밍을 결정**합니다. 인도처럼 젊은층/업무 사용이 동시에 급증한 시장에서는 로컬 파트너십과 가격정책이 기술력만큼 중요합니다.
