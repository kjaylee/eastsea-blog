- **GPT-5.4·Gemini 3.1·Grok 4.20**: 3월 한 달간 프론티어 모델 3종이 연달아 출시되며 AI 경쟁이 백열전으로 돌입했다.
- **MCP 9,700만 설치**: Model Context Protocol이 16개월 만에 9,700만 설치를 돌파, 에이전트 표준으로 자리 잡았다.
- **비트코인 2,000만 개 채굴**: 전량 공급량의 95% 이상이 채굴되며 희소성 논의가 다시 뜨겁다.

## 🔬 AI/ML

- **1. GPT-5.4·Gemini 3.1·Grok 4.20 — 3월 프론티어 모델 3종 동시 출시**

OpenAI는 3월 17일 GPT-5.4를 표준·Thinking·Pro 세 가지 변형으로 출시했다. Google은 3월 20일 Gemini 3.1 Ultra를 내놓으며 200만 토큰 컨텍스트 윈도우와 멀티모달 통합을 강조했다. xAI의 Grok 4.20은 3월 22일 출시되며 실시간 정보 처리에서 강점을 보였다. 23일 만에 세 모델이 연달아 나오면서 "모델 선택은 누가 더 똑똑한가가 아니라 어떤 모델이 어떤 작업에 맞는가"로 논의가 이동했다.
→ 원문: [March 2026 AI Roundup](https://www.digitalapplied.com/blog/march-2026-ai-roundup-month-that-changed-everything)
→ 교차확인: [AI News March 2026](https://www.humai.blog/ai-news-trends-march-2026-complete-monthly-digest/)

- **2. MCP 9,700만 설치 — 에이전트 표준으로 굳히다**

Model Context Protocol이 3월 25일 기준 9,700만 설치를 돌파했다. Anthropic이 2024년 말 소개한 이 프로토콜은 이제 OpenAI, Google, xAI, Mistral, Cohere 등 모든 주요 AI 제공자가 지원한다. 4,000개 이상의 MCP 서버가 등록되어 있으며, 보안 표준 v1.1도 발표되었다. "MCP는 이제 실험 단계가 아니라 안정적인 인프라"라는 평가다.
→ 원문: [March 2026 AI Roundup](https://www.digitalapplied.com/blog/march-2026-ai-roundup-month-that-changed-everything)
→ 교차확인: [AI News March 2026](https://www.humai.blog/ai-news-trends-march-2026-complete-monthly-digest/)

- **3. OpenAI 인력 두 배 확대 — Anthropic·Google 경쟁 대응**

Financial Times 보도에 따르면 OpenAI는 2026년 말까지 직원 수를 4,500명에서 8,000명으로 늘릴 계획이다. 새로운 채용은 주로 제품 개발, 엔지니어링, 연구, 영업 분야에 집중된다. 샌프란시스코에 100만 평방피트 이상의 사무실 공간을 확보했다. 이는 Anthropic과 Google의 Claude Code·Gemini 경쟁에 대한 대응으로 해석된다.
→ 원문: [OpenAI headcount plan](https://fortune.com/2026/03/21/openai-double-headcount-this-year-sam-altman-anthropic-google/)
→ 교차확인: [AI News March 2026](https://www.humai.blog/ai-news-trends-march-2026-complete-monthly-digest/)

- **4. Sora API 중단 — 비디오 생성 경제성 한계**

OpenAI가 3월 24일 Sora 공개 API를 30일 예고 후 중단한다고 발표했다. 이유는 "경제적으로 지속 불가능"한 비디오 생성 비용이다. 고품질 비디오 1분 생성에 드는 컴퓨팅 비용이 API 가격의 배수였다. 업계는 Runway Gen-4, Pika 2.1, Google Veo 2로 관심이 이동했다. 디즈니가 2025년 12월 Sora와 10억 달러 계약을 체결한 상태라 충격이 컸다.
→ 원문: [March 2026 AI Roundup](https://www.digitalapplied.com/blog/march-2026-ai-roundup-month-that-changed-everything)
→ 교차확인: [OpenAI Sora shutdown](https://www.reuters.com/technology/artificial-intelligence/artificial-intelligencer-openais-852-billion-problem-finding-focus-2026-04-01/)

- **5. Mistral Small 4 — 오픈소스 추격 가속**

Mistral이 3월 3일 220억 파라미터의 Small 4를 Apache 2.0 라이선스로 공개했다. 30B 이하 오픈 모델 중 MMLU-Pro, HumanEval, MATH 벤치마크 최고 성능을 기록했다. 단일 A100이나 양자화로 일반 하드웨어에서도 실행 가능하다. "오픈소스가 '충분히 좋은' 기준을 다시 높였다"는 평가다.
→ 원문: [March 2026 AI Roundup](https://www.digitalapplied.com/blog/march-2026-ai-roundup-month-that-changed-everything)

- **6. ChatGPT 기하학 문제 최초 해결 — 수학 연구 진입**

브뤼셀 자유대 연구팀이 ChatGPT-5.2 Thinking이 기존 미해결 기하학 문제의 증명을 독립적으로 개발했다고 보고했다. 7회 대화와 4회 수정을 거쳐 최종 증명이 완성됐다. "AI가 코딩·작문 지원을 넘어 원천 수학 연구에 기여했다"는 의미다.
→ 원문: [AI News March 2026](https://www.humai.blog/ai-news-trends-march-2026-complete-monthly-digest/)

## 💹 경제/블록체인

- **10. 비트코인 2,000만 개 채굴 달성 — 희소성 현실화**

3월 9일 기준 2,000만 비트코인이 채굴되었다. 전량 공급량 2,100만 개 중 95% 이상이 이미 채굴된 셈이다. 남은 100만 개는 채굴 보상 감소로 인해 수십 년이 걸릴 것이다. 소매 투자자(0.1 BTC 미만 보유)는 보유량을 0.52% 늘린 반면, 고래는 3월 말 25,500 BTC를 매도했다.
→ 원문: [Crypto Market March 2026](https://coinpedia.org/price-analysis/crypto-market-march-2026-war-whales-and-whiplash-define-the-month/)
→ 교차확인: [March 2026 Crypto Recap](https://alphanode.global/insights/march-2026-cryptocurrency/)

- **11. 3월 크립토 시장 — 지정학적 긴장과 변동성**

미국·이스라엘·이란 간 긴장으로 3월 크립토 시장이 요동쳤다. 비트코인은 -2.7%, 이더리움은 +2.1%를 기록했다. S&P 500은 2월 중순~3월 말 -9% 하락했다. Bittensor(TAO)와 MemeCore는 각각 +67% 상승하며 선택적 강세를 보였다. "시장은 차트가 아니라 불확실성을 거래했다"는 분석이다.
→ 원문: [Crypto Market March 2026](https://coinpedia.org/price-analysis/crypto-market-march-2026-war-whales-and-whiplash-define-the-month/)

- **12. FOMC 3월 — 기준금리 동결, 인플레이션 전망 상향**

연준은 기준금리를 3.5~3.75%로 동결했다. 2026년 남은 기간 단 한 차례 금리 인하만 예상된다. 2026년 인플레이션 전망을 2.7%로 상향 조정했다. 브렌트유가 배럴당 116달러 근처에서 거래되며 시스템적 에너지 압력을 인용했다. 비트코인은 발표 후 약 5% 하락하며 71,100달러 지지선을 테스트했다.
→ 원문: [March 2026 FOMC Recap](https://crypto.com/us/market-updates/march-2026-fomc-recap-btc-eth-price)

## 🛠️ 개발도구

- **13. VS Code 1.114 — 채팅 경험 간소화**

Visual Studio Code 1.114가 4월 1일 출시됐다. 채팅 첨부 이미지 캐러셀에서 비디오 미리보기가 가능해졌다. /troubleshoot 명령으로 이전 세션 문제를 진단할 수 있다. 워크스페이스 검색이 단순화되어 #codebase 도구가 순수 시맨틱 검색만 수행한다. TypeScript 6.0이 적용됐으며, Claude 에이전트 비활성화를 위한 그룹 정책도 추가됐다.
→ 원문: [VS Code 1.114](https://code.visualstudio.com/updates)
→ 교차확인: [VS Code Releases GitHub](https://github.com/microsoft/vscode/releases)

- **14. Dapr Agents v1.0 GA — 프로덕션급 에이전트 프레임워크**

Cloud Native Computing Foundation이 3월 23일 Dapr Agents v1.0을 정식 출시했다. 분산 애플리케이션 런타임 위에서 보안 멀티에이전트 조정, 상태 관리, 장애 복구를 제공한다. "에이전트가 프로토타입에서 프로덕션으로 넘어가는 데 필요한 신뢰성 기반"이라는 평가다.
→ 원문: [AI Developer Tools March 2026](https://dev.to/ajay_kumar_1daef5fe089885/ai-developer-tools-enter-autonomous-era-the-rise-of-agentic-systems-in-march-2026-31l0)

- **15. AI 개발도구 자율 시대 진입 — 어시스턴트에서 에이전트로**

3월 2026년은 AI 개발도구가 "도우미"에서 "협력 파트너"로 전환한 달이다. GitHub Copilot, Cursor, Amazon CodeWhisperer, Tabnine 모두 에이전트 기능을 강화했다. GStack Toolkit은 개발 수명주기를 8개 전용 워크플로로 분리해 각 단계에 최적화된 에이전트를 할당한다. "AI를 거부하거나 맹신하는 것이 아니라 인간-AI 파트너십을 조율하는 능력이 핵심이 됐다"는 분석이다.
→ 원문: [AI Developer Tools March 2026](https://dev.to/ajay_kumar_1daef5fe089885/ai-developer-tools-enter-autonomous-era-the-rise-of-agentic-systems-in-march-2026-31l0)
