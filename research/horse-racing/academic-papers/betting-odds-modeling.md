# 경마 배당률 모델링과 통계학적 접근

## 개요

배당률(ODDS)은 경마에서 핵심적인 정보이다. 본 문서는 배당률의 수학적 모델링과 예측 연구를 정리한다.

---

## 1. 배당률의 구조

### 1.1 기본 개념

| 용어 | 정의 |
|------|------|
| 파레이 (Paradox) | 균형 배당률 |
| 홀 HOUSE | 경마장 수익률 (약 17-25%) |
| 네트 파이어 | 총 투표액 대비配当금 비율 |

### 1.2的类型

- **승률**: 단승, 복승
- **연승률**: 연승, 쌍승, 복연승
- **승아스타**: 1-2위 순서 맞추기

## 2. 통계적 모델

### 2.1 회귀 모델

```python
# 단순 선형 회귀 예시
P(win) = β0 + β1(speed_rating) + β2(recency) + β3(class)
```

### 2.2 베이지안 모델

- Prior: 과거 성적 데이터
- Likelihood: 현재 조건
- Posterior: 예측 확률

### 2.3 머신러닝 접근

| 방법 | 정확도 | 한계 |
|------|--------|------|
| 로지스틱 회귀 | 52-55% | 과적합 위험 |
| 랜덤 포레스트 | 55-58% | 해석 어려움 |
| LSTM | 58-62% | 데이터 요구량 |

---

## 3. 효율적 시장 가설 (EMH)

### 3.1 경마 시장 특성

경마 배당률은 **약한 효율적 시장(Weak-form EMH)** 으로 간주된다.

- 공개 정보는 빠르게 반영
- 비공개 정보(내부 정보)로 수익 가능

### 3.2 베팅 전략

1. **가치 베팅 (Value Betting)**: 순위 ^ 1 이상일 때 배팅
2. **핸디캡 식별**: 레이서가 설정한 예상과 실제 차이
3. **트랙 핸디캡**: 특정 트랙 선호도

---

## 4. 한국 배당률 특성

### 4.1 한국 경마 배당 구조

- 단승: 1.5배 ~ 100배+
- 복승: 1.1배 ~ 50배
- 연승: 1.1배 ~ 500배+

### 4.2 시기별 패턴

| 시기 | 특징 |
|------|------|
| 시즌 초 | 변동성 높음 |
| 시즌 중 | 데이터 축적, 예측 향상 |
| 시즌 말 | 안정적 패턴 |
