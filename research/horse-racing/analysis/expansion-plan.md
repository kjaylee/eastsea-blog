# 코퍼스 확장 계획

## 현재 판단

- 기존 코퍼스는 예측, 운영, 복지, 혈통, 글로벌 레이스 쪽은 상당 부분 채워져 있다.
- 구조상 가장 약한 부분은 `industry economics`와 `live editorial materials`다.
- 새 요구사항으로 `말 보행/파행 분석용 영상자료` 큐레이션이 추가되었다.
- 디렉토리 구조상 `history/`는 비어 있어 전용 역사 섹션이 부재하다.

## 부족한 부분

1. 산업경제
   - 경마 산업 매출 구조
   - 국가/기관별 베팅 핸들, attendance, purse, ADW 비중
   - 세금/환급/재투자 구조

2. 역사 전용 섹션
   - 한국 경마 연표
   - 제도 변화와 경마장 이동
   - 국제화 이정표

3. 영상 기반 파행 분석 자료
   - trot-up
   - straight line vs circle
   - forelimb/hindlimb 예시
   - slow motion gait comparison
   - objective gait analysis 도구 설명

4. 정보지용 재료
   - 정기 발행용 꼭지 체계
   - 공식 결과/통계 데이터 소스
   - 규제·복지·산업 뉴스 흐름
   - 사진/영상/브랜드 사용 리스크

## 필요한 자료

### A. 산업경제 소스

- The Jockey Club Fact Book
- KRA Key Facts / Racing Statistic
- Kentucky Horse Racing and Gaming pari-mutuel reports
- IFHA/ARF 계열 betting duty / turnover 자료

### B. 파행 분석 소스

- 수의학회/수의대 교육자료
- objective gait analysis 설명 자료
- YouTube 교육 영상
- 느린 화면 보행 비교 영상

### C. 정보지 제작 소스

- 공식 일정/결과/기록 페이지
- 뉴스/보도자료/규정 개정 소스
- 말산업/복지/수의학 해설 소스

## 우선순위

1. 산업경제 보강
2. 파행 분석용 영상 큐레이션
3. 역사 전용 문서 확보
4. 정보지 발행 가능성 검토와 편집 구조 설계
