# 경마 정보지 가능성 검토

## 결론

- **가능하다.**
- 다만 현재 코퍼스로 바로 만들 수 있는 것은 `주간/격주 분석형 정보지`다.
- 아직 바로 만들기 어려운 것은 `실시간 배당·출전 변경·속보 중심 일간 정보지`다.

## 현재 코퍼스로 가능한 것

1. 분석형 특집
   - 데이터 분석
   - 말산업 구조
   - 혈통/훈련/복지
   - 세계 주요 레이스와 제도 비교

2. 정기 꼭지형 콘텐츠
   - 이번 주 산업 지표
   - 규제/복지 브리핑
   - 말 보행·파행 분석 코너
   - 앱/서비스 리뷰
   - 커뮤니티/미디어 모니터링

3. 교육형 콘텐츠
   - 경마 용어 해설
   - 배당과 데이터 읽는 법
   - 수의학·보행 분석 기초

## 아직 부족한 것

1. 실시간성
   - live odds
   - scratch / withdrawal
   - 직전 track condition
   - race-by-race card updates

2. 편집 운영
   - 발행 템플릿
   - 기사별 fact-check workflow
   - source freshness rules
   - issue calendar

3. 권리와 정책
   - 사진 사용권
   - 영상 썸네일/임베드 정책
   - 데이터 재배포 허용 범위

## 가능한 정보지 형태

### A. 지금 바로 가능한 형태

- `주간 경마 브리프`
- `말산업/데이터 분석 정보지`
- `교육형 경마 인사이트 레터`

### B. 추가 개발 후 가능한 형태

- `경주일 직전 분석지`
- `트랙별 race card 브리프`
- `실시간 업데이트형 배당·출전 정보지`

## 필요한 추가 데이터 소스

1. 공식 결과/통계
   - KRA Racing Statistic
   - Equibase official results/statistics
   - The Jockey Club Fact Book

2. 산업/행정
   - KRA Key Facts
   - KHRC pari-mutuel reports
   - IFHA / ARF wagering papers

3. 복지/수의학
   - HISA / HIWU
   - AAEP
   - Equinosis / 수의대 교육 자료

## 권장 발행 모델

1. 1단계
   - 주간 PDF 또는 Markdown 기반 정보지
   - 분석/브리핑/교육형 중심

2. 2단계
   - structured race data ingestion
   - 일정/결과/배당 자동 반영

3. 3단계
   - 한국 경마 + 글로벌 레이스 혼합형 멀티섹션 정보지

## 실무 판단

- **현재 프로젝트만으로도 정보지의 뼈대는 충분하다.**
- 강한 영역은 `분석`, `교육`, `산업 해설`, `수의학/복지`, `주제형 특집`이다.
- 약한 영역은 `실시간성`, `편집 자동화`, `데이터 권리 검토`다.
- 따라서 첫 제품은 `분석형 정보지`, 이후 확장형으로 `실시간 경마 정보지`가 적절하다.
