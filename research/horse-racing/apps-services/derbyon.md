# 더비온 (DerbyON) — 한국마사회 공식 온라인 마권 앱

## 개요
- **정식명**: DerbyON (더비온)
- **운영**: 한국마사회 (KRA, Korea Racing Authority)
- **출시**: 2024년 6월 19일 정식 서비스 개시
- **플랫폼**: 안드로이드 (구글 플레이스토어 미통과, 한국마사회 공식 웹사이트 직접 설치)
- ** hérit**: 기존 '전자카드 4.0' / '마이카드' 앱을 대체

## 주요 기능

| 기능 | 설명 |
|------|------|
| **온라인 마권 구매** | 전국 경마장(서울·부산·제주) 마권 실시간 구매 |
| **실시간 경주 중계** | 고화질 라이브 스트리밍, 전국 경마장 경기 동시 시청 |
| **배당률 확인** | 실시간 배당 추이 모니터링 |
| **충전 / 출금** | 계좌이체로 충전, 당첨금 자동 환급 |
| **지문 인증** | 생체 인증 기반 보안 체계 |
| **자가 이용 제한** | 스스로 금액·시간 한도 설정 가능 (책임 있는 베팅) |
| **출마표 / 출발 정보** | 경주 출마표, 과거 성적, 기수 정보 등 |

## 설치 방법

1. 한국마사회 공식 웹사이트(`derbyon.kra.co.kr`) 또는 `m.kra.co.kr` 방문
2. 'APP 다운로드' 메뉴选择
3. APK 파일 직접 다운로드 및 설치 (안드로이드만 가능)
4. 한국마사회 전자카드 창구에서 **대면 가입** (신분증 확인)
5. 회원번호 부여 후 앱 로그인

## 이용 조건

- 만 21세 이상 대한민국 성인
- 본인 명의 은행 계좌 필수
- 1회 충전 한도: 별도 안내 (이용자 등급에 따라 상이)

## 장점

- 경마장 방문 없이 전국 경기 참여 가능
- 실시간 고화질 중계
- 통합 정보 제공 (출마표·배당·결과一元)
- 책임 베팅 시스템 내장

## 단점 / 주의사항

- iOS 미지원 (안드로이드 전용)
- 설치 시 '출처 불명' 허용 필요
- 초보자는 대면 가입 절차 번거로움
- 충전 한도 관리 필요
