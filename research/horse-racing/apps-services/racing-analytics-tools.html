경마 분석 소프트웨어/도구

개요

경마 분석에 활용되는 주요 소프트웨어와 도구를 정리한다. 무료/유료, 웹/앱 기준으로 분류한다.

데이터 분석 도구

1. DRF (Daily Racing Form)

항목내용
유형웹 + 앱
가격월 $19.99~
지역미국
특징공인 데이터 제공

주요 기능

  • 레이서별 과거 성적
  • 트레이너/기수 통계
  • 트랙 조건별 분석
  • 베팅 계산기

2. Brisnet

항목내용
유형웹
가격월 $25~
지역미국

제공 데이터

  • Speed Figure (공식)
  • Performance Rating
  • Distance Analysis

3. TimeformUS

항목내용
유형웹 + 앱
가격월 $15~
지역미국, 영국

특징

  • 시각적 데이터 표현
  • 직관적 UI
  • 전문가 평론 포함

한국 경마 도구

KRA 공식 사이트

항목내용
URLkra.co.kr
가격무료
제공실시간 오딧, 과거 데이터

기능

  • 마체 정보 조회
  • 경주 결과 조회
  • 오딧 영상
  • 성적 통계

실전 적용 예시

# 의사코드: 레이서 승률 예측
features = [
    recent_form,      # 최근 5경기 平均順位
    class_level,      # 등급
    distance_score,   # 거리 적성도
    trainer_rating,   # 조련사 평가
    track_condition,  # 트랙 상태
]

model = load_trained_model()
prediction = model.predict(features)

보안 및 규정

  • 공식 데이터: 상업적 사용 시 라이선스 필요 가능
  • 개인 용도: 대부분의 분석 도구 개인 사용 가능
