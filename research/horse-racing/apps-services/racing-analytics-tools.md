# 경마 분석 소프트웨어/도구

## 개요

경마 분석에 활용되는 주요 소프트웨어와 도구를 정리한다. 무료/유료, 웹/앱 기준으로 분류한다.

## 데이터 분석 도구

### 1. DRF (Daily Racing Form)

| 항목 | 내용 |
|------|------|
| 유형 | 웹 + 앱 |
| 가격 | 월 $19.99~ |
| 지역 | 미국 |
| 특징 | 공인 데이터 제공 |

#### 주요 기능
- 레이서별 과거 성적
- 트레이너/기수 통계
- 트랙 조건별 분석
- 베팅 계산기

### 2. Brisnet

| 항목 | 내용 |
|------|------|
| 유형 | 웹 |
| 가격 | 월 $25~ |
| 지역 | 미국 |

#### 제공 데이터
- Speed Figure (공식)
- Performance Rating
- Distance Analysis

### 3. TimeformUS

| 항목 | 내용 |
|------|------|
| 유형 | 웹 + 앱 |
| 가격 | 월 $15~ |
| 지역 | 미국, 영국 |

#### 특징
- 시각적 데이터 표현
- 직관적 UI
- 전문가 평론 포함

### KRA 공식 사이트

| 항목 | 내용 |

| 제공 | 실시간 오딧, 과거 데이터 |

#### 기능
- 마체 정보 조회
- 경주 결과 조회

| 도구명 | 유형 | 설명 |

| 말재Keeper | 앱 | 마체 정보 관리 |

## 스프레드시트 템플릿

### 무료 템플릿

#### 레이서 추적 시트
```excel
마체명 | 최근5경기 | 평균順位 | 거리적성 | 트랙偏向 | 점수
```

#### 베팅 관리 시트
```excel
날짜 | 경기 | 마체 | 배당률 | 베팅액 | 결과 | 손익
```

### 유료 템플릿
- Advanced Handicapping Sheet (~$15)
- Pro Racing Tracker (~$25)

## 프로그래밍 도구

### 데이터 수집

| 도구 | 용도 | 난이도 |
|------|------|--------|
| Python + BeautifulSoup | 웹 스크래핑 | 중급 |
| R (rvest) | 데이터 분석 | 중급 |
| Excel Power Query | 데이터 변환 | 초급 |

### 분석 라이브러리

| 라이브러리 | 언어 | 용도 |
|------------|------|------|
| Pandas | Python | 데이터 처리 |
| Caret | R | 머신러닝 |
| Scikit-learn | Python | 예측 모델 |

## 머신러닝/AI 도구

### 예측 모델

| 도구 | 설명 | 접근성 |
|------|------|--------|
| TensorFlow | 딥러닝 프레임워크 | 고급 |
| H2O.ai | AutoML 플랫폼 | 중급 |
| DataRobot | 자동 머신러닝 | 초급 |

### 실전 적용 예시

```python
# 의사코드: 레이서 승률 예측
features = [
    recent_form,      # 최근 5경기 平均順位
    class_level,      # 등급
    distance_score,   # 거리 적성도
    trainer_rating,   # 조련사 평가
    track_condition,  # 트랙 상태
]

model = load_trained_model()
prediction = model.predict(features)
```

## 도구 선택 가이드

| 목적 | 추천 도구 | 예산 |
|------|-----------|------|
| 초보 입문 | KRA 공식 + 무료 템플릿 | 무료 |
| 중급 분석 | DRF + TimeformUS | ~$20/월 |
| 고급 연구 | Brisnet + Python 분석 | ~$30/월 + 프로그래밍 |

## 보안 및 규정

- **웹 스크래핑**: 각 사이트의 robots.txt 및 이용약관 확인
- **공식 데이터**: 상업적 사용 시 라이선스 필요 가능
- **개인 용도**: 대부분의 분석 도구 개인 사용 가능
