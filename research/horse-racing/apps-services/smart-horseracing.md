# 스마트 경마 (Smart Horse Racing) — 예상 및 정보 앱

## 개요
- **정식명**: Smart Horse Racing — Horse Racing Prediction Information Result News
- **플랫폼**: 안드로이드 (Google Play)
- **특징**: 예상·결과·소식于一身的 종합 정보 앱

## 주요 기능

| 기능 | 설명 |
|------|------|
| **경마 예상** | 빅데이터 기반 출마 분석 및 예상 제공 |
| **결과 확인** | 실시간 경기 결과 및 복기 |
| **속보 뉴스** | 경마 관련 최신 소식 전달 |
| **배당률 추적** | 실시간 배당 변동 확인 |
| **다트랙 지원** | 서울·부산·제주 등 국내 전 경마장 |

## 데이터 기반 예상 시스템

- 과거 경주 데이터 활용 통계 분석
- 마필 컨디션 지수
- 기수·조교사 과거 성적
- 배당률 패턴 분석

## 장점

- 설치 간편 (구글 플레이스토어에서 immed_download)
- 종합 정보 앱으로 다양한 기능 원스톱利用
- 실시간 업데이트
