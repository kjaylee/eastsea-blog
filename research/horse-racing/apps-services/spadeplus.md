# 스페이드플러스 (SpadePlus) — 경마 분석 앱

## 개요
- **정식명**: SpadePlus (스페이드플러스)
- **플랫폼**: 안드로이드 (Google Play)
- **개발사**: BP3Soft (猜測)
- **목적**: 경마 입문자부터 전문가까지 위한 차트 기반 분석 도구

## 주요 기능

| 기능 | 설명 |
|------|------|
| **차트 분석** | 시각적 차트로 경마 기록 분석 |
| **통계 제공** | 마필·기수·조교사 상세 통계 |
| **실시간 정보** | 출마표, 배당률, 경주 결과 |
| **입문자 지원** | 초보도 이해하기 쉬운 해설 제공 |
| **커뮤니티 연계** | 검빛경마 등 커뮤니티 데이터 활용 |

## 대상 사용자

- 경마 **입문자**: 기본 차트 읽는 법 학습
- **중급자**: 데이터 기반 예상 역량 강화
- **전문가**: 통계 수치 활용 심화 분석

## 기존 유사 앱 대비

- 차트 기반 시각화 특화
- 과거 성적 데이터检索 기능
- 다양한 필터링 옵션 제공
