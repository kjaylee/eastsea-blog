# 경주마 심박수 회복(Heart Rate Recovery) 분석

## 개요

경주 직후 심박수 회복 속도는 말이 경기 후 얼마나 빨리 회복하는지를 보여주며, 향후 성능 예측에 활용됩니다. 운동생리학(Exercise Physiology) 분야에서 활발히 연구되고 있습니다.

## 핵심 지표

### HRR (Heart Rate Recovery)
- **정의**: 경주 종료 후 1분/2분/5분 시점 심박수
- **계산**: HRR1 = HR종료 - HR1분후
- **정상 범위**: 1분에 30~50 BPM 하락

### 핵심 비율
```
Recovery Index = (HRmax - HR1min) / (HRmax - HRrest)
```
- 1에 가까울수록 빠른 회복 (우수한 심폐 기능)

## 측정 방법

### 전통적 방식
- 심전도(ECG) 모니터링
- 심박계(Polar 등)를使った 경주 중 실시간 측정
- Race Data Systems社 장비를 통해 실전 적용

### 최근 기술
- 장착형 센서 (등 중앙 배치)
- GPS + 심박 통합 추적
- AI 기반 영상 분석 (열화상 카메라)

## 연구 결과와 활용

### 주요 발견 (Sch也想 등 2021)

| 지표 | 高性能 마 | 평균 이하 마 |
|------|----------|-------------|
| HR 종료 시 | 210±8 BPM | 218±12 BPM |
| HR 1분 후 | 165±10 BPM | 185±15 BPM |
| HRR 1분 | 45±8 BPM | 33±10 BPM |

- **결론**: 회복이 빠른 말이次回 경기에서 더 높은 순위

### Recovery Rate와 관련 변수

1. **트레드밀 테스트**: 경주 전 유산소 능력 지표
2. **혈중 젖산(Lactate)**: 회복과 상관관계 (r=0.67)
3. **체온**: 과도한 체온 상승은 회복 저하 신호

## 실전 적용

### 데이터 수집 포인트

```
경주 전 30분: 휴식 심박수 (baseline)
경주 중 5분: 최고 심박수 + 경과 시간
경주 직후: 즉시 심박수
경주 후 1분: HRR1 측정
경주 후 5분: 완전 회복 여부 확인
```

### 예측 모델링

```python
# Conceptual Feature Set
features = [
    'resting_hr',           # 휴식 시 심박
    'max_hr',               # 최대 심박
    'hrr_1min',             # 1분 회복량
    'hrr_5min',             # 5분 회복량
    'lactate_post',         #赛后 젖산
    'temperature_delta',     # 체온 변화
    'race_distance',        # 경주 거리
    'weather',              # 날씨 조건
]
```

### 경기력 예측 정확도 향상

- 심박수 데이터 단독: Baseline +3%
- 기존 모델과 결합 시: AUROC 0.72 → 0.78
- **특히 2월~3월 시즌 初戦 예측에 효과적**

## 상관관계 분석

### 회복력이 높은 말의 특징

| 변수 | 상관관계 | 방향 |
|------|---------|------|
| 나이 | -0.34 | 연령 증가 시 회복 저하 |
| 체중 | -0.21 | 체중 과체중 시 회복 저하 |
| 경주 경험 | +0.45 | 경험 많을수록 회복 좋음 |
| 휴식 기간 | +0.38 | 충분한 휴식 시 회복 향상 |

## 연구 한계 및 향후 과제

1. **측정 환경**: 실경주와 트레드밀의 차이
2. **개별 차이**: 말이 개인의 심박 baseline이 다름
3. **센서 오류**: 운동 중 센서 이탈 문제
4. **데이터 통합**: Racing Form에 표준화되지 않음
