# 경마 데이터 분석 커리어

## 경마 데이터 분석이란

경마 데이터 분석은 **경주 성적, 마匹血统, 트랙 상태, 기수 기록** 등 다양한 변수를 종합하여 승률 예측, 마匹 컨디셔닝 진단, 배당률 최적화를 수행하는 분야다. 한국 경마의 경우 한국마사회(KRA)에서 공식 데이터를 제공하고 있으며, 해외에서는 Betfair, racingpost 등 플랫폼이 방대한 데이터를 축적하고 있다.

---

## 데이터 수집 출처

### 한국

| 출처 | 데이터 유형 | 접근 방식 |
|------|-----------|---------|
| 한국마사회 홈페이지 | 경주 결과, 마기록, 기수 기록 | 무료 공개 |
| KRATCH 시스템 | 실시간 배당률, 경주 진행 상황 | API (제한적) |
| 경마 전문 매체 | 분석 기사, 전문가 예측 | 웹 크롤링 |

### 해외

| 출처 | 국가 | 특징 |
|------|------|------|
| racingpost | 영국 | 최대 규모의 경마 데이터베이스 |
| equibase | 미국 |血统·경주 기록 광범위 |
| betfair | 글로벌 | 실시간 배당률·거래량 데이터 |

## 분석 프레임워크

### 1. Descriptive Analysis (기술적 분석)

**목적:** 과거 경주 결과를 정리하고 패턴을 발견

```
마匹 성적 요약 → 트랙별 기록 → 거리별 최적 거리 도출
```

**사용 지표:**
- Win Rate (승률)
- Place Rate (입상률)
- Average Margin (평균 차이)
- Speed Figure (속도 수치)

### 2. Predictive Analysis (예측적 분석)

**목적:** 향후 경주 결과를 예측

**주요 모델링 기법:**

| 기법 | 적용 사례 |
|------|---------|
| Logistic Regression | 승패 이진 분류 |
| Random Forest | 변수 중요도 분석 |
| Gradient Boosting | 승률 예측 모델 |
| Neural Network | 복잡한 패턴 학습 |
| Bayesian Inference | 불확실성 하의 예측 |

**입력 변수:**
- 최근 5경기 성적
- 트랙·거리 최적 여부
- 기수·조련사 교체 이력
- 체중 변화 추이
- 혈통 유사경주 성적

### 3. Prescriptive Analysis (처방적 분석)

**목적:** 최적의 배팅 전략 설계

- 배당률 vs 승률 괴리(overround) 분석
- Kelly Criterion 기반 배팅 금액 배분
- Portfolio 이론을 응용한多元化 배팅

## 커리어路径

### 데이터 애널리스트 → 데이터 사이언티스트

```
1~2년차: 성적 데이터 정리·간단한 보고서 작성
3~4년차: 예측 모델 개발·검증
5년차+: 마사·경마장 기업 컨설팅, 자체 모델 상용화
```

### 전문화 방향

| 방향 | 설명 |
|------|------|
| **배당률 분석** |책에 의존하지 않는 자체 배당률 모델 개발 |
| **마匹 건강 예측** | 부상 예방·컨디셔닝 모니터링 시스템 |
| **트랙 특성 분석** | 트랙별胜률 편향 연구 |
| **기수 성과 분석** | 기수별 강점·약점 도출 |

## 윤리적 고려사항

- **정보 비대칭 이용:** 공개되지 않은 내부 정보(말 건강 악화 등)를 분석에 활용하면 **법적 문제** 발생 가능
- **도박 과몰입:** 분석 결과를 타인에게 무분별하게 배포 시 사회적 문제 초래 가능
- **머신러닝 윤리:** 모델의 의사결정 과정을 투명하게 설명할 수 있어야 함 (특허 관련)
