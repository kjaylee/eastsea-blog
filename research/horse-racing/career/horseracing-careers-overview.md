# 경마 관련 커리어 개요

## 경마 산업의 직업 분류

경마 산업은 **마匹 관리**, **경기 운영**, **비즈니스**, **미디어**, **연구** 등 다양한 직종으로 구성된다. 한국에서는 한국마사회(KRA)가 주요 公營 기관이며, 그 외 사육·트레이닝·배팅 관련 민간 기업이 생태계를 형성한다.

---

## 주요 직종군 지도

### 1. 마匹 관리 계열

| 직업 | 설명 | 필요 자격·경력 |
|------|------|--------------|
| 마사(mare owner) | 번식용 암말 보유·관리 | 대규모 자본, 농업 경력 |
| 번식 전문가(breeding manager) | 혈통 분석, 교배 계획 수립 | 수의학·축산학 배경 |
| 조련사(trainer) | 경주마 훈련 프로그램 설계·실행 | 5년 이상 승마 경력, 면허 |
| 마체전(horse breaker) | 야생마·新生마 조련 | 3년 이상 실기 경험 |
| 마의료 전문가(horse vet) | 마匹 질병 예방·치료 | 수의사 면허 필수 |

### 2. 경기 운영 계열

| 직업 | 설명 | 필요 자격·경력 |
|------|------|--------------|
| 기수(jockey) | 경주 마 승마·조작 | 3년 이상 수련, 체중 관리 (48~54kg) |
| 부기수(amateur jockey) | 비전문 경주 기수 | 승마 능력 입문 |
| 경마장 운영 staff | 안내·안내·매표 | 고졸 이상, 서비스업 경험 |
| 트랙 관리자(track superintendent) | 트랙 상태 유지·관리 | 조경·농업 경력 |

### 3. 비즈니스 계열

| 직업 | 설명 | 필요 자격·경력 |
|------|------|--------------|
| 마匹 거래상(horse dealer) | 마匹 매매·수출입 | 마匹 감정 능력, 네트워크 |
| 경마 애널리스트 | 배당률·성적 데이터 분석 | 통계학·금융 배경 |
| 마주(horse owner) | 경주마 소유·투자 | 자본 투자, 마匹 소유 의향 |
| 배팅 시스템 개발자 | 승률 예측 모델 개발 | 데이터 과학·ML 경력 |

### 4. 미디어·교육 계열

| 직업 | 설명 | 필요 자격·경력 |
|------|------|--------------|
| 경마 해설가 | 경기 해설·马匹 분석 | 경마 경력 10년+ |
| 경마 기자 | 경마 뉴스 취재·报道 | 신문사 경력 |
| 승마 강사 | 승마 교육·입문 과정 | 승마 지도자 자격证 |
| 심판(referee) | 경주 규칙 위반 판정 | 경마 경력, 법규 지식 |

## 커리어 진입 경로

### 기수 (Jockey) 길

```
승마 경험 (5년+) → 부기수 수련 (2~3년) → 한국마사회 기수 시험 → 
마사회 소속 기수 (3년 수련) → 독립 기수 / 국제 대회 진출
```

- **체중 관리**: 일생 통해 가장 어려운 도전
- **부상 위험**: 낙마·충돌로 인한 부상률이 높음
- **수입**: 소속 마주 협약에 따라 정액报酬 + 상금 인센티브

### 조련사 (Trainer) 길

```
승마 경력 (5년+) → 부조련사 (3년) → 조련사 시험 (한국마사회) → 
소속 마 승인 → 독립 조련사 활동
```

- **수입**: 관리 마匹 수 × 마당료 + 경주 상금 %.
- **핵심 능력**: 마匹 심리 파악, 컨디셔닝 설계,獣医 네트워크

### 마주 (Horse Owner) 길

```
자본 투자 의사결정 → 마匹 구매 (국내· 해외) → 조련사 계약 → 
경주 출전 → 상금 + 마匹 시세 차익
```

- **리스크**: 마匹 부상·성적 부진으로 투자 회수 어려움
- **소요 비용**: 마匹 구매 2,000만~5억 원, 연간 관리비 1,000만~3,000만 원
