경마 전문직 역할 및 커리어 패스

기수 (Jockey) 전문 분석

일과 및 생활 패턴

기수의 하루는 체중 관리와 말과의 교감으로 구성된特殊な 생활이다.

시간활동
05:00기상, 체중 측정
05:30~07:00새벽 훈련 (트랙 질주 또는 도보)
07:30아침 식사 (고단백·저열량 식단)
09:00~11:00트레이닝 보드 검토
12:00점심 식사 (체중 조절식)
13:00~16:00마匹와의 개별 연계 훈련
17:00체중 재측정, 운동복 복용
18:00저녁 식사, 수면 준비

수입 구조

기수의 수입은 기본 급여 + 경주 상금 인센티브 + 스폰서십으로 구성된다.

  • 마사회 소속 기수: 연봉 3,000만~8,000만 원 (경력별)
  • 상금 인센티브: 입상 시 상금의 5~10%
  • 스폰서십: 유명 기수의 경우 마사·기업과 마케팅 계약

커리어 레벨

초급 (1~3년) → 중급 (4~7년, G1 경험) → 고급 (8년+, 명망 확립) → 
레전드 (국제 교류, 역사적 기록)

조련사 (Trainer) 전문 분석

조련사의 역할

조련사는 경주마의 경기력 총책임자다. 훈련 설계, 컨디셔닝, 마匹 선정, 경기 전략 수립을 모두 관장한다.

수입 구조

항목금액
마당료 (마匹 1마리당 월)150~300만 원
경주 상금 인센티브입상 시 5~10%
대회 상금 (소속 마匹 성과)실적으로 변동

조련사 커리어 패스

승마功底 → 부조련사 (3년) → 독립 조련사 시험 → 마 승인 → 
소속 마 확보 → 전문 마방组建 → 국제 트레이너 활동

마사 (Horse Breeder/Owner)

마사의 세계

마사는 경주마 산업의 上游에 위치하며, 혈통 관리·번식 기획이 핵심 업무다.

번식 프로세스:

  1. 혈통 분석: 系図調査, 近親交配 위험도 검토
  2. 교배 상대 선정: 수말(stallion)의 유전적 강점 分析
  3. 임신·분만 관리: 수의사 협업
  4. 신생마 관리: 초기 발육 모니터링
  5. 판매·경주 출전 결정: 성장 추적 후 최종 판단

마匹 투자conomics

단계비용기대 효과
번식용 암말 유지 (연간)1,500~3,000만 원혈통 보존
교배비 (유명한 수말)500만~5,000만 원고품질 자식 기대
자ungkau养育비 (2~3년)3,000~6,000만 원경주마 완성
경주마 판매 (2~3살)2,000만~10억 원초기 투자 회수

데이터·기술 전문가

경마 데이터 사이언티스트

경마의 디지털 전환으로 데이터 기반 의사결정 전문가의 수요가 증가하고 있다.

주요 업무:

  • 경주 성적 데이터 수집·정제
  • 배당률 예측 모델 개발
  • 마匹 컨디셔닝 모니터링 시스템 구축
  • 트랙 상태와 경기 결과 상관관계 분석

필요 스킬:

  • Python / R / SQL
  • Machine Learning (Random Forest, XGBoost 등)
  • 데이터 시각화 (Tableau, Power BI)
  • 경마 도메인 지식

경마 앱·플랫폼 개발자

플랫폼핵심 기능요구 기술
