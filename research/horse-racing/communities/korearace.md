# 코리아레이스 (Korearace) — 경마 종합 정보 사이트

## 개요
- **정식명**: 코리아레이스 (Korearace)
- **유형**: 경마 종합 정보 및 커뮤니티
- **주소**: https://www.korearace.com/ (모바일: m.korearace.com)
- **특징**: 경마AI 마패, 실시간 결과, VOD 방송, 전문가 분석, 대화방

## 주요 서비스

### 1. 경마AI 마패
- AI 기반 마필 분석 데이터 제공
- 마필별 성능指數 시각화
- 출전 조건별 맞춤 분석

### 2. 실시간 경마 결과
- 전국 경마장 실시간 결과 업데이트
- 배당률 추이 그래프
- 경주별 상세 기록

### 3. VOD 방송 아카이브
- 경기 영상 다시보기
- 전문가 해설 영상
- 명경주 하이라이트

### 4. 전문가 분석
- 각 분야 전문가 评论 및 예상
- 실시간Discuss 채널 (경마 대화방)
- 복기 분석

### 5. 경마/경정 종합 정보
- 서울·부산·제주 전 경기 정보
- 마필血统·성적数据库
- 경정(경주定型) 분석

## 유튜브 콘텐츠

- **불독경마**: 고배당 패턴 전문, 압도적 복병 찾기
- **삼복승 분석**: 제주경마 전문 예상
- **구디 分析**: 마필별 심층 분석

## 커뮤니티 기능

- 실시간 채팅방
- 회원 간 예상 공유
- 복기 후기 게시판
- 자료室 (통계·기록)
