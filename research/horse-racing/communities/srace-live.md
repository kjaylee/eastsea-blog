# 서울경마 라이브 (SRace Live) — 실시간 경마 중계 및 분석 플랫폼

## 개요
- **정식명**: 서울경마 라이브
- **주소**: https://live.srace.com/
- **유형**: 실시간 중계 + 전문가 예상 + 커뮤니티
- **특징**: 100인 전문가의 종합 예상, 실시간 배당추이, 무료 서비스

## 주요 기능

### 실시간 중계
- 서울경마 전 경기 실시간 중계
- 전문가 라이브 예상
- 실시간 배당추이 모니터링

### 100인 전문가 시스템
- 100명의 전문가 pool
- 각 전문가별 예상 공개
- 종합 예상 투표 결과 제공

### 배당률 추이
- 실시간 배당 변동 추적
- 그래프 시각화
- 직관적 Interface

### 경주 결과
- 빠른 결과 업데이트
- 상세 복기 데이터
- 적중 결과自動 정리

## 서비스 특징

- **무료**: 별도 가입 없이基本 기능 무료利用
- **실시간**: 경기 진행 중 실시간 정보 갱신
- **전문성**: 다수 전문가 예상 통합 제공

## 연계 서비스

- **한국마사회 공식**: KRA 경마 정보 연계
- **검빛경마**: 커뮤니티 자료 활용
- **코리아레이스**: AI 분석 데이터 활용

## 활용 팁

1. 경기 전: 100인 전문가 예상 참고
2. 경기 중: 실시간 배당추이 확인
3. 경기 후: 상세 복기 데이터 분석

## 외부 자료
- 공식: https://live.srace.com/
