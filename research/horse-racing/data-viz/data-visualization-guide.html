경마 데이터 시각화 가이드

경마 데이터 시각화의 목적

경마 데이터 시각화는 복잡한 경마 통계를 직관적으로 이해하도록 돕는 핵심 도구다. 마匹 성적 추이, 배당률 흐름, 트랙별 선호도, 기수별 강점 등을 시각적으로 표현하여 빠르고 정확한 의사결정을 지원한다.

---

핵심 시각화 유형

1. 시간 추이 차트 (Performance Trend)

활용 데이터: 마匹의 최근 경주 성적 변화

차트 유형: 꺾은선 그래프 (Line Chart)

마匹 최근 10경기 성적 추이:
x축: 경주 번호 (1~10)
y축: 순위 (1~10)
→ 하락 추세 → 경향 경고

Python 구현:

import matplotlib.pyplot as plt

races = list(range(1, 11))
rank = [3, 2, 4, 1, 5, 2, 3, 6, 4, 2]

plt.figure(figsize=(10, 5))
plt.plot(races, rank, marker='o', linewidth=2)
plt.gca().invert_yaxis()  # 순위가 낮을수록 좋으므로 반전
plt.title("마匹 최근 경주 추이")
plt.xlabel("경주 번호")
plt.ylabel("순위")
plt.grid(True, alpha=0.3)
plt.savefig('performance_trend.png')

---

2. 배당률 흐름 차트 (Odds Flow)

활용 데이터: 배팅 시작~종료까지의 배당률 변화

차트 유형: 영역 차트 (Area Chart) 또는 다중 선 그래프

해석 포인트:

  • 배당률이 급락 → 많은 배팅이 집중된热门马
  • 배당률이 급등 → 불안감 또는 내부 정보 유출 시사
  • 배당률 안정 → 시장이 마匹에 대해 의견을 모으는 중

---

3. 트랙·거리 히트맵 (Track-Distance Heatmap)

활용 데이터: 마匹의 트랙별·거리별 입상률

차트 유형: 히트맵 (Heatmap)

         1000m  1200m  1400m  1600m  2000m
서울모래  0.45   0.38   0.52   0.41   0.29
서울잔디  0.31   0.44   0.36   0.47   0.55
부산모래  0.52   0.41   0.33   0.28   0.22

Python 구현:

import seaborn as sns
import pandas as pd
import numpy as np

data = pd.DataFrame({
    '1000m': [0.45, 0.31, 0.52],
    '1200m': [0.38, 0.44, 0.41],
    '1400m': [0.52, 0.36, 0.33],
    '1600m': [0.41, 0.47, 0.28],
    '2000m': [0.29, 0.55, 0.22]
}, index=['서울모래', '서울잔디', '부산모래'])

sns.heatmap(data, annot=True, fmt='.2f', cmap='RdYlGn')
plt.title('마匹 트랙·거리 입상률 히트맵')
plt.savefig('track_distance_heatmap.png')

---

4. 출발 트레이스 (Scratch/Start Plot)

활용 데이터: 각 마匹의 출발~결승까지 위치 변화

차트 유형: 다중 꺾은선 (Multiple Line) 또는 산점도

해석:

  • 초반 내侧 이동이 많은 마匹 → 출발력 강함
  • 백 스트레치에서 추월하는 마匹 → 중경거리 강자
  • 최종 직선 급가속 → 결승력 강화형

---

5. 혈통 네트워크 (Pedigree Network)

활용 데이터: 마匹의 系図 정보

차트 유형: 네트워크 그래프

사용 도구: NetworkX / D3.js
노드: 조상 마匹 (크기 = 수상 경력)
엣지: 부모-자식 관계 (굵기 = 유전적 영향력)

시각화 시 주의사항

  1. 과적합 시각화: 복잡한 차트는 해석력을 떨어뜨린다 — 단순하게 유지
  2. 배당률 왜곡: 배당률은 시장 합의이지 확률이 아니다 — 별도 변환 필요
  3. 시계열 스냅샷: 경주 당일 데이터는 경주 전에 고정해야 의미가 있다
  4. 색상 의미 일관성: 빨강을 "좋음"으로 쓰면 계속 유지 — 혼동 방지
