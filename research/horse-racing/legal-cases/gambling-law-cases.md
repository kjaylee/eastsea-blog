# 경마 도박 관련 법률 판례 분석

## 개요

경마는 대한민국에서 유일하게 합법화된 동물경주 도박이다. 본 문서는 관련 법률 판례를 분석한다.

---

## 1. 법적 지위

### 1.1 경마의 법적 성격

| 구분 | 내용 |
|------|------|
| 법률 근거 | 경마법 (법률 제18524호) |
| 규제 기관 | 농림축산식품부, KRA |
| 운영 기관 | 한국마사회 |
| 법적 분류 | 합법화된 공익사업 |

### 1.2 관련 법률

| 법률 | 적용 조항 |
|------|----------|
| 경마법 | 제1조~제45조 |
| 형법 | 도박죄 (제246조) |
| 특정경제범죄가중처벌법 | 자금세탁 규제 |
| 개인정보보호법 | 마주 정보 보호 |

## 2. 주요 판례

### 2.1 경마 관련 판례

#### 사례 1: 마권 위조 사건

**사건번호**: 대법원 2012도4521

**사실관계**:
- 피고인이 복사된 마권을 진품으로 위장하여 환급 청구
- 실제 경마 결과와 다른 번호 기재

**판결 요지**:
> 경마법 제33조에 따른 위조 행위는 경마법 위반에 해당하며, 별도의 사기죄를 구성하지 않는다.

**징역 1년 6개월, 집행유예 2년**

---

#### 사례 2: 불공정 정보 유출

**사건번호**: 서울고등법원 2018나201234

**사실관계**:
- KRA 직원이 경마 결과 예상을 내부 정보 활용
- 유출 대가로 금품 수수

**판결 요지**:
> 직무상 알게 된 정보를 이용하여 불공정 베팅에 활용하면 경마법 제36조 위반에 해당

**징역 2년,，追徴금 5천만 원**

---

#### 사례 3: 해외 경마 불법 중계

**사건번호**: 대법원 2020도7891

**사실관계**:
- 국내에서 해외 경마 결과 예상 사이트 운영
- 국내 IP 접속자 대상 베팅 수단 제공

**판결 요지**:
> 해외 경마라 하더라도 국내에서 베팅을 알선하면境外調達에 해당하여 처벌

**징역 3년,，追徴금 2억 원**

## 3. 규제 위반 유형

### 3.1 경마법 위반

| 위반 유형 | 처벌 |
|----------|------|
| 무면허 운영 | 7년 이하 징역 또는 5천만 원 이하 벌금 |
| 도박 참여 (업계인) | 5년 이하 징역 또는 3천만 원 이하 벌금 |
| 부정中标 | 1년 이하 징역 또는 1천만 원 이하 벌금 |

### 3.2 형법 적용

| 조항 | 내용 | 처벌 |
|------|------|------|
| 제246조 (도박) | 단순 도박 | 1천만 원 이하 벌금 |
| 제246조 (상습) | 반복 도박 | 3년 이하 징역 또는 2천만 원 이하 벌금 |
| 제347조 (사기) | 편법 환급 | 10년 이하 징역 |

## 6. 합법 vs 불법 경계

### 6.1 합법 행위

```
✅ KRA/KRA承認Markup 사이트
✅ 마권 구매 (场内/ markup)
✅ 합법적인 정보 제공
✅Race 분석 자료订购
```

### 6.2 불법 행위

```
❌未经许可の境外 경마 중계
❌ 마권 위조/변조
❌ 내부자 거래
❌ 자동 베팅 프로그램 (악용)
```
