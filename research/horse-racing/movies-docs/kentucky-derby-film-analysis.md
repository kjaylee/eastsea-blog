# 케赔키 더비 영화 분석: Kentucky Derby Films

## 개요

케赔키 더비는 "the Run for the Roses"로 불리며, 경마계에서 가장 권위 있는 세 번째 날이다. 이에 영감을 받은 여러 영화들이 제작되었다.

## 주요 영화 목록

### 1. Secretariat (2000)

| 항목 | 내용 |
|------|------|
| 감독 | Randall Wallace |
| 개봉 | 2000 |
| 러닝타임 | 123분 |
| 평점 | IMDb 8.0 |

#### 경마사적 사실성
- **정확도**: ★★★★☆ (대부분 사실 기반)
- **주요 왜곡**: 일부 시간 순서 변경
- **기억할 점**: 1973년 트리플 크라운 달성 과정

### 2. Seabiscuit (2003)

| 항목 | 내용 |
|------|------|
| 감독 | Gary Ross |
| 개봉 | 2003 |
| 러닝타임 | 141분 |
| 평점 | IMDb 7.3 |

#### 시비스컷 vs 시에라시
- 1938년 메트ropolitan 핸디캡 대결이 영화의 클라이맥스
- 실제 경주: 시비스컷이 1위, 시에라시 2위

### 3. Dreamer (2005)

## 공통 패턴 분석

### 영화가 보여주는 경마의 본질
1. **인내와、根性**: 명마는 하루아침에 만들어지지 않는다
2. **관계의 소중함**: 조련사-말 신뢰 관계
3. **부정적 환경에서의 극복**: 불리한 조건을 역전하는 서사
