## 1. 법률 체계 개요


## 2. 한국마사회 (Korea Racing Authority, KRA)

### 2.1 조직 개요

| 항목 | 내용 |
|------|------|
| **설립** | 1999년 10월 (민법상 재단법인) |
| **소재지** | 경기 과천시 中央的人了町 |
| **자산** | 약 1조 2천억 원 |
| **직원 수** | 약 2,500명 |

**주요 시설:**
- 서울경마공원 (과천)
- 부산경마공원 (기장)
- 제주경마공원 (제주시)

---

### 2.2 주요 임무

1. **경마比賽 主催**
2. **馬券发售** (마닐라 系统)
3. **馬事 Scholarship** 운영
4. **마필 건강 관리** 및 교육
5. **경마 관련 R&D**

---

### 2.3 수익 구조

| 구분 | 비율 | 용도 |
|------|------|------|
| 마권 판매 수입 | 100% | 총 수입 |
| 당첨자 환급 | 75~80% | 배팅 환급 |
| 운영비 | 15~20% | 시설·인건비 |
| 국고 납부 | 5% | 국가 재원 |

## 3. 마권発売 규제

### 3.1発売 방식

**인터넷发售 (Nexen):**
- 2019년 이후 확대
- 한국마사회 공식 APP 통해 가입 회원 대상
- 본인 명의 계좌 필수

**현장発売:**
- 경마공원 内 발매소
- 부착 판매 (과거) → **2014년 폐지**

---

### 3.2発売 시간 규제

| 구분 | 내용 |
|------|------|
| **인터넷 발매** | 경주 5분 전까지 |
| **현장 발매** | 경주 3분 전까지 |
| **마감 후発売 금지** | 절대적 제한 |

---

### 3.3 구매 자격

- **만 19세 이상** (법적 성인)
- **본인 명의** 계좌만 사용
- 직원·관계자 구매 **금지**

---

### 3.4 1인 1일 구매 한도

| 구분 | 한도 |
|------|------|
| **연속 7일 구매 한도** | 500만 원 |
| **1회 구매 한도** | 100만 원 |
| **월간 구매 한도** | 1,000만 원 |

**현실:**
- 온라인 시스템에서 자동 enforcement
- 超過 시 거래 자동 차단

## 4. 배팅 시스템 규제

### 4.1 허용 배팅 유형

| 유형 | 상태 |
|------|------|
| 단승 | ✅ 허용 |
| 연승 | ✅ 허용 |
| 복승 | ✅ 허용 |
| 쌍단승 | ✅ 허용 |
| 쌍연승 | ✅ 허용 |

### 4.2 배당률 결정

- **총 발매액의 75~80%**를 당첨자에게 환급
- 배당률 = (총 환급금) ÷ (해당 조합 구매액)
- 배당률 **사전 공개 금지** → 경주 시작 후 공개

---

### 4.3 환급 방식

| 구분 | 방식 |
|------|------|
| **현장 구매** | 경마공원 내 환급소 |
| **인터넷 구매** | 즉시 계좌 이체 |
| **환급 기한** | 경주 종료 후 60일 이내 |

---

### 4.4 환급율 (환원율)

```
환급율 = 당첨금 ÷ 총 구매액 × 100%

한국 현실:
- 이론치: 75~80%
- 실제 평균: 78~82%
```

**목표 환급율:**
- 경매법 시행령 별표 1에서 **75% 이상** 규정
- 超過 시 시정 조치 대상

## 5. 동물 복지 관련 규정

### 5.1 마복지법 주요 내용

**마의 관리 기준:**

| 항목 | 기준 |
|------|------|
| **운동 시간** | 매일 최소 1시간 이상 |
| **사료** | 영양 균형 사육 |
| **수의사 상시 대기** | 경기장内 의무的配置 |
| **검사 결과 공개** | 연 2회 이상 공개 |

### 5.2 반ELD 금지 약물

**검출 시 의무:**

| 약물 유형 | 제재 |
|----------|------|
| **향정신성 약물** | 영구 출전 금지 |
| **스테로이드** | 6개월 출전 정지 |
| **진통제** | 경고·과태료 |

**검사 기관:**
- 한국마사회 위생검사소
- 대한마사협회 지정 실험실

---

### 5.3 경주 중 부상·사망

**현장 조치:**
- 부상 시 즉시獣医 참여 → 필요시 경주 중단
- 사망 시 부검 의무 (사인 조사)
-死亡 マの血液·조직 保存 (2년)

**보고 의무:**
- 경주 중 마 사망 → 24시간 내 농림부 신고

---

### 5.4 Retirement 후 관리

**의무:**
- 경마 은퇴 마필 목록 관리
- 종마·번식마 전환 지원
- 은퇴 후 사관학교·승마施設 引受斡旋

**현실 과제:**
- 일정比例 은퇴 마의 安楽死 논쟁
- 복지 정책 강화 요구

## 6. 위반 시 제재

### 6.1 마권 구매 관련

| 위반 유형 | 제재 |
|----------|------|
| 미성년자 구매 | 구매 무효·환급 거부 |
| 타인 계좌 사용 | 본인 계좌 차단 + 법적 고발 |
|1인 1일 한도 초과 | 초과분 무효 처리 |

---

### 6.2 경주 자체 관련

| 위반 유형 | 제재 |
|----------|------|
| **탄혈 (doping)** | 조교사·기수 영구 제재, 상금 환수 |
| **부정 경주** | 관련자 전원 영구 제재, 법적 조치 |
| **기수 체중 미달** | 출전 정지 + 과태료 |
| **조교사 無資格者** | 즉시 출전 정지 |

---

### 6.3 직원 관련

| 규정 | 내용 |
|------|------|
| **兼業 금지** | 경마 관계자, 다른 도박업종 겸업 금지 |
| **미 Filing** | 公務 전년도 5일 이내 Filing 의무 |
| **퇴직 후 기강** | 퇴직 후 2년간 관련업종 취업 제한 |
