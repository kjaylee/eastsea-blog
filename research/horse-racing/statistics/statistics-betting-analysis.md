## 2. 배당률 분석

### 2.1 배당률 결정 요인

```
배당률 = (총 팔판액 × 0.75~0.80) ÷ 해당 조합 팔판액
```

**영향 요인:**
1. 마필별 팔판액 비율
2. 과거 성적 데이터
3. 기수·조교사 평가
4. 무게(hweight) 포함 여부
5. 코스·거리适应性

---

### 2.2 배당률 분포 패턴

**단승 기준 (18두 출전):**

| 배당률 구간 | 발생 확률 | 조합 수 |
|------------|----------|--------|
| 1.0~1.5 | 15% | 2~3마 |
| 1.5~3.0 | 35% | 5~7마 |
| 3.0~5.0 | 25% | 4~5마 |
| 5.0~10.0 | 15% | 2~3마 |
| 10.0+ | 10% | 1~2마 |

**평균 배당률:** 약 3.5~5.0배

---

### 2.3热门·冷门 배당률 분석

**热门马 (Favorites):**
- 배당률 1.5배 이하
- 실제 승률: 약 35~40%
- 역배당률 ( Overlay) 발생 시 가치 있음

**冷門马 (Long Shots):**
- 배당률 10배 이상
- 실제 승률: 약 5~8%
- 조합 구성 시 보조적 활용

---

## 3.|win概率 예측 모델

### 3.1 기본 모델: 회귀 분석

**입력 변수:**
- 과거 경주 성적 ( 최근 5경주 )
- 거리별 전적
- 기수·조교사 복합지수
- 체중 변화 추이
-|pre-draw]] 위치 분석

**기본|win 확률 공식:**

```
P(win) = α + β₁(과거 승률) + β₂(거리 적합도) + β₃(기수 평가) + β₄(조교사 평가) + ε
```

---

### 3.2 Ratings 기반 예측

**BOWER Score:**
- Timeform Rating 기반
- 0~140 구간 (평균 100)
- 120+ = 세계적 수준

**BRIS Ratings:**
- 미국 표준화 지수
- 6개 하위 지수 결합

**예측 정확도:**
- 상위 3개 마 선별: 정답률 약 55~65%
- 상위 1개 마 정확도: 약 30~40%

---

### 3.3 시계열 분석

**최근 Form 곡선:**
```
Form = (최근 3경주 가중 평균) × 거리 보정 × 체력 지수
```

**가중치:**
- 최근 1경주: 50%
- 최근 2경주: 30%
- 최근 3경주: 20%

## 5. 해외 주요 배팅 시스템

### 5.1 Pool Betting (Paris Mutuel)

프랑스·일본·홍콩 등采用:

```
배당률 = (총 Pool - 세금) ÷ 단일 조합 Pool
```

**장점:**
- 배팅이 배당률에 영향
- 공정하고 투명한 배당률 형성

**단점:**
- 배팅 제한 없음 → 大口投注 가능

---

### 5.2 Fixed Odds (英国·호주)

**특징:**
- 배팅 즉시 배당률 확정
- Market 변동 없음
- 마권 거래소 (Betfair) 방식

**적용 국가:**
- 영국 (온라인·소매)
- 호주 (일부赛事)
- 싱가포르 (폐지前)

---

### 5.3 Spread Betting

**특징:**
- 여러 배팅 조합의 점수 합산
- 정확한 순서 불필요
- 손실 제한 없음 (무제한)

**예시:**
- Forecast Spread: 10~14점
- 결과가 12점 = 적중

---

## 6. 배팅 전략 통계

### 6.1 기본 전략

**戦略 1: 단승 집중**
- 적중률 상대적 높음
- EV ≈ -20% → 장기 불리

**전략 2:热门马 묶기**
- 배당률 낮지만 적중률 높음
- 복승 조합으로 적중률 확보

**戦略 3: 쌍연승狙击**
- 배당률 높음 (10배~100배)
- 적중률 극히 낮음 (<1%)
- 소액 다점 조합 전략

---

### 6.2 Kelly Criterion (켈리 기준)

**공식:**
```
f* = (bp - q) / b
where:
  f* = 투자 비율
  b = 배당률 - 1
  p =|win 확률
  q = 패배 확률 = 1 - p
```

**적용 예시:**
- 배당률 3.0배, 예상|win 확률 40%
- f* = (2 × 0.4 - 0.6) / 2 = 0.1
- **투자 비율: 10%**

**주의:** Kelly 기준은 변동성 높아 실제 투자에는 1/4 Kelly 권장

---

### 6.3 Dutching (달칭)

여러 선택지에 배팅하여 동일 수익 확보:

```
배팅액 = 목표 수익 ÷ (배당률 - 1)
```

**예시:**
- A马: 배당률 3.0
- B马: 배당률 5.0
- 목표 수익: 100단위

```
A 배팅 = 100 / (3-1) = 50
B 배팅 = 100 / (5-1) = 25
총 배팅 = 75
```

→ A 또는 B 승리 시 항상 100수령

---

### 6.4 베스트 디스크바트 (Best Discount Betting)

香港·싱가포르·영국 마권 시장을 활용:

1. 동일 마필의各国 배당률 비교
2. **Arbitrage (안전 수익) 기희 탐색**
3. 배당률 차이 활용

**예시:**
- Hong Kong: 배당률 2.1
- UK Betfair: 배당률 2.3
- → UK에서 매수 → HK 보다 높은 환금

## 7. 기대값 (EV) 분석표

| 배팅 유형 | 평균 배당률 | 적중률 | 환원율 | EV (100원 기준) |
|----------|-----------|--------|--------|----------------|
| 단승 | 5.0배 | 20% | 100% | -2원 |
| 연승 | 20배 | 5% | 100% | -10원 |
| 복승 | 10배 | 10% | 100% | -5원 |
| 3련単 | 100배 | 1% | 100% | -20원 |

**결론:** 모든 배팅의 장기 EV는 음수 → 경마는 장기 투자 대상이 아님

## 8. 통계 활용 실무 Tip

1. **상위 3두 선별 후 복승 조합** — 적중률 30%+, EV 개선 가능
2. **과거 3경주 Form 우선** — 최신 데이터의 예측력最高
3. **기수 교체 시 주의** — 10% 이상 적중률 변동 가능
4. **배당률 급변 패턴** — 大口投注 직후反向 배팅 위험
5. **한국 기준:** 연평균 환원율 78~82% → 장기 필수 손실
