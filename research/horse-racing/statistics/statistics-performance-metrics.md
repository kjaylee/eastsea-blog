## 1. 핵심 성능 지표 (KPI)

### 1.1 마필 성능 지표

| 지표 | 설명 | 산출식 |
|------|------|--------|
| **승률 (Win Rate)** | 전체 출전 중 1위 비율 | 1위 횟수 ÷ 출전 횟수 |
| **연속 실패 방지율 (Place Rate)** | 상위 3위 이내 비율 | 상위3위 횟수 ÷ 출전 |
| **평균 순위 (Avg Position)** | 최종 순위 평균 | Σ순위 ÷ 출전 횟수 |
| **평균 차이 (Avg Margin)** | 1위와의 평균 시간 차 | Σ(1위 시간 - 마필 시간) ÷ 수 |
| **수익률 (ROI)** | 배팅 수익률 | (총 환급 - 총 배팅) ÷ 총 배팅 |

---

| **말 교체 후 성과** | 특정 기수-마필 조합 성과 |

|------|------|

## 2. 마필 평가 시스템

### 2.1 Timeform Rating (타이름폼 레이팅)

**개념:**
- 0~140 구간 (평균 100)
- 영국·아일랜드 경마 표준화 평가

**산출 방법:**
1. 실제|time vs 추론|ideal time 비교
2. 코스·거리·상태 보정
3. 무게(hweight) 조정
4. 최종 수치 산출

**기준 등급:**
| Rating | 등급 |
|--------|------|
| 130+ | Exceptional (exceptional) |
| 120~129 | Very good |
| 110~119 | Above average |
| 100~109 | Average |
| <100 | Below average |

**역사 최고:**
- Frankel: 147 (2012)
- Sea-Bird: 141 (1965)
- Man o' War: 140 (1920)

---

### 2.2 Beyer Speed Figure (베이어 스피드 피겨)

**개념:**
- 미국 경마 표준 속도 지수
- 각 경주별 "Beyer" 수치 부여

**산출 방법:**
1. 完成|time → 同조건、同거리 과거 最速 비교
2. 트랙 보정係数 적용
3. 최종 수치 (최고 130+)

**활용:**
- 동일 마의 폼 추이 추적
- 마필 간 성능 비교

---

### 2.3 Equibase Performance Rating

**산출:**
```
Performance Rating = 100 + 10 × (표준편차 조정치)
```

**특징:**
- 미국 全 경마장 통합 표준
- 0~130 구간

---

### 2.4 Japan Racing Authority Rating (日本)

| 등급 | Rating 범위 | 설명 |
|------|-----------|------|
| A | 90+ | 상위 |
| B | 80~89 | 중상 |
| C | 70~79 | 중위 |
| D | 60~69 | 하위 |

**산출:** 최근 2년간 경주 成積 기반 加算 平均

## 3. Ratings 산출 방법론

### 3.1 기본 원리

```
Rating_n = Rating_{n-1} + ε + η + δ
where:
  ε =/time 오차 보정치
  η = 트랙 보정치
  δ = 무게 보정치
```

**동일 마 연속 출전:**
- 시간 경과에 따른 폼 저하係数 적용
- 平均: 매달 -0.5 rating point

---

### 3.2 트랙 보정 (Track Variant)

**문제:** 각 트랙은 고유한 속도 특성 보유

**보정係数 예시:**
| 트랙 | 보정치 (초) |
|------|-----------|
| Churchill Downs (더트) | +0.2 |
| Belmont Park (더트) | -0.3 |
| Arlington (더트) | +0.1 |

**산출:** 동일 트랙 과거 데이터 5년 평균 기반

---

### 3.3 프리드먼 모델 (Friedman Model)

**Timeform 내부 사용 모델:**
1. 直近 race 成積 → 초기 rating 부여
2. 경주 결과 → rating 업데이트
3. 경과 시간 → rating 自然減衰
4. 신규 데이터 → Bayesian 更新

```
R_new = α × R_observed + (1-α) × R_prior
α = 가중치 (일반적 0.3~0.5)
```

## 4. 데이터 기반 예측 모델

### 4.1 다중 회귀 모델

**종속 변수:**|win 여부 (0/1)

**독립 변수:**
- 최근 5경주 平均 Rating
- 거리별 전적 比
- 기수 Rating
- 조교사 Rating
- 체중 변화
- 出走回数 (경험)

**모델 형태:**
```python
P(win) = 1 / (1 + exp(-z))
z = β0 + β1·Rating + β2·Distance + β3·Jockey + β4·Trainer
```

---

### 4.2 랜덤 포레스트

**특징:**
- 비선형 관계 포착
- 변수 중요도 파악 가능

**변수 중요도 (일반적):**
1. 直近 Rating (25~30%)
2. 기수 Rating (15~20%)
3. 거리 적합도 (10~15%)
4. 폼 추이 (10~15%)
5. 조교사 Rating (5~10%)

---

### 4.3 신경망 모델 (Neural Network)

**입력:**
- 마필 16마 × 10개 feature = 160 dimension
- 마필 16마 × 10개 feature × 3개 과거 race = 480 dimension

**은익층:**
- 2~3층 MLP
- ReLU 활성화

**출력:**
- 마필별|win 확률 (Softmax)

---

### 4.4 실전 예측 정확도

| 방법 | 정확도 (Top-3) |
|------|--------------|
| 랜덤 | 16.7% |
| 단순 Rating | 30~35% |
| 다중 회귀 | 35~40% |
| 랜덤 포레스트 | 40~45% |
| 신경망 | 42~48% |
| 앙상블 (다수 모델) | 45~52% |

**한계:**
- 데이터 부족 (마필당年均 3~5경주)
- 시장 정보 미포함 (배팅 비율)

## 5. 실전 데이터 분석 사례

### 5.1 한국 경마 데이터 구조

```
Raw Data:
- 마필ID, 경주일, 경주번호, 코스, 거리, 마체중
- 최종순위,|time,通過타이밍
- 기수ID, 조교사ID, 마필나이, 성별
```

**분석 기간:** 2019~2023년

**표본:**
- 서울경마공원 기준
- 연평균 30,000 경주 records

---

### 5.2 거리별 승률 분포

| 거리 | 平均 1위 배당률 | 상위 3두 합산 승률 |
|------|--------------|-----------------|
| 1000m (스프린트) | 4.5배 | 55% |
| 1200m (스프린트) | 5.2배 | 52% |
| 1700m (미들) | 6.8배 | 48% |
| 2000m (미들) | 8.5배 | 45% |
| 2300m+ (롱) | 10.2배 | 42% |

**해석:**
- 거리 증가 → 승자 예측 어려움
- 롱런 → 폼·체력 管理更重要

---

### 5.3 마체중과 성능 상관관계

**데이터:**
- 출발 전 마체중 vs 최종 순위 상관계수: r = -0.12
- **통계적으로 유의하나 영향 미미**

**주의:**
- 급격한 체중 감소 (>5kg) → 부정적 신호
- 체중 증가는 물 Intake 可以

### 5.4 기수 교체 효과 분석

| 교체 유형 | 영향도 | 비고 |
|----------|--------|------|
|초高水平→ 低水平 | -15~20% 승률 | 명확한 부정적 효과 |
| 低水平→ 高水平 | +10~15% 승률 | 명확한 긍정적 효과 |
| 初回 탑재 | ±5% | 경향 미미 |

**주요高水平 기수:**
- 한국: 池祐식, 김동一, 梁在雄

## 6. Time 분석과 폼 곡선

### 6.1 Speed Figure 추이 분석

**Ideal 폼 곡선 (3세 마):**
```
Race 1: Rating 85 (初出走·未경험)
Race 2: Rating 88 (+3)  ← 適応
Race 3: Rating 91 (+3)  ← 成長
Race 4: Rating 93 (+2)  ← ピーク逼近
Race 5: Rating 94 (+1)  ← ピーク
Race 6: Rating 92 (-2)  ← 疲労·成長停
```

---

### 6.2 폼 저하 패턴

**인식할 위험 신호:**
1. **연속 하락 (3연속 하락)** → 管理問題 가능성
2. **급격한 상승 후 급락** → 病理 가능성
3. **장기간 휴식 (>90일)** → 폼 회복 不透明

**안전 신호:**
1. **점진적 상승 후_plateau** → 管理良好
2. **하락 후 2연속 상승** → 反弹 가능성

---

### 6.3 Time-Based 분석

**Target Time vs Actual Time:**

```
ΔT = Actual Time - Target Time (같은 조건)
ΔT < 0: 优于 期待
ΔT = 0: 期待 比 
ΔT > 0: 低于 期待
```

**적용:**
- ΔT < -0.5초 → 해당 마 Rating +3 이상
- ΔT > +0.5초 → 해당 마 Rating -3 이상

---

### 6.4 모델 앙상블 전략

**방법론:**
1. **단일 모델 × 5개** (다양한 파라미터)
2. **타입별 모델 3개** (Tree, Linear, Neural)
3. **가중 평균** → 최종 예측

**예시:**

| 모델 | Top-3 정확도 | 가중치 |
|------|------------|--------|
| RandomForest-v1 | 44.2% | 0.3 |
| XGBoost | 43.8% | 0.3 |
| LightGBM | 45.1% | 0.4 |

**앙상블 결과:** 46.3% (개별 최고 대비 +1.2%p)

## 7. 실전 적용 체크리스트

### 마필 선택 7단계

1. **Distance Adaptation** — 과거 同거리 성적 확인
2. **Recent Form** — 최근 3경주 추이 확인
3. **Weight Trend** — 마체중 추이 확인
4. **Jockey Change** — 기수 교체 영향 분석
5. **Trainer Pattern** — 조교사 출전 패턴 확인
6. **Race Class** — 클래스 변동 확인
7. **Draw Position** — 번호배정 분석

### 실전 고려사항

- **상위 3두 선별:** 적중률 45~55%, EV 개선
- **복승 조합 활용:** 적중률 + 배당률 균형
- **한국 기준:** 연환산 EV ≈ -20~25% → 소액娱乐目的
