## 5. 초안 인사이트 (Pre-research — 조건부 사실)

> ⚠️ 아래는 기존 자료 기반 조건부 분석. 실제 리서치 결과로 검증 필요.

### 초안 인사이트 1: KRA 데이터 독점 구조

```
KRA의 데이터 가치 체인:
[경주 진행] → [판정·기록] → [배당 결정] → [결과 공개]
     ↑              ↑             ↑            ↓
  내부 only    내부 only     실시간 내부  翌日 경과만
```

**조건부 결론:** KRA는 경주 직전~경주 중 실시간 데이터를 **완전히 독점**한다.
이는 내부자 거래 위험을 최소화하기 위한 구조로 보이지만, 시장 효율성 측면에서 손해.

### 초안 인사이트 2: 배당률 데이터의 이중 가치

|数据类型 | 공개 시점 | 경제적 가치 |
|---------|----------|------------|
| 최종 배당률 | 경주 후 | 낮음 (결과 알려짐) |
|のオWin 배당 | 경주 직전 | **매우 높음** (베팅 전략的核心) |
| 실시간 판정 | 경주 중 | **극도로 높음** (in-running 베팅) |
| 마체 biometric | 경주 후 | 낮음 (복수의율 문제) |

**조건부 결론:** 한국은 in-running 베팅이 금지되어 있기에 실시간 데이터 공개의 경제적 가치가 제한적. 그러나 배당률 공개 타이밍 하나가 수조 원 시장의 효율성을 좌우.

### 초안 인사이트 3: 외부 데이터 생태계의 부재

v14의 커뮤니티·인력 영역 Gap과 연결:
- 한국에는 Equibase/HorseWatch 같은 독립 데이터 플랫폼이 **존재하지 않음**
- KRA 공식 앱이 사실상 유일한 데이터 제공자
- 이는 독점 데이터 구조의 직접적 결과
