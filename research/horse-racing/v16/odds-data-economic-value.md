## 2. 환급 구조 — 배당률의 경제적 설계

### 기본 공식
```
총 베팅 금액 = 1억원
환급금 = 73% (단승·연승은 80%)
정부·마사회 취급 수수료 = 27% (단승·연승 20%)
```

| 구분 | 환급률 | 비고 |
|------|--------|------|
| 단승식 | **80%** | 1头馬 적중 |
| 연승식 | **80%** | 연속 马 적중 |
| 복승식 | **73%** | 2头馬 순서 무관 |
| 쌍승식 | 73% | 2头馬 순서 맞추기 |
| 삼복승식 | 73% | 3마터 적중 |
| 삼쌈승식 | 73% | 3마터 순서 맞추기 |

### 의미
- 마사회는 **27%의 확정 수익**을 먼저 차감하고 배당률을 형성
- 배당률은 베팅 금액 편향에 따라 실시간 변동
- **배당률 데이터 = 수조 원 베팅 시장의 방향을 결정하는 핵심 정보**

## 3. 배당률 데이터의 실시간 업데이트 주기

### 공식 문서에 따른 배당률 갱신
- **30초마다** 경마공원 배당률 게시대 + KRA Plaza 배당률 모니터 갱신
- 발매 마감 시 최종 예상 배당률 확정
- 전자시스템에 의해 **자동 갱신** — 인적 개입 없음

```
[마권발매 개시] ──30초 간격 갱신──→ [마권발매 마감] → [확정 배당률]
         실시간 배당률 공개              최종예상배당률
```

### 배당률 데이터 흐름
1. 고객 베팅 금액 입력
2. 시스템이 전체 금액 대비 배분률 계산
3. 30초마다 배당률 갱신 → 게시대/모니터 반영
4. **마권 발매 마감 전까지** 최종 예상 배당률이 실시간 공개

## 4. 배당률 데이터의 계층별 경제적 가치

### Layer 1: 최종 확정 배당률 (경주 후)
- **가치:** 낮음
- **이유:** 결과가 이미 확정됨, 사후 분석용
- **공개 채널:** KRA 공식 사이트, 경마공원 게시판, 공공데이터포털 API

### Layer 2: 최종 예상 배당률 (경주 직전 ~ 마감 시점)
- **가치: 높음 (전략적 핵심)**
- **이유:** 베팅 최종 결정 직전의 시장 균형 상태
- **공개 채널:** KRA Plaza 배당률 모니터 (실시간 30초 갱신)

### Layer 3: 실시간 동적 배당률 (마권 발매 중)
- **가치: 가장 높음 (in-running 전)**
- **이유:** 다른 베터들의 금액 흐름을 실시간으로 추적 가능
- **한국 특수 상황:** in-running 베팅 **불허** → 경주 중 실시간 베팅 불가
- **결과:** 경주 중 배당률 데이터의 투명한 실시간 공개가 가능 (시장 효율성 향상)

### Layer 4: 과거 배당률 시계열 데이터
- **가치: 높음 (AI/예측 모델 학습용)**
- **현재 상태:** 공공데이터포털에 경주일·경주번호·승식별 매출액·확정배당률 제공
- **한계:** 실시간 중간 배당률(30초 갱신분) 시계열은 **공개되지 않음**

## 5. 핵심 인사이트

### 인사이트 1: 배당률 갱신 주기(30초)의 전략적 의미
- 30초마다 갱신 = 하루 약 120~200회 배당률 변동 기회 (경마장 운영 시간 기준)
- 베팅 금액이 급격히 몰리는 시점에서 **배당률突变(급변)** 가 발생
- 이突变 패턴 자체가 **고价值的 시장 심리 데이터**

### 인사이트 2: 한국은 In-Running 금지 → 실시간 공개의 부담이 적음
- 실시간 배당률 공개가 있으면 내부자 거래 위험 ↑
- 한국은 경주 중 베팅 금지 → 배당률 공개 시점 = 경주 시작 시 한정
- **한국의 30초 배당률 공개는 세계적으로 봐도 비교적 투명한 편**

### 인사이트 3: 공공데이터포털 API의 존재
- **경마 매출액 및 확정배당율** API 제공 (data.go.kr)
- 경주년/월/일/번호별 요청 가능
- **단순 매출·확정배당 데이터만 제공** — 실시간 중간 배당률은 미제공
- 향후 중개BETting 플랫폼에서 실시간 odds API 수요 ↑ 예상

### 인사이트 4: 배당률 데이터의 부채(?) 효과
- 배당률이 "과대 반응"하는 경향: 소위 "热门马"에 금액 급증
- 73% 환급 구조는 마사회 수익 보장, 베팅 시장 전체의 기대값은 음수
- **배당률 데이터로 초과 수익을 기대하기 어려운 구조** (시장 효율성 높은 편)
