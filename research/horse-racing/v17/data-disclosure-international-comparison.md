## 2. 데이터 공개 수준 비교

### 2.1 배당률 (Odds) 공개

| 항목 | 한국 | 일본 | 홍콩 | 영국 |
|------|------|------|------|------|
| **경주 전 최종 배당률** | 경주 직후 (약 3분 후) | 경주 20분 전 **공식 발표** | 경주 15분 전 **실시간 갱신** | 경개시 2분 전 **표준화** |
| **복수의율(Par)* 공식** | ❌ 미공개 | ✅ Race Index에 공개 | ✅ HKJC API 직접 제공 | ✅ HRI/RPA 공개 |
| **내부 독점 여부** | ✅ 독점 | ❌ 공개 | ❌ 공개 | ❌ 공개 |

> **Par (Parimutuel Rate):** 특정 환급률에서 단승 1점당 환급액. 베팅 배당률의 기준선.

**한국 특수성:** 한국은 복수의율을 공식 공개하지 않아 **배당률 정상성 검증이 외부에서 불가능**. 이는 마권 부정교환(broiler room) 감시 부족과 직결.

## 3. 데이터 공개 속도 비교

```
[경주 종료 시점 기준 — "+"는 종료 후 경과 시간]

한국:  +3분   (배당률)  / +翌日 (세부 데이터)
일본:  -20분  (배당률)  / +30분 (결과)  / +1일 (Race Index)
홍콩:  -15분  (배당률)  / +1분  (결과)  / 실시간 (API)
영국:  -2분   (배당률)  / +5분  (결과)  / 실시간 (RaceTech)
```

**핵심 관찰:** 영국과 홍콩은 경주 **이전**에 배당률을 공개 — 베팅 시장이 사전에 효율적 가격 발견 가능. 한국은 事後 공개로 **시장 시간 차이**가 구조적으로 존재.

## 7. 핵심 시사점 (한국에 대한)

### 7.1 시사점 1: 배당률 공개 시간의 시장 효율성 효과

- **문제:** 경주 후 배당률 공개는 사전 효율적 가격 발견을 불가능하게 함
- **비교:** 영국·홍콩은 事後 배당률 공개 → 베팅 시장이 수렴
- **정책 옵션:** 배당률 공개 시점 조기화 (경주 10분 전) 또는 복수의율 공개

### 7.2 시사점 2: API 부재가 생태계 innovation을 저해

- **문제:** KRA는 외부 데이터 플랫폼이 성장할 구조적 토대를 제공하지 않음
- **비교:** 홍콩 HKJC Data API → Racing Info Services 등 다수 3자 플랫폼 성장
- **정책 옵션:** 제한적 공개 API 또는 데이터 공유 파트너십

### 7.3 시사점 3: Biometric 데이터 — 글로벌 트렌드への対応

- **문제:** 한국은 마체 데이터 수집·공개 모두 미비
- **비교:** 영국은 주요 마필 대상 (동물 복지 + 경기력 데이터)
- **정책 옵션:** 마체 센서 도입 검토, 生物統計 数据库 구축

### 7.4 시사점 4: 내부자 거래 감시 강화 필요성

- **문제:** 배당률 사후 공개 구조는 내부자 거래 검증 불가능
- **비교:** 영국 Gambling Commission, 홍콩 외부 감사 체계
- **정책 옵션:** 독립 감시 위원회 설치, 배당률 공개 전 거래 기록 감사
