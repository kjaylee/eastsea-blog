| 항목 | 내용 |
|------|------|
| 주소 | 경기도 과천시 경마대로 107 |
| 영업일 | 화~일 (월休) |
| 관중 수용력 | 약 50,000명 |
| 트랙 종류 | 모래·잔디겸용 (더IRT) |
| 특징 | 한국 최대 규모, 국가경마보험公営 |

**주요 시설:**
- 새벽경마 (05:30~)
- 승식장 1~12번
- Members-only 구역 (별도 입구)
- 중앙 Hodugyeok 투시 시스템

**관람 포인트:**
- 스탠드 3층: 전 구간 조망 가능
- 잔디 트랙 쪽 2층: 최종 직선 최고의 시야
- 회원 투시실: 전문 배당률 분석 가능

## 방문 준비물

- **신분증**: 만 19세 이상 (경마장 출입 가능 연령)
- **현금 또는 카드**: 배팅用 (일부 장비만 카드 가능)
- **舒适한 옷**: 실내 금연,冷房设施完备
- **지퍼백**: 작은 가방만 허가
- **스포츠 행사**: 필수는 아니나 캠페인 가능

---

## 경마장 내부 동선

```
입구 → 신분확인 → 출석게시판 확인 → 배팅구간 → 관중석 → 출주현장(선택)
```

1. **입구**: 신분증 제시, 금연·반입 금지물 확인
2. **출석게시판**: 오늘 경주 일정, 마匹名单, 배당률 확인
3. **배팅구간**: 자동 배팅기 또는 창구
4. **관중석**: 1~3층 (계층별 시야 차이)
5. **출주현장**: 트랙 جانب 특별관람석 (별도 티켓)

## 배팅 방법 초보자 가이드

### 기본 승식

| 승식 | 설명 | 배당률 특징 |
|------|------|-----------|
| 단승 | 1着馬 예측 | 배당률 낮음, 당첨 확률 높음 |
| 복승 | 2着 이내 예측 | 단승보다 배당률 높음 |
| 쌍승 | 특정 마 1着 + 다른 마 2着 | 정확한 예측 필요 |

### 중급 승식

| 승식 | 설명 |
|------|------|
| 삼복승 | 3개 마匹 예측 (순서 무관) |
| 삼쌍승 | 3개 마匹 예측 (순서 정확) |
| 오리엔탈 | 1着~3着 순서 예측 (3마匹) |
| 오리엔탈 三쌍승 | 동일 (순서 정확) |

### 초급자가 피해야 할 승식
- **완승**: 순서 정확히 1→2→3 예측 (난이도 극히 높음)
- **다복승**: 4마匹 이상 예측 (배당률 높지만 당첨률 극히 낮음)

## 경마장 이용 에티켓

- 경주 진행 중 클립보드·휴대폰으로 소음 금지
- 배팅은 경주 시작 3분 전까지 완료
- 투시실 출입은 규정 복장 준수 필수
- 경주 중 관중석からの声援은 자제
- 실내 금연 (야외 흡연 구역是利用)
