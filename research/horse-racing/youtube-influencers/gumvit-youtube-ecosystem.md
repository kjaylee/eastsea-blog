# 검빛 유튜브 분석 생태계 메모

## 개요

- 이 문서는 `communities/gumvit.md`를 바탕으로 정리한 국내 경마 유튜브 연계 메모다.
- 공식 방송이 아닌 `커뮤니티 기반 해설/예상 채널`을 분리해 기록하는 목적이 있다.
- 실시간 구독자 수나 최신 운영 상태는 이번 실행에서 외부 확인하지 못했다.

## 1. 채널 성격

| 항목 | 내용 |
|------|------|
| 운영 축 | 검빛경마 커뮤니티 기반 |
| 핵심 포맷 | 사전 예상, 경기 복기, 새벽조교 정보 해설 |
| 연결 자산 | 웹 커뮤니티, 예상지, 유튜브 방송 |
| 활용도 | 팬·베터 관점의 해설 자료 |

## 2. 코퍼스에 잡히는 대표 채널

| 채널 | 포지션 |
|------|------|
| A.I경마연구소 | 데이터·AI 분석형 |
| 불독경마 | 고배당·복병 탐색형 |
| 검빛 전문위원 | 전문가별 해설형 |

## 3. KRBC와 다른 점

- KRBC가 `공식 중계/인터뷰/하이라이트` 중심이라면, 검빛 연계 채널은 `예상과 복기` 비중이 높다.
- 공식 채널은 기록성과 공신력이 강하고, 커뮤니티 채널은 해석과 의견이 풍부하다.
- 따라서 정보지용 큐레이션에서는 `사실 확인용 앵커`와 `시장 분위기 파악용 보조 자료`를 구분해야 한다.

## 4. 편집 활용 포인트

- 경주 전 여론·관심 말 파악
- 복기 서사와 팬 반응 수집
- 새벽조교·체감 정보의 키워드 탐색
- 한국 경마 미디어 소비 습관 파악

## 5. 리스크 메모

- 예상 콘텐츠는 사실 문서보다 의견 문서에 가깝다.
- 사설 사이트 유도나 과장성 표현이 섞일 수 있어 1차 출처로 쓰기 어렵다.
- 후속 확장 시에는 `공식 채널`, `커뮤니티 채널`, `전문 해설 채널`을 분리해 색인하는 편이 좋다.
