# Bond Yield Calculator - Research

## What It Does
Calculates key bond metrics: current yield, yield to maturity (YTM), total return, and annual coupon income for fixed-income investments.

## Target Users
- Individual bond investors
- Financial advisors
- Finance students
- Retirement portfolio managers

## Key Formulas

### Current Yield
`Current Yield = (Annual Coupon Payment / Current Market Price) * 100`

### Yield to Maturity (YTM) - Approximation Formula
`YTM approx = [C + (F - P) / N] / [(F + P) / 2] * 100`
Where:
- C = annual coupon payment
- F = face value (par value)
- P = current market price
- N = years to maturity

### Total Return
`Total Coupon Income = Annual Coupon * Years to Maturity`
`Capital Gain/Loss = Face Value - Purchase Price`
`Total Return = Total Coupon Income + Capital Gain/Loss`
`Total Return % = (Total Return / Purchase Price) * 100`
`Annualized Return % = Total Return % / Years`

### After-Tax Yield
`After-Tax Annual Income = Annual Coupon * (1 - Tax Rate / 100)`
`After-Tax Yield = Current Yield * (1 - Tax Rate / 100)`
