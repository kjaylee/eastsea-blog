  - 결과 경로 기록
  - 남은 확장 배치용 프롬프트 매니페스트 작성

## 자동 실행 순서
1. 글 제목과 핵심 문단에서 커버 이미지 프롬프트 추출
2. 글당 대표 이미지 한 장씩 우선 생성
3. 결과 파일 경로를 results.md 에 누적
4. 같은 프롬프트 계열로 세로 카드와 변주 이미지를 후속 배치로 확장
5. 이후 영상과 음성은 별도 채널 소음 통제 하에 후속 실행

## 이번 배치 성공 기준
- 글 일곱 개에 대해 최소 대표 이미지 일곱 장 생성 시작
- 결과 파일 경로와 프롬프트 기록
- eastsea-blog 커밋

## 제약
- 현재 도구 표면에서 즉시 무소음으로 대량 실행 가능한 것은 이미지 생성이 우선이다
- 음성 도구는 사용자 채널로 바로 전달될 수 있어 이번 배치에서는 보류
- 동영상과 음악은 현재 이 세션의 일급 도구 표면이 없어 후속 배치로 넘긴다

## 레드 팀
- 공격 일: 한 번에 너무 많이 생성하면 채널이 시끄러워질 수 있음
- 공격 이: 프롬프트 품질이 낮으면 블로그 대표 이미지로 못 쓸 수 있음
- 방어: 이번 배치는 글당 한 장 우선 생성 후 품질 확인, 이후 확장
- 합의: 위험 수용
