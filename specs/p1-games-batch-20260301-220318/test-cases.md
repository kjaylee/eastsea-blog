# Test Cases — 20260301-220318

## 공통 기능 테스트 (각 게임 공통)
- **TC-F001**: 첫 로드 시 타이틀/설명/UI가 정상 렌더링된다.
- **TC-F002**: Start/Restart 입력 후 게임 루프가 시작되고 점수 또는 진행 지표가 변화한다.
- **TC-F003**: 게임오버/라운드종료 후 재시작이 정상 동작한다.

## 입력/플랫폼
- **TC-I001**: 키보드 입력(방향키/문자/스페이스/엔터)이 즉시 반응한다.
- **TC-I002**: 터치 입력(버튼 탭)이 동일 기능으로 동작한다.
- **TC-U001**: 390x844 모바일 뷰에서 UI가 겹치지 않고 플레이 가능하다.

## 기술 요구사항
- **TC-T001**: Web Audio API(`AudioContext`) 효과음 호출이 있다.
- **TC-T002**: localStorage로 최고 기록을 저장/복원한다.
- **TC-T003**: 테마 컬러 `#0a0a1a`가 적용된다.
- **TC-T004**: 외부 JS/CSS 라이브러리 참조가 없다(단일 HTML).
- **TC-T005**: 각 `index.html` 파일 크기가 500KB 미만이다.
- **TC-T006**: 각 게임 폴더의 `manifest.webmanifest`가 유효 JSON으로 존재한다.

## 배치 통합 테스트
- **TC-B001**: `games/manifest.json` 최상단 3개가 신규 slug 순서로 prepend 된다.
- **TC-B002**: `games/manifest.json`의 `count`가 176으로 갱신된다.
- **TC-B003**: `updatedAt`가 현재 작업 시각 기준으로 갱신된다.
