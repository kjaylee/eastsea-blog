# Research — Contribution Margin Calculator

## Goal
Ship one new static business calculator that answers high-intent contribution margin questions with immediate operational outputs:
- contribution margin dollars
- contribution margin ratio
- break-even units
- target-profit units

## Product requirements distilled
The tool should let a user enter:
- selling price per unit
- variable cost per unit
- fixed costs
- optional target profit

And should return:
- contribution margin per unit in dollars
- contribution margin ratio
- break-even units
- target-profit units

The page should also include:
- clear explanation of what contribution margin means
- explicit formula block
- worked examples
- copyable result summary
- practical interpretation/status copy

## Calculation model
Core formulas:
- contribution margin dollars per unit = selling price per unit - variable cost per unit
- contribution margin ratio = contribution margin dollars per unit / selling price per unit
- break-even units = fixed costs / contribution margin dollars per unit
- target-profit units = (fixed costs + target profit) / contribution margin dollars per unit

Implementation assumptions:
- units for break-even and target-profit should round up with `Math.ceil`, because partial units do not satisfy the threshold in most planning cases
- when selling price is `0` or negative, or when variable cost per unit is greater than or equal to selling price, break-even and target-profit units are not achievable; UI should explain this instead of returning misleading numbers
- target profit is optional and defaults to `0`
