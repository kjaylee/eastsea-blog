# Research: Wedding Budget Calculator

## Why This Tool
- "wedding budget calculator" has ~40K+ monthly searches (Google Keyword Planner estimates)
- High commercial intent (couples planning weddings spend $30K+ average in US)
- No existing wedding-related tool in the catalog (0 results)
- Clear calculator format fits the site pattern
- Evergreen demand — weddings happen year-round

## Competitor Analysis (known patterns)
- The Knot, WeddingWire, Zola all have wedding budget tools
- Most are behind signups or complex apps
- Opportunity: fast, free, no-signup static HTML calculator

## Feature Scope
1. Total budget input
2. Category breakdown with recommended percentages:
   - Venue & catering (45-50%)
   - Photography/videography (10-12%)
   - Flowers & decor (8-10%)
   - Music/entertainment (5-8%)
   - Attire & beauty (5-8%)
   - Invitations/stationery (2-3%)
   - Transportation (2-3%)
   - Favors & gifts (2-3%)
   - Officiant (1%)
   - Miscellaneous/buffer (5-8%)
3. Custom adjustments per category
4. Per-guest cost calculator
5. Visual breakdown (bar/pie chart)
6. Export/print summary

## Technical Approach
- Single HTML file with inline CSS/JS (site pattern)
- No external dependencies
- Responsive design
- SEO meta tags for "wedding budget calculator" keywords
