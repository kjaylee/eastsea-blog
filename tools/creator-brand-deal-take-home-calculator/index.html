What This Page Assumes

Same-currency planning: every money field is assumed to use the same currency. USD labels are defaults, not a hard platform restriction.

Percentage deductions: manager, agency, platform, and processing percentages are each applied to the gross quote so the math stays transparent and easy to audit.

Tax reserve is optional: this is a reserve assumption on positive pre-tax net, not a tax-law engine.

Use the target solver carefully: if deductions approach 100%, the required quote becomes unavailable or economically unrealistic.
