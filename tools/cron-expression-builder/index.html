# 실행 시각
유효한 식을 입력하면 다음 5회 실행 시각이 나타납니다.

참고: 이 도구는 표준 5필드 크론(분 시 일 월 요일)을 기준으로 계산합니다. 시스템별 확장 문법(예: Quartz의 초/년, L/W/#)은 지원하지 않습니다.
