📚 CSS 단위 가이드 / CSS Units Reference

px — Pixel (절대 단위) 화면의 물리적 픽셀. 고정 크기가 필요할 때 사용.
font-size: 16px
rem — Root Em (권장) 루트 <html> 요소의 font-size 기준. 접근성에 유리.
padding: 1.5rem
em — Relative Em 현재 요소의 font-size 기준. 중첩 시 주의 필요.
margin: 0.5em
vw — Viewport Width % 뷰포트 너비의 1%. 반응형 타이포그래피에 활용.
font-size: 5vw
vh — Viewport Height % 뷰포트 높이의 1%. 전체 화면 레이아웃에 사용.
height: 100vh
% — Percentage 부모 요소 기준 비율. width, height, padding 등에 활용.
width: 50%
pt — Point (인쇄 단위) 1pt = 1.333px. 인쇄 스타일시트에 주로 사용.
font-size: 12pt
cm / mm — 물리적 단위 실제 센티미터/밀리미터. 인쇄물 레이아웃에 사용.
margin: 2cm
