How It Works

This tool computes cryptographic hash digests of your file using the Web Crypto API (and a JavaScript MD5 implementation). The entire operation runs in your browser — no data is transmitted to any server.

Use checksums to verify file integrity after downloads, detect tampering, or compare files without opening them.

MD5 128-bit. Fast but not collision-resistant. Legacy use only.
SHA-1 160-bit. Deprecated for security; still used in git.
SHA-256 256-bit. Standard choice for integrity checks and signatures.
SHA-512 512-bit. Strongest option; faster than SHA-256 on 64-bit CPUs.
