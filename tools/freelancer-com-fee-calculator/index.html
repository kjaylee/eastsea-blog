FAQ / 설명

Why does fixed-price use the awarded amount?

Freelancer’s public fee language states fixed-price projects use 10% of the selected bid or $5 minimum, so the base fee is tied to the awarded amount rather than only the amount released so far.

What counts as overage?

Any project payments above the original awarded amount are treated as overage here and charged at 10%, matching the public fee wording.

Why exclude the 15% branch?

Preferred Freelancer / Recruiter projects follow a different fee branch. This page stays narrow so the default calculator remains deterministic and honest.
