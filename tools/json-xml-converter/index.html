JSON ↔ XML 변환기

복잡한 데이터 포맷을 빠르게 변환하고 오류를 즉시 확인하세요. 개발/기획 실무에서 바로 쓰는 무료 도구입니다.

입력 (JSON)

0자 0줄

팁: Ctrl/Cmd + Enter로 빠르게 변환할 수 있습니다.

출력 (XML)

0자 0줄
대기 중 — 입력 후 변환 실행을 눌러주세요.
변환 규칙 안내
  • XML 속성은 JSON 변환 시 @속성명으로 변환됩니다.
  • 텍스트와 자식 노드가 함께 있을 때 텍스트는 #text 키를 사용합니다.
  • 동일 태그가 반복되면 JSON 배열로 자동 묶입니다.
