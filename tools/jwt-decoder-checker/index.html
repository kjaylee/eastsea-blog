What this JWT checker can and cannot do

This tool decodes Base64URL-encoded JWT header and payload sections so you can inspect claims like exp, iat, nbf, iss, and aud. It is useful for debugging tokens, but it does not validate signatures, keys, issuers, or audience rules.
