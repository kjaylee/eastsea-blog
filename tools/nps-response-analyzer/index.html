FAQ

NPS는 어떻게 계산되나요?

Promoter 비율(9–10점)에서 Detractor 비율(0–6점)을 뺀 값을 정수로 반올림합니다.

7점과 8점은 왜 NPS에 직접 기여하지 않나요?

7–8점 응답자는 Passive로 분류됩니다. 전체 맥락 해석에는 중요하지만 NPS 수식에는 가산/차감되지 않습니다.

데이터가 서버로 전송되나요?

아니요. 이 도구는 브라우저 안에서만 계산하며 업로드나 외부 API 호출이 없습니다.
