Reverb 수수료·순이익 계산기

Reverb 리스트 가격, 오퍼 할인, 구매자 배송비·세금, 셀러 프로그램, 상품 원가와 배송/포장 비용을 넣어 실제 순이익과 안전한 할인 한도를 계산합니다.

v1은 Reverb 공식 도움말 스니펫의 5% selling fee와 published processing baseline을 반영한 추정기입니다. bump fee·shipping label·cross-border fee는 수동 입력으로만 반영하며, 지역별 세부 예외는 포함하지 않습니다.

가정 및 메모
  • Reverb selling fee는 실거래가 + 구매자 배송비 기준 5%로 계산합니다.
  • 결제 처리 수수료는 실거래가 + 구매자 배송비 + 구매자 세금 기준으로 계산합니다.
  • 추가 플랫폼 수수료는 bump / shipping label / cross-border fee 같은 항목을 수동으로 반영하는 입력값입니다.
  • 구매자 세금은 seller revenue로 더하지 않고 processing fee 부담만 증가시키는 값으로 취급합니다.
왜 세금이 순이익에 영향을 주나요?
공식 스니펫 기준으로 processing fee는 주문 총액(배송비와 적용 세금 포함)에 걸립니다. 그래서 세금이 seller revenue는 아니더라도 결제 처리 수수료 부담은 키울 수 있습니다.
