💡 사용 가이드 / Usage Guide

  • OG 태그: <meta property="og:title">, <meta property="og:description">, <meta property="og:image"> 등으로 설정
  • 권장 이미지: 1200×630px (Twitter 1200×628px) — 작은 이미지는 크롭될 수 있음
  • Twitter 제한: 제목 70자, 설명 200자
  • Facebook 제한: 제목 100자, 설명 300자
  • LinkedIn 제한: 제목 150자, 설명 300자
  • 이미지 저장: Export PNG 버튼으로 카드 미리보기를 이미지로 다운로드
