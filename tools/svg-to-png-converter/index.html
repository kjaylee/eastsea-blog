💡 사용 가이드 / Usage Guide

  • 단일 변환: SVG 파일 업로드 또는 코드 붙여넣기 후 "PNG로 변환" 클릭
  • 일괄 변환: 여러 SVG 파일을 선택하면 배치 목록이 나타납니다
  • 배율: 2× = 실제 크기의 2배 해상도 출력 (Retina 디스플레이용)
  • 투명 배경: 아이콘, 로고 등 배경 없는 PNG 생성
  • 자동 감지: SVG의 원본 width/height/viewBox를 읽어 크기를 자동 설정
  • 보안: 모든 변환은 브라우저에서 처리되며 서버로 파일을 전송하지 않습니다
