📚 Unix Epoch 란?

Unix Epoch 기준 1970년 1월 1일 00:00:00 UTC 로부터의 경과 시간 (초)
초 vs 밀리초 대부분의 Unix 시스템은 초 단위. JavaScript Date.now()는 밀리초 반환.
2038년 문제 32비트 시스템은 2038-01-19 03:14:07 UTC에 오버플로 발생.
ISO 8601 날짜 시간 국제 표준 형식: YYYY-MM-DDTHH:mm:ssZ
