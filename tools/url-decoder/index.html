💡 사용 가이드 / Usage Guide

  • 디코딩: %XX 형태의 percent-encoding을 원래 문자로 변환합니다
  • URL 파싱: URL을 프로토콜, 호스트, 경로, 쿼리 파라미터, 프래그먼트로 분해합니다
  • 일괄 디코딩: 여러 줄을 한꺼번에 디코딩합니다 (API 로그 분석에 유용)
  • 복사: 결과 텍스트를 클릭하거나 복사 버튼을 누르면 클립보드에 복사됩니다
