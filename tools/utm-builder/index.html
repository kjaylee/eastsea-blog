🗂️ 저장된 히스토리

최근 생성 링크를 보관합니다. 최대 50개까지 저장됩니다.

✅ 작성 가이드

  • utm_source: 유입 채널 (newsletter, facebook, naver)
  • utm_medium: 매체 (email, cpc, social)
  • utm_campaign: 캠페인 키
  • utm_content: 버튼/배너 변형 추적
  • utm_term: 검색 키워드 추적
  • utm_id: 내부 식별자
